Class bookmark

expand

Description

A bookmark (corresponds to Acrobat’s PDBookmark object).

  • Plural name:
    • bookmarks
  • Defined in:
    • Acrobat Viewer Suite

Properties

  • best type (r/o) type class -- The best descriptor type.
  • class (r/o) type class -- The class.
  • default type (r/o) type class -- The default descriptor type.
  • destination page number integer -- The number of the page the PDF Window goes to when the bookmark is performed.
  • destination rectangle list of small real -- The boundary rectangle for the view of the destination, in PDF space (left, top, right, bottom). [Set this only after setting the fit type property].
  • fit type constant -- Controls how the destination rectangle is fitted to the window when the bookmark is executed. Valid values are: “Left Top Zoom”, “Fit Page”, “Fit Width”, “Fit Height”, “Fit Rect”, “Fit BBox”, “Fit BB Width”, and “Fit BB Height”.
  • index (r/o) integer -- The bookmark’s index within the Document.
  • name Unicode text -- The bookmark’s title.
  • zoom factor small real -- If fit type is “Left Top Zoom”, then this specifies the zoom factor, otherwise, this property is ignored. Setting this property automatically sets the fit type to “Left Top Zoom”.
