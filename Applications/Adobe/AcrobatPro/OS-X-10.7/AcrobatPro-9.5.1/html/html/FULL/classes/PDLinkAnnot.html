Class PDLinkAnnot

reduce

Description

Obsolete. Please use Link Annotation instead.

  • Defined in:
    • Acrobat Viewer Suite
  • Inherits from:
    • Link Annotation (Acrobat Viewer Suite) → annotation (Acrobat Viewer Suite)

Properties

  • best type (r/o) type class -- The best descriptor type.
  • bounds list of list -- The boundary rectangle for the annotation, in PDF space (left, top, right, bottom).
  • class (r/o) type class -- The class.
  • color <class RGB > -- The color of the border around the annotation.
  • contents Unicode text -- Text subtype only: The textual contents of the annotation.
  • default type (r/o) type class -- The default descriptor type.
  • destination page number integer -- Link subtype only: The number of the page the PDF Window goes to when the link annotation is performed.
  • destination rectangle list of small real -- Link subtype only: The boundary rectangle for the view of the destination, in PDF space (left, top, right, bottom).
  • fit type constant -- Link subtype only: Controls how the destination rectangle is fitted to the window when the annotation is performed. Valid values are: “Left Top Zoom”, “Fit Page”, “Fit Width”, “Fit Height”, “Fit Rect”, “Fit BBox”, “Fit BB Width”, and “Fit BB Height”.
  • index (r/o) integer -- The annotation’s index within a Page object.
  • modification date date -- The date and time the annotation was last modified.
  • name Unicode text -- Text subtypes only: The annotation’s label.
  • open state boolean -- Text subtype only: Whether the annotation is open.
  • subtype (r/o) Unicode text -- The subtype of the annotation.
  • zoom factor small real -- Link subtype only: If fit type is “Left Top Zoom”, then this specifies the zoom factor, otherwise, this property is ignored. Setting this property automatically sets the fit type to “Left Top Zoom”.
