Class application

reduce

Description

The Acrobat application.

  • Defined in:
    • Core Suite

Properties

  • active doc reference -- The active document.
  • active tool Unicode text -- The type of the currently active tool.
  • anti_alias text boolean -- Whether to anti-alias text and monochrome images.
  • best type (r/o) type class -- The best descriptor type.
  • case sensitivity boolean -- Whether searches are case sensitive.
  • class (r/o) type class -- The class.
  • default type (r/o) type class -- The default descriptor type.
  • default zoom factor small real -- The default zoom factor at which new documents are displayed.
  • default zoom type constant -- The default zoom type when opening a new document. Valid values are “no vary”, “fit page”, “fit width”, “fit height”, and “fit visible width”.
  • download entire file boolean -- Whether to download the entire file.
  • frontmost boolean -- Whether Acrobat is the frontmost application. Value can be set to true only; false not supported.
  • fullscreen transition (r/o) Unicode text -- Default fullscreen transition.
  • fullscreen loop (r/o) boolean -- Loop after last page in fullscreen mode.
  • fullscreen click advances boolean -- Mouse click advances in fullscreen mode.
  • fullscreen escape boolean -- Escape key exits fullscreen mode.
  • fullscreen cursor constant -- Cursor visibility in fullscreen mode. Valid values are “always visible”, “always hidden” or “hidden after delay”).
  • use fullscreen timer boolean -- Whether to use a timer to advance pages in fullscreen mode.
  • fullscreen timer delay integer -- The number of seconds to pause before advancing to next page in fullscreen mode.
  • highlight color <class RGB > -- Color used to highlight selections.
  • maximum documents (r/o) integer -- Maximum number of open documents.
  • name (r/o) Unicode text -- The application’s name.
  • note color <class RGB > -- The color of the border around newly created text annotations.
  • open in place boolean -- Open cross document links in the same window.
  • page units Unicode text -- Default page display units. One of Points, Picas, Inches, Millimeters or Centimeters.
  • page layout Unicode text -- Default page layout preference for a document on open. (“Single Page”, “Continuous”, “Continuous - Facing”, “Facing”)
  • save as linearize boolean -- Whether to save documents as linearized. Primarily used to optimize viewing documents in a web browser.
  • show splash at startup boolean -- Whether the splash screen is shown at startup.
  • shrink to fit boolean -- Obsolete in Acrobat 6. No longer used.
  • skip warnings boolean -- Whether to skip warning dialogs during program execution.
  • text note label Unicode text -- The text that will appear in the “title bar” of all newly created text notes.
  • toolbar visibility boolean -- Whether the ToolBar is visible.
  • UI language (r/o) Unicode text -- Identifys which language Acrobat’s UI is using. This is a 3 character language codes (ENU is English, for instance).
  • version (r/o) Unicode text -- The version number of the application.
  • whole word searching boolean -- Whether searches are for whole words.
  • note font name Unicode text -- Obsolete in Acrobat 6. No longer used. (was default note annotation font name).
  • note font size integer -- Obsolete in Acrobat 6. No longer used. (was default note annotation font size).
  • PS level integer -- Obsolete in Acrobat 6. No longer used. Please set Postscript Level when using “Save” or “Print pages” commands.
