Class page

reduce

Description

A document page (corresponds to Acrobat’s internal PDPage object).

  • Plural name:
    • pages
  • Defined in:
    • Acrobat Viewer Suite

Properties

  • best type (r/o) type class -- The best descriptor type.
  • class (r/o) type class -- The class.
  • art box list of list -- The art box rectangle for the page, in PDF space (left, top, right, bottom).
  • bleed box list of list -- The bleed box rectangle for the page, in PDF space (left, top, right, bottom).
  • crop box list of list -- The crop rectangle for the page, in PDF space (left, top, right, bottom).
  • default type (r/o) type class -- The default descriptor type.
  • label text (r/o) Unicode text -- The label (or custom page number) used to describe the page.
  • media box list of list -- The media bounds rectangle for the page, in PDF space (left, top, right, bottom).
  • page number (r/o) integer -- The page’s number.
  • rotation integer -- The rotation angle of the page (0, 90, 180, 270).
  • trim box list of list -- The trim box rectangle for the page, in PDF space (left, top, right, bottom).
  • bounds list of list -- Obsolete: use media box, crop box, art box, bleed box, and/or trim box instead.
