Class document

expand

Description

An Acrobat document.

  • Plural name:
    • documents
  • Defined in:
    • Core Suite

Properties

  • best type (r/o) type class -- The best descriptor type.
  • bounds (r/o) bounding rectangle -- The boundary rectangle for the document’s window, in screen coordinates (left, top, right, bottom). Note that (0,0) is in the upper left.
  • class (r/o) type class -- The class.
  • default type (r/o) type class -- The default descriptor type.
  • file alias (r/o) alias -- An alias to the file where the doc will be saved to if no other name is supplied. This is usually the same file as where it the document was read in from.
  • name (r/o) Unicode text -- The document’s name (as shown in the window’s titlebar).
  • modified (r/o) boolean -- Whether the document has been modified enough to warrant saving.
  • view mode constant -- The view mode of the document (one of: “just pages”, “pages and thumbs”, or “pages and bookmarks”).

  • bookmark -- by name, index
  • PDF Window -- by index, name, range, relative, test
