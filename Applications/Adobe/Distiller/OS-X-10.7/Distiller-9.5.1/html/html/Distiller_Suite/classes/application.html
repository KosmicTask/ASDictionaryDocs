Description

  • Defined in:

    • Distiller Suite
  • Inherits from:
    • application (Standard Suite) → item (Standard Suite)

Properties

  • frontmost (r/o) boolean -- Is this the frontmost (active) application?
  • name (r/o) Unicode text -- The name of the application.
  • version (r/o) Unicode text -- The version of the application.
  • locale (r/o) Unicode text -- Three character language code for Distiller's UI (ENU is English)
  • postScriptVersion (r/o) Unicode text -- PostScript interpreter version
