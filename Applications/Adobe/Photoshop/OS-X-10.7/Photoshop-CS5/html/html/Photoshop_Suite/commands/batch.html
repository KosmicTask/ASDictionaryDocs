Command batch

Description

run the batch automation routine

  • Defined in:
    • Photoshop Suite

Parameters

  • Unicode text -- the name of the action to play (note that the case of letters in the Action name is important and must match the case of the name in the Actions palette)
  • from files list of alias -- list of input files to operate on
  • from Unicode text -- the name of the action set containing the action being played (note that the case of letters in the Action Set name is important and must match the case of the name in the Actions palette)
  • [with options batch options] -- options for Batch

Result

  • Unicode text -- run the batch automation routine
