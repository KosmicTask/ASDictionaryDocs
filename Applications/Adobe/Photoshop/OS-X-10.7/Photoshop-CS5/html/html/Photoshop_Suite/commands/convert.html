Command convert

Description

convert the document from using one color profile to using an other

  • Defined in:
    • Photoshop Suite

Parameters

  • document -- the document object or objects to be operated upon
  • to profile Unicode text -- name of color profile to convert to. Either a string specifying a specific color profile or 'Working RGB', 'Working CMYK', 'Working Gray', 'Lab Color' meaning one of the working color spaces or Lab color
  • intent absolute colorimetric / perceptual / relative colorimetric / saturation intent -- conversion intent
  • [blackpoint compensation boolean] -- use black point compensation?
  • [dithering boolean] -- use dither? ( default: true )
