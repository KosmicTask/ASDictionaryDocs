Command trim

Description

  • Defined in:
    • Photoshop Suite

Parameters

  • document -- the document object or objects to be operated upon
  • [basing trim on bottom right pixel / top left pixel / transparent pixels] -- color or type of pixels to base trim on ( default: top left pixel )
  • [top trim boolean] -- trim away top of document ( default: true )
  • [left trim boolean] -- trim away left of document ( default: true )
  • [bottom trim boolean] -- trim away bottom of document ( default: true )
  • [right trim boolean] -- trim away right of document ( default: true )
