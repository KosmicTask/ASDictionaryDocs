Class picture package options

reduce

Description

options for the Picture Package command

  • Defined in:
    • Photoshop Suite

Properties

  • content caption text / copyright text / credit text / filename text / no text / origin text / user text -- content information ( default: no text )
  • custom text Unicode text -- picture package custom text ( default: )
  • flatten final boolean -- flatten all layers in the final document ( default: true )
  • font Arial / Courier New / Helvetica / Times New Roman -- font used for the text ( default: Arial )
  • font size integer -- font size used for the caption ( default: 12 )
  • layout Unicode text -- layout to use to generate the picture package ( default: (2)5x7 )
  • mode bitmap / CMYK / grayscale / Lab / RGB -- document mode (Grayscale, RGB, CMYK or Lab) ( default: RGB )
  • opacity integer -- web page security opacity as a percent ( default: 100 )
  • resolution real -- the resolution of the document (in pixels per inch) ( default: 72.0 )
  • text color RGB color -- text color
  • text position gallery centered / gallery lower left / gallery lower right / gallery upper left / gallery upper right -- text position ( default: gallery centered )
  • text rotate clock wise 45 / clock wise 90 / counter clock wise 45 / counter clock wise 90 / zero -- text rotate ( default: zero )
