Class presentation options

reduce

Description

options for the PDF presentation command

  • Defined in:
    • Photoshop Suite

Properties

  • auto advance boolean -- auto advance when viewing ( default: true )
  • include file name boolean -- include file name for image ( default: false )
  • interval integer -- time in seconds before auto advancing the view ( default: 5 )
  • loop boolean -- loop after last page ( default: false )
  • magnification actual size / fit page -- magnification type when viewing the image ( default: actual size )
  • PDF options PDF save options -- Options used when creating the PDF file
  • presentation boolean -- true if the file type is presentation false for Multi-Page document ( default: false )
  • transition blinds horizontal / blinds vertical / box in / box out / dissolve / glitter down / glitter right / glitter right down / none / random / split horizontal in / split horizontal out / split vertical in / split vertical out / wipe down / wipe left / wipe right / wipe up -- transition type when switching to the next document ( default: none )
