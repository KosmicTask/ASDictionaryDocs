Command $$$/AETE/Scripts=Scripts

Description

Photoshop scripting support plug-in

  • Defined in:
    • AdobeScriptAutomation

Parameters

  • [<class #ImR>]
  • $$$/JavaScript/Name=Name string -- The JavaScript name to execute
  • $$$/JavaScript/File=File string -- The JavaScript file to execute
  • $$$/JavaScript/Text=Text string -- The JavaScript text to execute
  • $$$/JavaScript/Debugging=Debugging boolean -- should the javascript be executed in debug mode?
  • $$$/JavaScript/Message=Message string -- the JavaScript parameters
