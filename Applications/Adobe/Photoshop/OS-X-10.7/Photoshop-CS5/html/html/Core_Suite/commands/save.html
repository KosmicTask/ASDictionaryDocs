Command save

Description

save the specified document

  • Defined in:
    • Core Suite

Parameters

  • document -- the document object or objects to be operated upon
  • [in file specification] -- the file to save the document in
  • [as Alias PIX / BMP / CompuServe GIF / Electric Image / JPEG / PCX / Photoshop DCS 1.0 / Photoshop DCS 2.0 / Photoshop EPS / Photoshop format / Photoshop PDF / PICT file / PICT resource / Pixar / PNG / Portable Bitmap / raw / Scitex CT / SGI RGB / SoftImage / Targa / TIFF / Wavefront RLA / Wireless Bitmap] -- The file type to save the document as. Note: Wireless Bitmap format not supported, for that format, use export and save for web.
  • [copying boolean] -- save as copy leaving original open. Unsaved files cannot be saved as copy
  • [appending lowercase extension / no extension / uppercase extension] -- append an extension to the document name
  • [with options save options] -- options for the file type specified

Result

  • document -- save the specified document
