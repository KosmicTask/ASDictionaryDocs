Properties

  • class (r/o) type class -- the object's class
  • default type (r/o) type class -- the default type for the object's value
  • properties record -- all of this object's properties returned in a single record
  • anchor count (r/o) integer -- The number of anchors in the tracing result.
  • area count (r/o) integer -- The number of areas in the tracing result.
  • container (r/o) reference -- the object's container
  • image resolution (r/o) real -- The resolution of the source image (in pixels per inch).
  • original art (r/o) page item -- The source art used when creating a new tracing object.
  • path count (r/o) integer -- The number of paths in the tracing result.
  • tracing options (r/o) tracingoptions -- The options used when tracing the artwork.
  • used color count (r/o) integer -- The number of colors used in the tracing result.
