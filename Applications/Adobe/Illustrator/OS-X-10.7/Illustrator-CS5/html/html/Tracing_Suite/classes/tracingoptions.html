Class tracingoptions

expand

Description

Tracing options that guide the tracing process.

  • Plural name:
    • multiple tracing options
  • Defined in:
    • Tracing Suite

Properties

  • best type (r/o) type class -- the best type for the object's value
  • class (r/o) type class -- the object's class
  • default type (r/o) type class -- the default type for the object's value
  • properties record -- all of this object's properties returned in a single record
  • container (r/o) reference -- the object's container
  • corner angle real -- Corner angle.
  • fills boolean -- Tracing with fills. At least one of Fills and Strokes must be on.
  • ignore white boolean -- Controls whether to ignore white fill color.
  • live paint output boolean -- Controls whether result art is LivePaint or classic art.
  • maximum colors integer -- The number of colors allowed for automatic palette generation.
  • maximum stroke weight real -- Maximum stroke weight (stroke only).
  • minimum area integer -- Minimum area.
  • minimum stroke length real -- Minimum stroke length (stroke only).
  • output swatches boolean -- Controls whether named colors (swatches) are generated.
  • palette Unicode text -- The color palette used for tracing, empty string indicates automatic palette.
  • path fitting real -- Path fitting.
  • preprocess blur real -- The blur amount during preprocessing.
  • preset (r/o) Unicode text -- The preset name use. Read-only.
  • resample boolean -- The resampling control.
  • resample resolution real -- The resampling resolution.
  • strokes boolean -- Tracing with strokes. At least one of Fills and Strokes must be on.
  • threshold integer -- The threshold value for a black and white tracing. ( 0 - 255 )
  • tracing mode bw tracing mode / color tracing mode / gray tracing mode -- The tracing mode: color, gray, black and white.
  • view raster view adjusted image / view no image / view original image / view transparent image -- The raster visualization mode.
  • view vector view no tracing result / view outlines / view outlines tracing / view tracing result -- The vector visualization mode.

Notes

  • Contained by:
    • tracingobject (Tracing Suite)
