Class postscript options

reduce

Description

the PostScript options

  • Defined in:
    • Print Suite

Properties

  • binary printing boolean -- whether to print in binary mode ( default: false )
  • compatible shading boolean -- use PostScript level 1 compatible gradient and gradient mesh printing ( default: false )
  • force continuous tone boolean -- whether to force continuous tone ( default: false )
  • image compression JPEG / none / RLE -- the image compression type ( default: none )
  • negative printing boolean -- whether to print in negative mode ( default: false )
  • PostScript level 1 / level 2 / level 3 -- the PostScript language level ( default: level 2 )
  • shading resolution real -- the shading resolution ( 1.0 - 9600.0; default: 300.0 )
