Class EPS save options

reduce

Description

Options which may be supplied when saving a document as an Illustrator EPS file

  • Defined in:
    • Save Options Suite

Properties

  • artboard range Unicode text -- If SaveMultipleArtboards is true,this will be considered for multi-asset extraction which specifies artboard range.Empty string will extracts all the artboards.Default is empty string ( default: )
  • CMYK PostScript boolean -- use CMYK PostScript ( default: true )
  • compatibility Illustrator 10 / Illustrator 11 / Illustrator 12 / Illustrator 13 / Illustrator 14 / Illustrator 15 / Illustrator 3 / Illustrator 8 / Illustrator 9 / Japanese 3 -- what Illustrator file format version to create ( default: Illustrator 15 )
  • compatible gradient printing boolean -- create a raster item of the gradient or gradient mesh so that PostScript Level 2 printers can print the object ( default: false )
  • embed all fonts boolean -- embed all fonts used by the document in the saved file (version 7 or later) ( default: false )
  • embed linked files boolean -- are linked image files to be included in the saved document ( default: false )
  • flatten output preserve appearance / preserve paths -- how should transparency be flattened for older file format versions (pre-version 9) ( default: preserve appearance )
  • include document thumbnails boolean -- include thumbnail image of the EPS artwork ( default: true )
  • overprint discard / preserve -- ( default: preserve )
  • PostScript level 2 / level 3 -- PostScript level to use (level 1 valid for file format version 8 or older) ( default: level 2 )
  • preview BW Macintosh / BW TIFF / color Macintosh / color TIFF / none / transparent color TIFF -- EPS preview format
  • save multiple artboards boolean -- All the artboards or range of the artboards will be saved ( default: false )
