Class raster effect options

reduce

Description

The document raster effects settings

  • Defined in:
    • Adobe Illustrator Suite

Properties

  • antialiasing boolean -- should the resulting image be antialiased ( default: false )
  • clipping mask boolean -- should a clipping mask be created for the resulting image ( default: false )
  • color model bitmap rasterization / default rasterization / grayscale rasterization -- The color model for the rasterization ( default: default rasterization )
  • convert spot colors boolean -- whether to convert all spot colors to process colors in the resulting image ( default: false )
  • padding real -- the amount of white space (in points) to be added around the object during rasterization ( default: 0.0 )
  • resolution real -- The rasterization resolution in dots-per-inch (dpi) ( 72.0 - 2400.0; default: 300.0 )
  • transparency boolean -- should the resulting image use transparency ( default: false )

Notes

  • Contained by:
    • document (Core Suite)
