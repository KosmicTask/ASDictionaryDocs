Class coordinate options

reduce

Description

the print coordinate options

  • Defined in:
    • Print Suite

Properties

  • emulsion boolean -- whether to flip artwork horizontally ( default: false )
  • fit to page boolean -- whether to proportionally scale the artwork to fit on the page ( default: false )
  • horizontal scale real -- the horizontal scaling factor expressed as a percentage (100 = 100%) ( 1.0 - 10000.0; default: 100.0 )
  • orientation auto rotate / landscape / portrait / reverse landscape / reverse portrait -- the artwork orientation ( default: portrait )
  • position bottom / bottom left / bottom right / center / left / right / top / top left / top right -- the artwork position on media ( default: center )
  • tiling full pages / imageable areas / single full page -- the page tiling mode ( default: single full page )
  • vertical scale real -- the vertical scaling factor expressed as a percentage (100 = 100%) ( 1.0 - 10000.0; default: 100.0 )

Notes

  • Contained by:
    • print options (Print Suite)
