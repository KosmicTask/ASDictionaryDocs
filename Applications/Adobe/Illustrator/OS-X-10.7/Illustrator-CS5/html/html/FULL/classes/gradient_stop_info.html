Class gradient stop info

reduce

Description

Gradient stop information (returned by entire gradient of gradient class)

  • Defined in:
    • Adobe Illustrator Suite

Properties

  • color color info -- the color linked to this gradient stop
  • midpoint real -- midpoint value in percent ( 13.0 - 87.0; default: 50.0 )
  • ramp point real -- location of color in the blend in percent ( 0.0 - 100.0; default: 0.0 )
  • stop opacity real -- The opacity value for the gradient stop ( 0.0 - 100.0; default: 1.0 )

Notes

  • Contained by:
    • gradient (Adobe Illustrator Suite)
