Class polygon

reduce

Description

Used to create a regular polygon path item. Not for path item access.

  • Defined in:
    • Shape Creation Suite

Properties

  • center point list -- the center point of the polygon
  • radius real -- the radius of the polygon points ( default: 50.0 )
  • sides integer -- the number of sides on the polygon ( default: 8 )
  • reversed boolean -- is the polygon path reversed? ( default: false )
