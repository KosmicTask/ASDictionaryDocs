Class page marks options

reduce

Description

the page marks options

  • Defined in:
    • Print Suite

Properties

  • bleed offset list -- the bleed offset rect
  • color bars boolean -- whether to enable color bars printing ( default: false )
  • marks offset list -- the page marks offset rect
  • page info marks boolean -- whether to enable page info marks printing ( default: false )
  • page marks style Japanese style / Roman -- the page marks style ( default: Roman )
  • registration marks boolean -- whether to enable registration marks printing ( default: false )
  • trim marks boolean -- whether to enable trim marks printing ( default: false )
  • trim marks weight real -- stroke weight of trim marks ( minimum 0.0; default: 0.125 )

Notes

  • Contained by:
    • print options (Print Suite)
