Class AutoCAD options

reduce

Description

Options which may be supplied when opening a AutoCAD file

  • Defined in:
    • Open Options Suite

Properties

  • best type (r/o) type class -- the best type for the object's value
  • class (r/o) type class -- the object's class
  • default type (r/o) type class -- the default type for the object's value
  • properties record -- all of this object's properties returned in a single record
  • center artwork boolean -- to center the created artwork on the artboard (default: true)
  • container (r/o) reference -- the object's container
  • global scale options fit artboard / original size / scale by value -- how to scale the AutoCAD drawing on import (default: Fit Artboard)
  • global scale percent real -- percentage scaling to apply globally on the AutoCAD drawing (default: 1.0)
  • merge layers boolean -- to merge the layers of the artwork (default: false)
  • scale lineweights boolean -- to scale lineweights by the same amount as rest of the drawing (default: false)
  • scale ratio real -- ratio by which to scale while mapping the units (default: 1.0)
  • scale unit autocad centimeters / autocad inches / autocad millimeters / autocad picas / autocad pixels / autocad points -- units to which to map (default: Millimeters)
  • selected layout name Unicode text -- name of layout in the AutoCAD drawing to import
