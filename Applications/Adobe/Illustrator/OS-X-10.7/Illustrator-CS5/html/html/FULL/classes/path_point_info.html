Class path point info

reduce

Description

Path point information (returned by entire path dataClassProperty of path item class)

  • Defined in:
    • Adobe Illustrator Suite

Properties

  • anchor list -- the position of the anchor (in coordinates)
  • left direction list -- location of the left direction point (in position)
  • point type corner / smooth -- the point type, smooth/corner ( default: smooth )
  • right direction list -- location of the left direction point (out position)

Notes

  • Contained by:
    • path item (Adobe Illustrator Suite)
    • text path (Text Suite)
