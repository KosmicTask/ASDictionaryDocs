Class job options

reduce

Description

the print job options

  • Defined in:
    • Print Suite

Properties

  • artboard range Unicode text -- Artboard Range to be printed if PrintAllArtboards is false ( default: 1- )
  • bitmap resolution real -- the bitmap resolution ( minimum 0.0; default: 0.0 )
  • collate boolean -- whether to collate print pages ( default: false )
  • copies integer -- the number of copies to print ( minimum 1; default: 1 )
  • designation all layers / visible layers / visible printable layers -- the layers/objects to be printed ( default: visible printable layers )
  • file path file specification -- the file to be printed to
  • name Unicode text -- the print job name
  • print all artboards boolean -- whether to print all artboards ( default: true )
  • print area artboard bounds / artwork bounds / crop bounds -- the printing bounds ( default: artboard bounds )
  • print as bitmap boolean -- whether to print as bitmap ( default: false )
  • reverse pages boolean -- print pages in reverse order ( default: false )

Notes

  • Contained by:
    • print options (Print Suite)
