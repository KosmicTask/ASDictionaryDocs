Class artboard

reduce

Description

an artboard object

  • Plural name:
    • artboards
  • Defined in:
    • Adobe Illustrator Suite

Properties

  • best type (r/o) type class -- the best type for the object's value
  • class (r/o) type class -- the object's class
  • default type (r/o) type class -- the default type for the object's value
  • index (r/o) integer -- the index of this instance of the object
  • properties record -- all of this object's properties returned in a single record
  • artboard rectangle list -- size and position of artboard
  • container (r/o) reference -- the object's container
  • name Unicode text -- The name of the artboard
  • ruler origin list -- Ruler origin of artboard.It is relative to left-bottom corner of the Artboard.
  • ruler PAR real -- Pixel aspect ratio, used in ruler visualization if the units are pixels ( 0.1 - 10.0 )
  • show center boolean -- Show center mark
  • show cross hairs boolean -- Show cross hairs
  • show safe areas boolean -- Show title and action safe areas (for video)
