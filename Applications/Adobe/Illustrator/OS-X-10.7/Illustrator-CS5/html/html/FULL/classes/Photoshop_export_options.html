Class Photoshop export options

reduce

Description

Options which may be supplied when exporting a document as a Photoshop file

  • Defined in:
    • Export Options Suite

Properties

  • antialiasing boolean -- should the resulting image be antialiased ( default: true )
  • artboard range Unicode text -- If SaveMultipleArtboards is true,this will be considered for multi-asset extraction which specifies artboard range.Empty string will extracts all the artboards.Default is empty string ( default: )
  • color space CMYK / DeviceN / Gray / Indexed / LAB / RGB / Separation -- the color space of the exported file ( default: RGB )
  • editable text boolean -- export text objects as editable text layers ( default: true )
  • embed ICC profile boolean -- embed an ICC profile when exporting ( default: false )
  • maximum editability boolean -- preserve as much of the original document's structure as possible when exporting ( default: true )
  • resolution real -- The resolution of the exported file ( 72.0 - 2400.0; default: 150.0 )
  • save multiple artboards boolean -- All the artboards or range of the artboards will be exported ( default: false )
  • warnings boolean -- should a warning dialog be displayed because of conflicts in the export settings ( default: true )
  • write layers boolean -- preserve document layers when exporting ( default: true )
