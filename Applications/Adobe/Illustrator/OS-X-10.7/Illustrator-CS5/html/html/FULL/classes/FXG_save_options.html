Class FXG save options

reduce

Description

Options which may be supplied when saving a document as an FXG file

  • Defined in:
    • Save Options Suite

Properties

  • artboard range Unicode text -- If SaveMultipleArtboards is true,this will be considered for multi-asset extraction which specifies artboard range.Empty string will extracts all the artboards.Default is empty string ( default: )
  • blends policy autoConvert blends / rasterize blends -- the policy used by FXG to expand blends ( default: autoConvert blends )
  • downsample linked images boolean -- choose to downsample the linked images(72ppi) ( default: false )
  • filters policy expand filters / keep filters editable / rasterize filters -- the policy used by FXG to preserve filters ( default: keep filters editable )
  • fxg version version 1.0 / version 2.0 -- the version of the FXG file format to create ( default: version 2.0 )
  • gradients policy autoConvert gradients / keep gradients editable -- the policy used by FXG to preserve gradients ( default: autoConvert gradients )
  • include metadata boolean -- choose to include metadata (XMP) ( default: false )
  • include unused symbols boolean -- choose to include unused symbols ( default: false )
  • preserve editing capabilities boolean -- choose to preserve the editing capabilities of FXG ( default: true )
  • save multiple artboards boolean -- All the artboards or range of the artboards will be saved ( default: false )
  • text policy autoConvert text / keep text editable / outline text / rasterize text -- the policy used by FXG to preserve text ( default: autoConvert text )
