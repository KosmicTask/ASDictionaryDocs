Class color separation options

reduce

Description

Print color separation options

  • Defined in:
    • Print Suite

Properties

  • convert spot colors boolean -- whether to convert all spot colors to process colors ( default: false )
  • inks list of ink -- the list of inks for color separation
  • over print black boolean -- whether to overprint in black ( default: false )
  • separation mode composite / host based separation / InRIP separation -- the color separation type ( default: composite )

Notes

  • Contained by:
    • print options (Print Suite)
