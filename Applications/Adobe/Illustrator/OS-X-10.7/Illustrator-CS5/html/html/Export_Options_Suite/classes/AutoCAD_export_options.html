Class AutoCAD export options

expand

Description

Options which may be supplied when exporting a document to AutoCAD formats (.dwg or .dxf)

  • Defined in:
    • Export Options Suite

Properties

  • alter paths for appearance boolean -- alter paths for appearance ( default: false )
  • AutoCAD version AutoCAD release 13 / AutoCAD release 14 / AutoCAD release 15 / AutoCAD release 18 -- release of AutoCAD to export to ( default: AutoCAD release 18 )
  • colors max 16 colors / max 256 colors / max 8 colors / true colors -- number of colors to export into the AutoCAD file ( default: true colors )
  • convert text to outlines boolean -- whether to convert text to outlines ( default: false )
  • export file format dwg / dxf -- which format to export the file as ( default: dwg )
  • export option maintain appearance / maximize editability -- whether to preserve appearance or editability during export ( default: maximize editability )
  • export selected art only boolean -- export selected art only ( default: false )
  • raster format JPEG raster / PNG raster -- raster format in which to export raster art ( default: PNG raster )
  • scale lineweights boolean -- whether to scale lineweights by the same amount as rest of the drawing ( default: false )
  • scale ratio real -- ratio by which to scale the output ( default: 1.0 )
  • scale unit autocad centimeters / autocad inches / autocad millimeters / autocad picas / autocad pixels / autocad points -- units from which to map ( default: autocad millimeters )
