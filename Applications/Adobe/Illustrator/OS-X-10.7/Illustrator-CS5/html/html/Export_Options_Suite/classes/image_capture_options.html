Class image capture options

expand

Description

Options which may be supplied when capturing a portion of the artwork as an 24 bit PNG file

  • Defined in:
    • Export Options Suite

Properties

  • antialiasing boolean -- should the resulting image be antialiased ( default: false )
  • matte boolean -- should the artboard be matted with a color ( default: false )
  • matte color RGB color info -- the color to use when matting the artboard (default: white)
  • resolution real -- The resolution of the captured image file ( 72.0 - 2400.0; default: 150.0 )
  • transparency boolean -- should the resulting image use transparency ( default: false )
