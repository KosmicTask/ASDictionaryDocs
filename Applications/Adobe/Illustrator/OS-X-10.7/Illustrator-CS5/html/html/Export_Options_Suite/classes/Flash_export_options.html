Class Flash export options

expand

Description

Options which may be supplied when exporting a document as an Flash (.SWF) file

  • Defined in:
    • Export Options Suite

Properties

  • art clipping output art bounds / output artboard bounds / output croprect bounds -- how should the arts be clipped during the output ( default: output art bounds )
  • artboard range Unicode text -- If SaveMultipleArtboards is true,this will be considered for multi-asset extraction which specifies artboard range.Empty string will extracts all the artboards.Default is empty string ( default: )
  • background color RGB color info -- the background color
  • background layers list of layer -- a list of layers to be included as the static background in all exported Flash frames
  • blend animation in build / in sequence / none -- controls how the blend art objects are animated when export to Flash frames ( default: none )
  • compressed boolean -- should the exported file be compressed ( default: false )
  • convert text to outlines boolean -- should all text be converted to outlines ( default: false )
  • curve quality integer -- how much curve information should be preserved ( 0 - 10; default: 7 )
  • export all symbols boolean -- should all symbol definitions in the palette be exported to the SWF File ( default: false )
  • export style Artboards to Files / Flash file / layers to files / layers to frames / layers to symbols -- how should the Flash file be created ( default: Flash file )
  • export version SWF version 1 / SWF version 2 / SWF version 3 / SWF version 4 / SWF version 5 / SWF version 6 / SWF version 7 / SWF version 8 / SWF version 9 -- which version of SWF to export ( default: SWF version 9 )
  • Flash Playback Security flash playback local access / flash playback network access -- what access should the SWF file have - local or network access ( default: flash playback local access )
  • frame rate real -- when exporting layers to Flash frames ( 0.01 - 120.0; default: 12.0 )
  • image format lossless / lossy -- how should the images in the exported Flash file be compressed ( default: lossless )
  • include metadata boolean -- If true, include minimal XMP metadata in the exported file ( default: false )
  • JPEG method optimized / standard -- what method to use ( default: standard )
  • JPEG quality integer -- level of compression ( 0 - 10; default: 3 )
  • layer order bottom up / top down -- the order in which the layers will be exported to Flash frames ( default: bottom up )
  • looping boolean -- should the Flash file be set to loop when run ( default: false )
  • preserve appearance boolean -- choose whether to preserve artwork appearance or editability (default) during export ( default: false )
  • read only boolean -- prevent the exported file from being imported by other applications ( default: false )
  • replacing ask / no / yes -- if a file with the same name already exists, should it be replaced? ( default: ask )
  • resolution real -- pixels per inch ( 72.0 - 2400.0; default: 72.0 )
  • save multiple artboards boolean -- All the artboards or range of the artboards will be exported ( default: false )
  • textkerning boolean -- should the kerning information for text objects be ignored ( default: false )
