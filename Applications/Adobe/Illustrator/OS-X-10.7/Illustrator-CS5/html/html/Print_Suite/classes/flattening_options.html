Class flattening options

expand

Description

the transparency flattening options

  • Defined in:
    • Print Suite

Properties

  • clip complex regions boolean -- whether to clip complex regions ( default: false )
  • convert strokes to outlines boolean -- whether to convert all strokes to outlines ( default: false )
  • convert text to outlines boolean -- whether to convert all text to outlines ( default: false )
  • flattening balance integer -- the flattening balance ( 0 - 100; default: 100 )
  • gradient resolution real -- the gradient resolution ( 1.0 - 9600.0; default: 300.0 )
  • overprint discard / preserve -- overprint ( default: preserve )
  • rasterization resolution real -- the rasterization resolution ( 1.0 - 9600.0; default: 300.0 )
