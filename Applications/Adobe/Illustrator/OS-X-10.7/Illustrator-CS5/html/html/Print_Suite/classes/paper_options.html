Class paper options

expand

Description

the paper options

  • Defined in:
    • Print Suite

Properties

  • height real -- the custom height (in points) for using the custom paper ( default: 0.0 )
  • name Unicode text -- the paper's name
  • offset real -- custom offset (in points) for using the custom paper ( default: 0.0 )
  • transverse boolean -- whether to transverse the artwork (rotate 90 degrees) on the custom paper ( default: false )
  • width real -- the custom width (in points) for using the custom paper ( default: 0.0 )
