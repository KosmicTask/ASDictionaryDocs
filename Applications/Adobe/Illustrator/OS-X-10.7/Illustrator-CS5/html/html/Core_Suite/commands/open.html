Command open

Description

Open the specified document file(s)

  • Defined in:
    • Core Suite

Parameters

  • anything -- the file(s) to be opened
  • [forcing CMYK / RGB] -- choose color space only for documents saved with multiple color models (pre-Illustrator 9)
  • [dialogs boolean] -- show warning and error dialogs when opening the file(s)? ( default: true )
  • [with options anything] -- options for opening a particular type of file
