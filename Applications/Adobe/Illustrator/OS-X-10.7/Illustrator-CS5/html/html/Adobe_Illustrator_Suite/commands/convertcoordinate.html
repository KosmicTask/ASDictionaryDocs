Command convertcoordinate

Description

Converts the coordinate system of a single point from one coordinate system to another.

  • Defined in:
    • Adobe Illustrator Suite

Parameters

  • document -- the document object or objects to be operated upon
  • coordinate list -- The Coordinate to convert.
  • source artboard coordinate system / document coordinate system -- The source coordinate system.
  • destination artboard coordinate system / document coordinate system -- The destination coordinate system.

Result

  • list -- Converts the coordinate system of a single point from one coordinate system to another.
