Command rasterize

Description

rasterize the source art(s) within the specified clip bounds. The source art(s) are disposed as a result of the rasterization.

  • Defined in:
    • Adobe Illustrator Suite

Parameters

  • document -- the document object or objects to be operated upon
  • source art anything -- the page item(s) to be rasterized
  • [inside list] -- the rectangular region of the artwork for the rasterization. If the parameter is omitted, the bounds of the source art(s) is used instead.
  • [with options rasterize options] -- describes the rasterization options

Result

  • page item -- rasterize the source art(s) within the specified clip bounds. The source art(s) are disposed as a result of the rasterization.
