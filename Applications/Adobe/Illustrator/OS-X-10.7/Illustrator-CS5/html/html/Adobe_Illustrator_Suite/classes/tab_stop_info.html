Class tab stop info

expand

Description

Tab stop information (returned by tab stops from a paragraph object)

  • Defined in:
    • Adobe Illustrator Suite

Properties

  • alignment center / decimal / left / right -- the alignment of the tab stop ( default: left )
  • decimal character Unicode text -- the character used for decimal tab stops ( default: . )
  • leader Unicode text -- the leader dot
  • position real -- the position of the tab stop expressed in points ( default: 0.0 )

Notes

  • Contained by:
    • paragraph (Text Suite)
    • paragraph style (Text Suite)
