Class raster item

expand

Description

Raster artwork item

  • Plural name:
    • raster items
  • Defined in:
    • Adobe Illustrator Suite
  • Inherits from:
    • page item (Adobe Illustrator Suite)

Properties

  • bits per channel (r/o) integer -- the number of bits per channel
  • bounding box list -- dimensions of raster art object regardless of transformations
  • channels (r/o) integer -- the number of image channels
  • color space (r/o) CMYK / DeviceN / Gray / Indexed / LAB / RGB / Separation -- the color space of the raster image
  • colorants (r/o) list of Unicode text -- list of colorant names
  • colorized (r/o) boolean -- is the raster art a colorized grayscale image?
  • content variable anything -- the content variable bound to this raster art object
  • embedded boolean -- is the raster art embedded within the illustration?
  • file path (r/o) file specification -- the file containing the raster artwork
  • matrix matrix -- The transformation matrix of the raster art object
  • overprint boolean -- is the raster art overprinting?
  • status (r/o) data from file / modified data / no data -- status of the linked image
  • transparent (r/o) boolean -- is the raster art transparent?

Notes
