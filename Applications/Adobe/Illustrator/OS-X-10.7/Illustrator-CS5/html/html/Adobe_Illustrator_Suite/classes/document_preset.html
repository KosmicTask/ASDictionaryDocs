Class document preset

expand

Description

the new document preset to use for creating a new document

  • Defined in:
    • Adobe Illustrator Suite

Properties

  • artboardLayout column / grid by column / grid by row / rl grid by col / rl grid by row / rl row / row -- layout for artboards ( default: grid by row )
  • artboardRowsOrCols integer -- Number of rows (for rows layout) OR column(for column layouts)of artboards.Range is 1 to (docNumArtboards - 1) or 1 for single row or column layouts ( default: 1 )
  • artboardSpacing real -- spacing between artboards ( default: 20.0 )
  • color mode CMYK / RGB -- the color mode for the new document ( default: CMYK )
  • document units centimeters / inches / millimeters / picas / pixels / points / qs / unknown -- the units for the new document ( default: points )
  • height real -- the height for the new document ( default: 792.0 )
  • numArtboards integer -- number of artboards for new document.Range (1:100). ( default: 1 )
  • preview mode default preview / overprint preview / pixel preview -- the preview mode for the new document ( default: default preview )
  • raster resolution high resolution / medium resolution / screen resolution -- the raster resolution for the new document ( default: screen resolution )
  • title Unicode text -- the title for the new document ( default: Untitled )
  • transparency grid blue transparency grids / dark color transparency grids / green transparency grids / hide transparency grids / light color transparency grids / medium color transparency grids / orange transparency grids / purple transparency grids / red color transparency grids -- the transparency grid for the new document ( default: hide transparency grids )
  • width real -- the width for the new document ( default: 612.0 )

Notes

  • Contained by:
    • artwork (Core Suite)
