Command export

Description

Export the specified document(s)

  • Defined in:
    • Adobe Illustrator Suite

Parameters

  • document -- the document object or objects to be operated upon
  • to file specification -- the file to export the document to
  • as AutoCAD / Flash / GIF / JPEG / Photoshop / PNG24 / PNG8 / SVG -- the file type to export the document as
  • [with options anything] -- options for the file type specified
