Command rearrangeartboards

Description

Rearrange Artboards in the document

  • Defined in:
    • Adobe Illustrator Suite

Parameters

  • document -- the document object or objects to be operated upon
  • [artboardLayout column / grid by column / grid by row / rl grid by col / rl grid by row / rl row / row] -- layout of artboards for rearrangement ( default: grid by row )
  • [artboardRowsOrCols integer] -- Number of rows (for rows layout) OR column(for column layouts)of artboards.Range is 1 to (docNumArtboards - 1) or 1 for single row or column layouts ( default: 1 )
  • [artboardSpacing real] -- spacing between artboards ( default: 20.0 )
  • [artboardMoveArtwork boolean] -- Whether to move artwork with artboards ( default: true )

Result

  • boolean -- Rearrange Artboards in the document
