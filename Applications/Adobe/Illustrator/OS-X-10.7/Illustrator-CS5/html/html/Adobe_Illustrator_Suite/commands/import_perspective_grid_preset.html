Command import perspective grid preset

Description

loads mentioned perspective grid preset, if preset name is specified, else loads all(if no preset name is specified) presets, from the specified file

  • Defined in:
    • Adobe Illustrator Suite

Parameters

  • document -- the document object or objects to be operated upon
  • from file specification -- file to import from
  • [perspective preset Unicode text] -- name of perspective grid preset. ( default: )
