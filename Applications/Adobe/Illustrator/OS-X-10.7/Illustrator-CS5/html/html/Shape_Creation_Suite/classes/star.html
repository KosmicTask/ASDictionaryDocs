Class star

expand

Description

Used to create a star-shaped path item. Not for path item access.

  • Defined in:
    • Shape Creation Suite

Properties

  • center point list -- the center point of the rectangle
  • radius real -- the outside radius of the star points ( default: 50.0 )
  • inner radius real -- the inside radius of the star points ( default: 20.0 )
  • point count integer -- the number of points on the star ( default: 5 )
  • reversed boolean -- is the star path reversed? ( default: false )
