Command convert sample color

Description

Converts a sample-component color from one color space to another.

  • Defined in:
    • Color Suite

Parameters

  • source color space CMYK / DeviceN / Gray / Indexed / LAB / RGB / Separation -- The source color space
  • source color list of real -- The color to convert, an array of color components. First location of the array should contain alpha if source-has-alpha is true.
  • destination color space CMYK / DeviceN / Gray / Indexed / LAB / RGB / Separation -- The destination color space
  • color conversion purpose default purpose / dummy purpose option / export purpose / preview purpose -- The parameter which passes the purpose of conversion
  • [source has alpha boolean] -- True if alpha channel is present in source color. ( default: false )
  • [destination has alpha boolean] -- True if alpha channel is present in destination color. ( default: false )

Result

  • list of real -- Converts a sample-component color from one color space to another.
