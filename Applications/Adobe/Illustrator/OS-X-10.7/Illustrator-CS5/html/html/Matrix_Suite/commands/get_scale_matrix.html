Command get scale matrix

Description

Returns a scale transformation matrix

  • Defined in:
    • Matrix Suite

Parameters

  • [horizontal scale real] -- horizontal scaling factor expressed as a percentage (100 = 100%) ( default: 100.0 )
  • [vertical scale real] -- vertical scaling factor expressed as a percentage (100 = 100%) ( default: 100.0 )

Result

  • matrix -- Returns a scale transformation matrix
