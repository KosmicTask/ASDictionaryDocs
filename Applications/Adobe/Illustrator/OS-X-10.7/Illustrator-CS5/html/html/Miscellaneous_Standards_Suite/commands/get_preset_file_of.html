Command get preset file of

Description

given a preset type, returns the full path to the application's default document profile for the type

  • Defined in:
    • Miscellaneous Standards Suite

Parameters

  • preset type basic CMYK document / basic RGB document / mobile document preset / print document preset / video document preset / web document preset -- the preset type

Result

  • file specification -- given a preset type, returns the full path to the application's default document profile for the type
