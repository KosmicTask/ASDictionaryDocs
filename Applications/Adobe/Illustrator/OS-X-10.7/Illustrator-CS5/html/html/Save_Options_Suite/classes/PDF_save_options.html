Class PDF save options

expand

Description

Options which may be supplied when saving a document as a PDF file

  • Defined in:
    • Save Options Suite

Properties

  • acrobat layers boolean -- create acrobat layers from top-level layers - acrobat 6 only option ( default: false )
  • allow printing pdf 128 print high res / pdf 128 print low res / pdf 128 print none / pdf 40 print high res / pdf 40 print none -- PDF security printing permission ( default: pdf 128 print high res )
  • artboard range Unicode text -- Considered for multi-asset extraction which specifies artboard range.Empty string will extracts all the artboards.Default is empty string ( default: )
  • bleed link boolean -- link 4 bleed values ( default: true )
  • bleed offset list -- the bleed offset rect
  • changes allowed pdf 128 any changes / pdf 128 commenting allowed / pdf 128 edit page allowed / pdf 128 fill form allowed / pdf 128 no changes / pdf 40 any changes / pdf 40 commenting allowed / pdf 40 no changes / pdf 40 page layout allowed -- PDF security changes allowed ( default: pdf 128 any changes )
  • color bars boolean -- draw color bars ( default: false )
  • color compression automatic JPEG high / automatic JPEG low / automatic JPEG maximum / automatic JPEG medium / automatic JPEG minimum / automatic JPEG2000 high / automatic JPEG2000 lossless / automatic JPEG2000 low / automatic JPEG2000 maximum / automatic JPEG2000 medium / automatic JPEG2000 minimum / JPEG high / JPEG low / JPEG maximum / JPEG medium / JPEG minimum / JPEG2000 high / JPEG2000 lossless / JPEG2000 low / JPEG2000 maximum / JPEG2000 medium / JPEG2000 minimum / none / ZIP4bit / ZIP8bit -- how should color bitmap images be compressed ( default: none )
  • color conversion id color conversion repurpose / color conversion to dest / none -- PDF color conversion policy. Three choices are available: (1)No Color Conversion (2) Repurpose (3) Convert to Destination ( default: none )
  • color destination id color dest doc cmyk / color dest doc rgb / color dest profile / color dest working cmyk / color dest working rgb / none -- When NoColorConversion is specified for Color Conversion, NoColorDestination is set ( default: none )
  • color downsampling real -- if zero, no downsampling, otherwise, the resolution to downsample color bitmap images to ( default: 150.0 )
  • color downsampling threshold real -- downsample if the image's resolution is above this value ( default: 225.0 )
  • color profile id include all profiles / include all rgb / include dest profile / leave profile unchanged / none -- If CMS is off, Don't Include Profiles is set ( default: none )
  • color resample average downsampling / bicubic downsample / nodownsample / subsampling -- how should color bitmap images be resampled ( default: nodownsample )
  • color tile size integer -- tile size when compressing with JPEG2000 ( default: 256 )
  • compatibility Acrobat 4 / Acrobat 5 / Acrobat 6 / Acrobat 7 / Acrobat 8 -- the version of the Acrobat file format to create ( default: Acrobat 5 )
  • compress art boolean -- should line art and text be compressed? ( default: true )
  • document password Unicode text -- a password string to open the document ( default: )
  • enable access boolean -- enable accessing 128-bit ( default: true )
  • enable copy boolean -- enable copying of text 128-bit ( default: true )
  • enable copy and access boolean -- enable copying and accessing 40-bit ( default: true )
  • enable plaintext boolean -- enable plaintext metadata 128-bit - available only for acrobat 6 ( default: false )
  • flattener preset Unicode text -- the transparency flattener preset name
  • flattener settings flattening options -- the printing flattener options
  • font subset threshold real -- include a subset of fonts when less than this percentage of characters are used ( 0.0 - 100.0; default: 100.0 )
  • generate thumbnails boolean -- generate thumbnails for the saved document ( default: true )
  • grayscale compression automatic JPEG high / automatic JPEG low / automatic JPEG maximum / automatic JPEG medium / automatic JPEG minimum / automatic JPEG2000 high / automatic JPEG2000 lossless / automatic JPEG2000 low / automatic JPEG2000 maximum / automatic JPEG2000 medium / automatic JPEG2000 minimum / JPEG high / JPEG low / JPEG maximum / JPEG medium / JPEG minimum / JPEG2000 high / JPEG2000 lossless / JPEG2000 low / JPEG2000 maximum / JPEG2000 medium / JPEG2000 minimum / none / ZIP4bit / ZIP8bit -- how should grayscale bitmap images be compressed ( default: none )
  • grayscale downsampling real -- if zero, no downsampling, otherwise, the resolution to downsample grayscale images to ( default: 150.0 )
  • grayscale downsampling threshold real -- downsample if the image's resolution is above this value ( default: 225.0 )
  • grayscale resample average downsampling / bicubic downsample / nodownsample / subsampling -- how should grayscale bitmap images be resampled ( default: nodownsample )
  • grayscale tile size integer -- tile size when compressing with JPEG2000 ( default: 256 )
  • monochrome compression CCIT3 / CCIT4 / none / run length / ZIP -- how should monochrome bitmap images be compressed ( default: none )
  • monochrome downsampling real -- If zero, no downsampling, otherwise, the resolution to downsample images to ( default: 300.0 )
  • monochrome downsampling threshold real -- downsample if the image's resolution is above this value ( default: 450.0 )
  • monochrome resample average downsampling / bicubic downsample / nodownsample / subsampling -- how should monochrome bitmap images be resampled ( default: nodownsample )
  • offset real -- custom offset (in points) for using the custom paper ( default: 0.0 )
  • optimization boolean -- should the PDF document be optimized for fast web viewing ( default: false )
  • output condition Unicode text -- This is an optional comment which, if present, is added to the PDF file and describes the intended printing condition ( default: )
  • output condition id Unicode text -- If selected for Output Intent Profile Name, you can set the name of a registered printing condition ( default: )
  • output intent profile Unicode text -- When CMS is on, the output intent profile is the same profile selected for Destination in the Color group box ( default: )
  • page info boolean -- draw page information ( default: false )
  • page marks style Japanese style / Roman -- the page marks style ( default: Roman )
  • PDF preset Unicode text -- The max string length is 255 bytes. Name of PDF preset to use.
  • pdfXstandard PDFX 1a 2001 / PDFX 1a 2003 / PDFX 3 2001 / PDFX 3 2002 / PDFX 3 2003 / PDFX 4 2007 / PDFX None -- This control includes the None option for when the user is not complying with any PDF standard ( default: PDFX None )
  • pdfXstandard descripton Unicode text -- This displays the description from the selected preset
  • permission password Unicode text -- a password string to restrict editing security settings ( default: )
  • preserve editability boolean -- preserve Illustrator editing capabilities when saving the document ( default: true )
  • printer resolution real -- flattening printer resolution ( default: 800.0 )
  • registration marks boolean -- draw registration marks ( default: false )
  • registry name Unicode text -- URL to the site where the specified output condition is registered. No validation is performed on the URL ( default: )
  • require doc password boolean -- require a password to open the document ( default: false )
  • require perm password boolean -- use a password to restrict editing security settings ( default: false )
  • trapped boolean -- This indicates if manual trapping has been prepared in the document ( default: false )
  • trim mark weight trimmarkweight0125 / trimmarkweight025 / trimmarkweight05 -- trim mark weight ( default: trimmarkweight0125 )
  • trim marks boolean -- draw trim marks ( default: false )
  • view pdf boolean -- view PDF after saving ( default: false )
