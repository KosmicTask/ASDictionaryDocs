Class open options

expand

Description

Options which may be supplied when opening a file

  • Defined in:
    • Open Options Suite

Properties

  • as brushes library / graphic styles library / Illustrator artwork / swatches library / symbols library -- open the file as library ( default: Illustrator artwork )
  • convert crop area to artboard boolean -- Convert crop area to Artboard when opening legacy document (pre-Illustrator CS4) in CS4 or later. If false then crop areas are discarded. ( default: true )
  • convert tiles to artboard boolean -- Convert print tiles to Artboard when opening legacy document (pre-Illustrator CS4) in CS4 or later. ( default: false )
  • create artboard with artwork bounding box boolean -- Create Artboard with dimentions of artwork bounding box when opening legacy document (pre-Illustrator CS4) in CS4 or later. ( default: false )
  • preserve legacy artboard boolean -- Preserve Legacy Artboard (pre-Illustrator CS4) when opening in CS4 or later. ( default: true )
  • update legacy gradient mesh boolean -- choose to preserve the spot colors in the gradient mesh objects for legacy documents (pre-Illustrator CS3) ( default: true )
  • update legacy text boolean -- choose to update all text objects for documents saved with legacy texts (pre-Illustrator 11) ( default: false )
