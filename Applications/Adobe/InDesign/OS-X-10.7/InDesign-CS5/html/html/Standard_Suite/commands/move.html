Command move

Description

Moves the object to a new location. Note: Either the to or the by parameter must be specified. Do not provide values for both parameters.

  • Defined in:
    • Standard Suite

Parameters

  • reference
  • to location reference -- The new location of the object.
  • [by number | string] -- The amount to move the object relative to its current position, in the format [x, y].
  • [binding right align / left align / default] -- The side of the binding spine on which to place the object. Note: Valid only when moving a page. Can accept: right align/left align/default (Default: default).

Result

  • reference -- The moved object.
