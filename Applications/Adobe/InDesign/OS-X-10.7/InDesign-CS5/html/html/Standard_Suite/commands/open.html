Command open

Description

Opens the specified document, book, or library.

  • Defined in:
    • Standard Suite

Parameters

  • anything -- The file path(s) to the document, book, or library.
  • [showing window boolean] -- If true, opens the document in a window. If false, the document is opened but is not displayed in a window. Can accept: boolean (Default: TRUE).
  • [open option default / open original / open copy] -- How to open the document. Can accept: default/open original/open copy (Default: default).

Result

  • anything -- The opened document, book, or library. Can return: document, book, library or list of documents, books or libraries.
