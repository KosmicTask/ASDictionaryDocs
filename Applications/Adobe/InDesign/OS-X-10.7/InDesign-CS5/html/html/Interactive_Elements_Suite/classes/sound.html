Class sound

expand

Description

A sound clip.

  • Plural name:
    • sounds
  • Defined in:
    • Interactive Elements Suite
  • Inherits from:
    • media item (Layout Suite) → page item (Layout Suite)

Properties

  • absolute flip none / horizontal / vertical / horizontal and vertical / both -- Indicates whether the sound has been flipped independently of its parent object and, if yes, the direction in which the sound was flipped.
  • absolute horizontal scale fixed -- The horizontal scale of the sound relative to its containing object.
  • absolute rotation angle fixed -- The rotation angle of the sound relative to its containing object. (Range: -360 to 360)
  • absolute shear angle fixed -- The skewing angle of the sound relative to its containing object. (Range: -360 to 360)
  • absolute vertical scale fixed -- The vertical scale of the sound relative to its containing object.
  • all graphics (r/o) list of graphic -- Lists all graphics contained by the sound.
  • all page items (r/o) list of page item -- Lists all page items contained by the sound.
  • applied object style object style -- The object style applied to the sound.
  • associated XML element (r/o) XML item -- The XML element associated with the sound.
  • content transparency settings (r/o) content transparency setting -- Transparency settings for the content of the sound.
  • description string -- The description of the sound.
  • do not print poster boolean -- If true, the sound poster does not print with the document.
  • embed in PDF boolean -- If true, the file is embedded in the PDF. If false, the file is linked to the PDF. Note: To embed movie files, acrobat compatibility must be acrobat 6 or higher.
  • end join miter end join / round end join / bevel end join -- The corner join applied to the sound.
  • file path string -- The file path (colon delimited on the Mac OS). Can also accept: alias or string.
  • fill color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the fill of the sound. . Can also accept: string.
  • fill tint fixed -- The percent of tint to use in the sound's fill color. (To specify a tint percent, use a number in the range of 0 to 100; to use the inherited or overridden value, use -1.)
  • fill transparency settings (r/o) fill transparency setting -- Transparency settings for the fill applied to the sound.
  • geometric bounds list of number | list of string -- The bounds of the sound excluding the stroke width, in the format [y1, x1, y2, x2], which give the coordinates of the top-left and bottom-right corners of the bounding box.
  • gradient fill angle fixed -- The angle of a linear gradient applied to the fill of the sound. (Range: -180 to 180)
  • gradient fill length number | string -- The length (for a linear gradient) or radius (for a radial gradient) applied to the fill of the sound.
  • gradient fill start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the fill of the sound, in the format [x, y].
  • gradient stroke angle fixed -- The angle of a linear gradient applied to the stroke of the sound. (Range: -180 to 180)
  • gradient stroke length number | string -- The length (for a linear gradient) or radius (for a radial gradient) applied to the stroke of the sound.
  • gradient stroke start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the stroke of the sound, in the format [x, y].
  • horizontal scale fixed -- The horizontal scaling applied to the sound.
  • id (r/o) integer -- The unique ID of the sound.
  • index (r/o) integer -- The index of the sound within its containing object.
  • item layer layer -- The layer that the sound is on.
  • item link (r/o) link -- The source file of the link.
  • local display setting high quality / typical / optimized / default -- Display performance options for the sound.
  • locked boolean -- If true, the sound is locked.
  • name string -- The name of the sound.
  • nonprinting boolean -- If true, the sound does not print.
  • object reference (r/o) sound -- An object reference for the object.
  • overprint fill boolean -- If true, the sound's fill color overprints any underlying objects. If false, the fill color knocks out the underlying colors.
  • overprint stroke boolean -- If true, the sound's stroke color overprints any underlying objects. If false, the stroke color knocks out the underlying colors.
  • parent (r/o) reference -- The parent of the sound (a spline item, polygon, graphic line, rectangle or oval). Can return: spline item, polygon, graphic line, rectangle or oval.
  • play on page turn boolean -- If true, the movie plays automatically when a user views the page that contains the movie poster in the PDF document.
  • poster file string -- The poster file.
  • rotation angle fixed -- The rotatation angle of the sound. (Range: -360 to 360)
  • shear angle fixed -- The skewing angle applied to the sound. (Range: -360 to 360)
  • sound loop boolean -- If true, sound loops forever (SWF only).
  • sound poster type none / standard / proxy image -- The type of sound poster.
  • stop on page turn boolean -- If true, sounds stops playing when its page turns.
  • stroke alignment center alignment / inside alignment / outside alignment -- The stroke alignment applied to the sound.
  • stroke color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the stroke of the sound. Can also accept: string.
  • stroke corner adjustment none / dashes / gaps / dashes and gaps -- The corner adjustment applied to the sound.
  • stroke dash and gap list of number | list of string -- The dash and gap measurements that define the pattern of a custom dashed line. Define up to six values (in points) in the format [dash1, gap1, dash2, gap2, dash3, gap3].
  • stroke weight number | string -- The weight (in points) to apply to the sound's stroke.
  • vertical scale fixed -- The vertical scaling applied to the sound.
  • visible boolean -- If true, the sound is visible.
  • visible bounds list of number | list of string -- The bounds of the sound including the stroke width, in the format [y1, x1, y2, x2], which give the coordinates of the top-left and bottom-right corners of the bounding box.
  • after attribute changed (r/o) string -- Dispatched when the value of a property changes on this sound. This event bubbles. This event is not cancelable.
  • after place (r/o) string -- Dispatched after a sound is placed. This event bubbles. This event is not cancelable.
  • before place (r/o) string -- Dispatched before a sound is placed. This event bubbles. This event is cancelable.
