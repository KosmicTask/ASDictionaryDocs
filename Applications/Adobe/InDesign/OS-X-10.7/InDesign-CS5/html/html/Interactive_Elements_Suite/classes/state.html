Class state

expand

Description

A state (states define the display of the field in an exported PDF or SWF).

  • Plural name:
    • states
  • Defined in:
    • Interactive Elements Suite

Properties

  • active boolean -- If true, the state is active in the exported PDF.
  • class (r/o) type class -- The class descriptor type.
  • enabled boolean -- If true, the state is enabled in PDF documents.
  • id (r/o) integer -- The unique ID of the state.
  • index (r/o) integer -- The index of the state within its containing object.
  • name string -- The name of the state.
  • object reference (r/o) state -- An object reference for the object.
  • parent (r/o) reference -- The parent of the state (a button or multi state object). Can return: button or multi state object.
  • properties record -- A property that allows setting of several properties at the same time.
