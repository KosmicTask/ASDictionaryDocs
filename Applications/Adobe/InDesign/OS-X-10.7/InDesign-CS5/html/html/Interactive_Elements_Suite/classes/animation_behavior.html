Description

A animation behavior.

  • Inherits from:

Properties

  • animated page item page item -- The animation page item.
  • auto reverse on roll off boolean -- If true, will automatically play the animation in reverse on roll off of the rollover event.
  • id (r/o) integer -- The unique ID of the animation behavior.
  • index (r/o) integer -- The index of the animation behavior within its containing object.
  • name (r/o) string -- The name of the animation behavior.
  • object reference (r/o) animation behavior -- An object reference for the object.
  • operation play / stop / pause / resume / reverse playback / stop all -- The playback mode.
  • parent (r/o) button -- The parent of the animation behavior (a button).
