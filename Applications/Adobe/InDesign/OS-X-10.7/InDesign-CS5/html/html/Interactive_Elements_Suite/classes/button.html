Properties

  • absolute flip none / horizontal / vertical / horizontal and vertical / both -- Indicates whether the button has been flipped independently of its parent object and, if yes, the direction in which the button was flipped.
  • absolute horizontal scale fixed -- The horizontal scale of the button relative to its containing object.
  • absolute rotation angle fixed -- The rotation angle of the button relative to its containing object. (Range: -360 to 360)
  • absolute shear angle fixed -- The skewing angle of the button relative to its containing object. (Range: -360 to 360)
  • absolute vertical scale fixed -- The vertical scale of the button relative to its containing object.
  • all graphics (r/o) list of graphic -- Lists all graphics contained by the button.
  • all page items (r/o) list of page item -- Lists all page items contained by the button.
  • applied object style object style -- The object style applied to the button.
  • associated XML element (r/o) XML item -- The XML element associated with the button.
  • content transparency settings (r/o) content transparency setting -- Transparency settings for the content of the button.
  • description string -- The description of the button.
  • end join miter end join / round end join / bevel end join -- The corner join applied to the button.
  • fill color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the fill of the button. . Can also accept: string.
  • fill tint fixed -- The percent of tint to use in the button's fill color. (To specify a tint percent, use a number in the range of 0 to 100; to use the inherited or overridden value, use -1.)
  • fill transparency settings (r/o) fill transparency setting -- Transparency settings for the fill applied to the button.
  • geometric bounds list of number | list of string -- The bounds of the button excluding the stroke width, in the format [y1, x1, y2, x2], which give the coordinates of the top-left and bottom-right corners of the bounding box.
  • gradient fill angle fixed -- The angle of a linear gradient applied to the fill of the button. (Range: -180 to 180)
  • gradient fill length number | string -- The length (for a linear gradient) or radius (for a radial gradient) applied to the fill of the button.
  • gradient fill start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the fill of the button, in the format [x, y].
  • gradient stroke angle fixed -- The angle of a linear gradient applied to the stroke of the button. (Range: -180 to 180)
  • gradient stroke length number | string -- The length (for a linear gradient) or radius (for a radial gradient) applied to the stroke of the button.
  • gradient stroke start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the stroke of the button, in the format [x, y].
  • horizontal scale fixed -- The horizontal scaling applied to the button.
  • id (r/o) integer -- The unique ID of the button.
  • index (r/o) integer -- The index of the button within its containing object.
  • item layer layer -- The layer that the button is on.
  • local display setting high quality / typical / optimized / default -- Display performance options for the button.
  • locked boolean -- If true, the button is locked.
  • name string -- The name of the button.
  • nonprinting boolean -- If true, the button does not print.
  • object reference (r/o) button -- An object reference for the object.
  • overprint fill boolean -- If true, the button's fill color overprints any underlying objects. If false, the fill color knocks out the underlying colors.
  • overprint stroke boolean -- If true, the button's stroke color overprints any underlying objects. If false, the stroke color knocks out the underlying colors.
  • parent (r/o) reference -- The parent of the button (a place gun, spread, master spread, spline item, polygon, graphic line, rectangle, oval, group, character or snippet). Can return: place gun, spread, master spread, spline item, polygon, graphic line, rectangle, oval, group, c...
  • rotation angle fixed -- The rotatation angle of the button. (Range: -360 to 360)
  • shear angle fixed -- The skewing angle applied to the button. (Range: -360 to 360)
  • stroke alignment center alignment / inside alignment / outside alignment -- The stroke alignment applied to the button.
  • stroke color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the stroke of the button. Can also accept: string.
  • stroke corner adjustment none / dashes / gaps / dashes and gaps -- The corner adjustment applied to the button.
  • stroke dash and gap list of number | list of string -- The dash and gap measurements that define the pattern of a custom dashed line. Define up to six values (in points) in the format [dash1, gap1, dash2, gap2, dash3, gap3].
  • stroke weight number | string -- The weight (in points) to apply to the button's stroke.
  • vertical scale fixed -- The vertical scaling applied to the button.
  • visibility in pdf visible in pdf / hidden in pdf / visible but does not print in pdf / hidden but printable in pdf -- The field's visibility in the PDF document.
  • visible boolean -- If true, the button is visible.
  • visible bounds list of number | list of string -- The bounds of the button including the stroke width, in the format [y1, x1, y2, x2], which give the coordinates of the top-left and bottom-right corners of the bounding box.
  • after place (r/o) string -- Dispatched after a button is placed. This event bubbles. This event is not cancelable.
  • before place (r/o) string -- Dispatched before a button is placed. This event bubbles. This event is cancelable.
