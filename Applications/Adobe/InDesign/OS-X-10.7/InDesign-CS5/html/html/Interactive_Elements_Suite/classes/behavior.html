Description

A behavior object.

  • Plural name:
    • behaviors
  • Defined in:

Properties

  • behavior event mouse up / mouse down / mouse enter / mouse exit / on focus / on blur -- The event which triggers the behavior.
  • class (r/o) type class -- The class descriptor type.
  • enable behavior boolean -- If true, the behavior is enabled.
  • id (r/o) integer -- The unique ID of the behavior.
  • index (r/o) integer -- The index of the behavior within its containing object.
  • label string -- A property that can be set to any string.
  • name (r/o) string -- The name of the behavior.
  • object reference (r/o) behavior -- An object reference for the object.
  • parent (r/o) button -- The parent of the behavior (a button).
  • properties record -- A property that allows setting of several properties at the same time.
