Command convert shape

Description

Converts the object to a different shape.

  • Defined in:
    • Interactive Elements Suite

Parameters

  • reference -- Supported by the following objects: form field, button, multi state object, movie, sound.
  • given convert to rectangle / convert to rounded rectangle / convert to beveled rectangle / convert to inverse rounded rectangle / convert to oval / convert to triangle / convert to polygon / convert to line / convert to straight line / convert to open path / convert to closed path -- The object's new shape.
  • [number of sides integer] -- The number of sides for the resulting polygon. (Range: 3 to 100). Can accept: integer (3 - 100).
  • [inset percentage integer] -- The star inset percentage for the resulting polygon. (Range: 0.0 to 100.0) . Can accept: integer (0 - 100).
  • [corner radius number | string] -- The corner radius of the resulting rectangle. Can accept: number or string (0 - 800 points).
