Description

a timing target.

Properties

  • class (r/o) type class -- The class descriptor type.
  • delay seconds fixed -- The time delay in seconds for a single target or a group of targets after the previoius group has finished. Can return: fixed (0 - 60).
  • dynamic target reference -- A page item target that is an animation, media, or mso. Can return: page item, graphic, behavior or media item.
  • index (r/o) integer -- The index of the timing target within its containing object.
  • object reference (r/o) timing target -- An object reference for the object.
  • parent (r/o) timing group -- The parent of the timing target (a timing group).
  • properties record -- A property that allows setting of several properties at the same time.
  • reverse animation boolean -- Set to true if reversing animation on rolloff. Valid only for self rolloff trigger event.
