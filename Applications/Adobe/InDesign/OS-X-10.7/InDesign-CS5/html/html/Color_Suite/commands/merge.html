Command merge

Description

Merges the specified swatches with the swatch.

  • Defined in:
    • Color Suite

Parameters

  • reference -- Supported by the following objects: mixed ink, mixed ink group, gradient, swatch, color, tint.
  • with anything -- The swatches to merge. Can accept: list of swatches, swatch or string.

Result

  • swatch -- The merged swatch.
