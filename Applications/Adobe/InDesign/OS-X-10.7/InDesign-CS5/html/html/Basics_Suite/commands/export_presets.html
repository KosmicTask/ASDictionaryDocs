Command export presets

Description

Export the presets to a file.

  • Defined in:
    • Basics Suite

Parameters

  • format PDF export presets format / printer presets format / flattener presets format / document presets format -- The preset format.
  • to alias | string -- The file to export to.
  • [version comments string] -- The comments for this version.
  • [force save boolean] -- Forcibly save a version. Can accept: boolean (Default: FALSE).
