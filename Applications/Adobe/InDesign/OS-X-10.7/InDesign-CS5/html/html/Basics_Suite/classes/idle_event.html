Class idle event

expand

Description

An idle event.

  • Defined in:
    • Basics Suite
  • Inherits from:
    • event (Basics Suite)

Properties

  • id (r/o) integer -- The unique ID of the idle event.
  • index (r/o) integer -- The index of the idle event within its containing object.
  • object reference (r/o) idle event -- An object reference for the object.
  • parent (r/o) idle task -- The parent of the idle event (a idle task).
  • time allocated (r/o) integer -- Amount of time allocated to this task at the time the event was dispatched.
  • on idle (r/o) string -- Dispatched at idle time for this object. This event does not bubble. This event is not cancelable.
