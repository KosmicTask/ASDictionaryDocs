Command package UCF

Description

Package a folder into a UCF file.

  • Defined in:
    • Basics Suite

Parameters

  • source folder alias | string -- The folder to be packaged into an IDML file. Does not validate structure of the folder pursuant to the IDML spec. Caller is responsible for making sure the files in the folder are correctly organized.
  • UCF file alias | string -- The destination UCF file. Will be overwritten if it already exists.
  • [MIME media type string] -- The MIME media type, default value identifies package as IDML. Can accept: string (Default: "application/vnd.adobe.indesign-idml-package").
