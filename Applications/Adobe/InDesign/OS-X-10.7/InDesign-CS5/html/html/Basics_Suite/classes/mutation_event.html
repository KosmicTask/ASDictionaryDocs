Class mutation event

expand

Description

A mutation event is dispatched for attribute changes.

  • Defined in:
    • Basics Suite
  • Inherits from:
    • event (Basics Suite)

Properties

  • attribute name (r/o) string -- The name of the property that changed.
  • attribute value (r/o) anything -- The current value of the property that changed.
  • id (r/o) integer -- The unique ID of the mutation event.
  • index (r/o) integer -- The index of the mutation event within its containing object.
  • object reference (r/o) mutation event -- An object reference for the object.
  • parent (r/o) reference -- The parent of the mutation event (a object, UID based object, cell style group, cell style, table style group, condition set, hidden text, condition, Motion Preset, assigned story, assignment, object style group, object style, numbering list, snippet, ...
  • after attribute changed (r/o) string -- Dispatched when the value of a property changes on this object. This event bubbles. This event is not cancelable.
