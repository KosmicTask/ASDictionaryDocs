Command import styles

Description

Imports the specified styles.

  • Defined in:
    • Basics Suite

Parameters

  • reference -- Supported by the following objects: application, document.
  • format character styles format / paragraph styles format / text styles format / TOC styles format / object styles format / stroke styles format / table styles format / cell styles format / table and cell styles format -- The types of styles to import.
  • from alias | string -- The file containing the styles you want to import.
  • [global strategy load all with overwrite / load all with rename / do not load the style] -- The resolution strategy to employ for imported styles that have the same names as existing styles. Can accept: load all with overwrite/load all with rename/do not load the style (Default: load all with overwrite).
