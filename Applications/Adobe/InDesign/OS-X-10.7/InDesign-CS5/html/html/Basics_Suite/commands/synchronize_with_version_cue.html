Command synchronize with version cue

Description

Synchronizes the file with the Version Cue project.

  • Defined in:
    • Basics Suite

Parameters

  • document
  • [sync conflict resolution skip conflicts / ask about conflicts / prefer local / prefer project] -- The conflict resolution method to use during synchronization. Can accept: skip conflicts/ask about conflicts/prefer local/prefer project (Default: skip conflicts).
  • [version comments string] -- The comments that describe the version.

Result

  • file downloaded / file uploaded / file unlocked / file skipped / file no change -- The synchronization status.
