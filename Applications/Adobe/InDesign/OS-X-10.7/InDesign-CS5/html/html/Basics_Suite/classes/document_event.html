  • full name (r/o) alias | string -- The full path to the document event, including the name of the document event.
  • id (r/o) integer -- The unique ID of the document event.
  • index (r/o) integer -- The index of the document event within its containing object.
  • object reference (r/o) document event -- An object reference for the object.
  • parent (r/o) reference -- The parent of the document event (a application or document). Can return: application or document.
  • user interaction level never interact / interact with all / interact with alerts -- Controls the display of dialogs and alerts during script processing.
  • after close (r/o) string -- Dispatched after a Document is closed. This event bubbles. This event is not cancelable.
