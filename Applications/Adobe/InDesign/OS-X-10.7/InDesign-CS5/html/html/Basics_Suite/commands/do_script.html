Command do script

Description

Executes the script in the specified language as a single transaction.

  • Defined in:
    • Basics Suite

Parameters

  • anything -- The script to execute.
  • [language unknown / javascript / applescript language] -- The language of the script to execute. If not specified, uses the language used to call this method. Can accept: unknown/javascript/applescript language (Default: unknown).
  • [with arguments anything] -- An array of arguments passed to the script.
  • [undo mode script request / entire script / auto undo / fast entire script] -- How to undo this script. Can accept: script request/entire script/auto undo/fast entire script (Default: script request).
  • [undo name string] -- The name of the undo step for entire script undo mode. Can accept: string (Default: "Script").

Result

  • anything -- The return value of the script.
