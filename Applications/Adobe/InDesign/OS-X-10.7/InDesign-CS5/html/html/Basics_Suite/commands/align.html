Command align

Description

Align page items.

  • Defined in:
    • Basics Suite

Parameters

  • document
  • align distribute items page item -- The page items to align or distribute.
  • align option left edges / top edges / right edges / bottom edges / horizontal centers / vertical centers -- The type of alignment to apply.
  • [align distribute bounds item bounds / page bounds / margin bounds / spread bounds] -- The bounds within which to align or distribute the page items.
