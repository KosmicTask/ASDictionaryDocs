Class import export event

expand

Description

An import or export event.

  • Defined in:
    • Basics Suite
  • Inherits from:
    • event (Basics Suite)

Properties

  • format (r/o) string -- The import/export file format.
  • full name (r/o) alias | string -- The full path to the import export event, including the name of the import export event.
  • id (r/o) integer -- The unique ID of the import export event.
  • index (r/o) integer -- The index of the import export event within its containing object.
  • object reference (r/o) import export event -- An object reference for the object.
  • parent (r/o) reference -- The parent of the import export event (a application or document). Can return: application or document.
  • user interaction level never interact / interact with all / interact with alerts -- Controls the display of dialogs and alerts during script processing.
  • after export (r/o) string -- Dispatched after a object is exported. This event bubbles. This event is not cancelable.
  • after import (r/o) string -- Dispatched after importing a file into a object. This event bubbles. This event is not cancelable.
  • before export (r/o) string -- Dispatched before a object is exported. This event bubbles. This event is cancelable.
  • before import (r/o) string -- Dispatched before importing a file into a object. This event bubbles. This event is cancelable.
  • failed export (r/o) string -- Dispatched after a object export is canceled or fails. This event bubbles. This event is not cancelable.
