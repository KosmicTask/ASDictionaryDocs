Command save a copy

Description

Saves a copy of the document.

  • Defined in:
    • Basics Suite

Parameters

  • document
  • [to alias | string] -- The file path for the copy. Note: Leaves the original document open and does not open the copy.
  • [stationery boolean] -- If true, saves the file as stationery (Mac OS) or as a template (Windows). Note: The file extension for stationery and templates is different than the extension for regular files. Can accept: boolean (Default: FALSE).
