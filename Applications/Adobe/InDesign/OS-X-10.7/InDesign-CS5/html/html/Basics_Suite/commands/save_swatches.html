Command save swatches

Description

Saves the specified swatch(es) to a swatchbook file.

  • Defined in:
    • Basics Suite

Parameters

  • reference -- Supported by the following objects: application, document.
  • to alias | string -- The swatchbook file to save to.
  • swatch list swatch -- The swatch(es) to save.
  • [version comments string] -- The comment for this version.
  • [force save boolean] -- If true, forcibly saves a version. Can accept: boolean (Default: FALSE).
