Class event

expand

Description

An event.

  • Plural name:
    • events
  • Defined in:
    • Basics Suite

Properties

  • bubbles (r/o) boolean -- If true, the event supports the bubbling phase of propagation.
  • cancelable (r/o) boolean -- If true, the default behavior of the event on its target can be canceled.
  • class (r/o) type class -- The class descriptor type.
  • current target (r/o) reference -- The current propagation target of the event.
  • default prevented (r/o) boolean -- If true, the default behavior of the event on its target has been canceled.
  • event phase (r/o) not dispatching / at target / bubbling phase / done -- The current propagation phase of the event.
  • event type (r/o) string -- The name of the event.
  • id (r/o) integer -- The unique ID of the event.
  • index (r/o) integer -- The index of the event within its containing object.
  • object reference (r/o) event -- An object reference for the object.
  • parent (r/o) reference -- The parent of the event (a object, UID based object, cell style group, cell style, table style group, condition set, hidden text, condition, Motion Preset, assigned story, assignment, object style group, object style, numbering list, snippet, dialog, s...
  • propagation stopped (r/o) boolean -- If true, propagation of the event beyond the current target has been stopped.
  • properties record -- A property that allows setting of several properties at the same time.
  • target (r/o) reference -- The target of the event.
  • time stamp (r/o) date -- The time the event was initialized.
  • after activate (r/o) string -- Dispatched after the object becomes active. This event bubbles. This event is not cancelable.
  • after close (r/o) string -- Dispatched when a object is closing. Since the close has been committed, it can no longer be canceled. This event bubbles. This event is not cancelable.
  • after context changed (r/o) string -- Dispatched after the active context changes. This event bubbles. This event is not cancelable.
  • after delete (r/o) string -- Dispatched after a object is deleted. This event bubbles. This event is not cancelable.
  • after embed (r/o) string -- Dispatched after a object is embedded. This event bubbles. This event is not cancelable.
  • after invoke (r/o) string -- Dispatched after the object is invoked. This event does not bubble. This event is not cancelable.
  • after links changed (r/o) string -- Dispatched after one or more links in the object have been added, deleted, or modified. This event bubbles. This event is not cancelable.
  • after move (r/o) string -- Dispatched after a object is relocated from one object to another. This event bubbles. This event is not cancelable.
  • after new (r/o) string -- Dispatched after a object is created. This event bubbles. This event is not cancelable.
  • after open (r/o) string -- Dispatched after a object is opened. This event bubbles. This event is not cancelable.
  • after place (r/o) string -- Dispatched after a object is placed. This event bubbles. This event is not cancelable.
  • after quit (r/o) string -- Dispatched when the object is quitting. Since the quit has been committed, it can not be canceled. This event bubbles. This event is not cancelable.
  • after selection attribute changed (r/o) string -- Dispatched after an attribute on the active selection changes. This event bubbles. This event is not cancelable.
  • after selection changed (r/o) string -- Dispatched after the active selection changes. This event bubbles. This event is not cancelable.
  • after unembed (r/o) string -- Dispatched after a object is unembedded. This event bubbles. This event is not cancelable.
  • after update (r/o) string -- Dispatched after a object is updated. This event bubbles. This event is not cancelable.
  • before close (r/o) string -- Dispatched before a object is closed. This event bubbles. This event is cancelable.
  • before deactivate (r/o) string -- Dispatched before the object becomes inactive. This event bubbles. This event is not cancelable.
  • before delete (r/o) string -- Dispatched before a object is deleted. This event bubbles. This event is cancelable.
  • before display (r/o) string -- Dispatched before the object is displayed. This event does not bubble. This event is not cancelable.
  • before embed (r/o) string -- Dispatched before a object is embedded. This event bubbles. This event is cancelable.
  • before invoke (r/o) string -- Dispatched before the object is invoked. This event does not bubble. This event is cancelable.
  • before move (r/o) string -- Dispatched before a object is relocated from one object to another. This event bubbles. This event is cancelable.
  • before place (r/o) string -- Dispatched before a object is placed. This event bubbles. This event is cancelable.
  • before quit (r/o) string -- Dispatched before the object is quit. Allows the quit to be canceled. This event bubbles. This event is cancelable.
  • before unembed (r/o) string -- Dispatched before a object is unembedded. This event bubbles. This event is cancelable.
  • before update (r/o) string -- Dispatched before a object is updated. This event bubbles. This event is cancelable.
  • on invoke (r/o) string -- Dispatched when the object is invoked. This event does not bubble. This event is not cancelable.
