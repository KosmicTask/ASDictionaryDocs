Class layout window

expand

Description

A layout window.

  • Plural name:
    • layout windows
  • Defined in:
    • Basics Suite
  • Inherits from:
    • window (Basics Suite)

Properties

  • active layer layer -- The active layer. Can also accept: string.
  • active page page -- The front-most page.
  • active spread reference -- The front-most spread. Can return: spread or master spread.
  • bounds list of integer -- The bounds of the window (specified in pixels) in the form [top, left, bottom, right].
  • index (r/o) integer -- The index of the layout window within its containing object.
  • name (r/o) string -- The name of the layout window.
  • object reference (r/o) layout window -- An object reference for the object.
  • overprint preview boolean -- If true, display a simulation of overprinting.
  • parent (r/o) document -- The parent of the layout window (a document).
  • preserve color numbers boolean -- If true, leaves color values unchanged for CMYK objects without embedded profiles and native objects such as line art or type. Note: Converts images whose profiles differ from the profile of the simulated device. Valid only when proofing type is custom.
  • proofing profile string -- The profile used for proofing colors.
  • proofing type proof off / document CMYK / working CMYK / custom -- The method of proofing colors.
  • screen mode preview off / preview to page / preview to bleed / preview to slug / presentation preview -- The screen mode for layout view.
  • selection list of reference -- The selected object(s). Can also accept: object or nothing.
  • simulate ink black boolean -- If true, simulates the dark gray produced by many printers in place of solid black, according to the proofing profile. Note: Valid only when proofing type is custom.
  • simulate paper white boolean -- If true, simulates the dingy white of real paper, according to the proofing profile. Note: Valid only when proofing type is custom.
  • transform reference point anything -- The default anchor point around which to transform objects. Can return: top left anchor/top center anchor/top right anchor/left center anchor/center anchor/right center anchor/bottom left anchor/bottom center anchor/bottom right anchor or list of 2 num...
  • view display setting high quality / typical / optimized -- The display performance preferences override for the view.
  • zoom percentage fixed -- The size (as a percentage) to which to enlarge or reduce the view of the document. (Range: 5 to 4000). Can return: fixed (5 - 4000).
  • after activate (r/o) string -- Dispatched after the layout window becomes active. This event bubbles. This event is not cancelable.
  • after attribute changed (r/o) string -- Dispatched when the value of a property changes on this layout window. This event bubbles. This event is not cancelable.
  • after close (r/o) string -- Dispatched when a layout window is closing. Since the close has been committed, it can no longer be canceled. This event bubbles. This event is not cancelable.
  • after open (r/o) string -- Dispatched after a layout window is opened. This event bubbles. This event is not cancelable.
  • before close (r/o) string -- Dispatched before a layout window is closed. This event bubbles. This event is cancelable.
  • before deactivate (r/o) string -- Dispatched before the layout window becomes inactive. This event bubbles. This event is not cancelable.
