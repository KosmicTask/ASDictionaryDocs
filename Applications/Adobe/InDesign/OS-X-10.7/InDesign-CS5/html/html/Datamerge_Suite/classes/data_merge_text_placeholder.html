Class data merge text placeholder

expand

Description

An object that can contain a data merge text field.

  • Plural name:
    • data merge text placeholders
  • Defined in:
    • Datamerge Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • field (r/o) data merge field -- The data merge field to insert in the placeholder. For information, see data merge text placeholder and data merge image placeholder.
  • index (r/o) integer -- The index of the data merge text placeholder within its containing object.
  • length (r/o) integer -- The number of objects in the collection.
  • object reference (r/o) data merge text placeholder -- An object reference for the object.
  • parent (r/o) document -- The parent of the data merge text placeholder (a document).
  • parent story (r/o) story -- The story that contains the text.
  • properties record -- A property that allows setting of several properties at the same time.
  • story offset (r/o) insertion point -- The insertion point before the table in the story containing the table.
