Command create guides

Description

Creates multiple guides on all pages of the spread.

  • Defined in:
    • Layout Suite

Parameters

  • reference -- Supported by the following objects: spread, master spread.
  • [number of rows integer] -- The number of rows to create on each page. Can accept: integer (0 - 40) (Default: 0).
  • [number of columns integer] -- The number of columns to create on each page. . Can accept: integer (0 - 40) (Default: 0).
  • [row gutter number | string] -- The height of the gutter between rows. Can accept: number or string (0 - 1440 points) (Default: 0.0 points).
  • [column gutter number | string] -- The width of the gutter between columns. Can accept: number or string (0 - 1440 points) (Default: 0.0 points).
  • [guide color anything] -- The color to make the guides, specified either as an array of three doubles, each in the range 0 to 255 and representing R, G, and B values, or as a UI color. Can accept: list of 3 fixed (0 - 255) or light blue/red/green/blue/yellow/magenta/cyan/gray/b...
  • [fit margins boolean] -- If true, the row height and column width are calculated based on the space within the page margins. If false, row height and column width are calculated based on the full page. Can accept: boolean (Default: FALSE).
  • [remove existing boolean] -- If true, removes existing guides when creating new ones. Can accept: boolean (Default: FALSE).
  • [layer layer] -- The layer on which to create the guides.
