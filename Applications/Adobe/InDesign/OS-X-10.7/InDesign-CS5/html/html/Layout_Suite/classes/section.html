Class section

expand

Description

A section.

  • Plural name:
    • sections
  • Defined in:
    • Layout Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • continue numbering boolean -- If true, continues page numbers sequentially from the previous section.
  • id (r/o) integer -- The unique ID of the section.
  • include section prefix boolean -- If true, places the specified prefix before page numbers on all pages in the section.
  • index (r/o) integer -- The index of the section within its containing object.
  • label string -- A property that can be set to any string.
  • length (r/o) integer -- The number of pages in the section.
  • marker string -- The section marker.
  • name string -- The name of the section.
  • object reference (r/o) section -- An object reference for the object.
  • page number start integer -- The page number assigned to the first page in the section. Note: Valid only when continue numbering is false. Can return: integer (1 - 999999).
  • page number style anything -- The page number style. Can return: upper roman/lower roman/upper letters/lower letters/arabic/kanji/arabic alif ba tah/arabic abjad/hebrew biblical/hebrew non standard/single leading zeros/double leading zeros/triple leading zeros or string.
  • page start page -- The start page for the section.
  • parent (r/o) document -- The parent of the section (a document).
  • properties record -- A property that allows setting of several properties at the same time.
  • section prefix string -- The prefix to place before page numbers on pages in the section. May include up to 8 characters. Note: Valid only when include section prefix is true.
