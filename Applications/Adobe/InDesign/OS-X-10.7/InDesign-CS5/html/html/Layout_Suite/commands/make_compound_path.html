Command make compound path

Description

Creates a compound path by combining the path(s) of the object with the paths of other objects.

  • Defined in:
    • Layout Suite

Parameters

  • reference -- Supported by the following objects: oval, rectangle, graphic line, polygon, spline item.
  • with page item -- The other objects whose paths to include in the new compound path.

Result

  • page item -- The new compound path.
