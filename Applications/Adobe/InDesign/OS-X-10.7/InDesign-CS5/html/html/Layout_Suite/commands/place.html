Command place

Description

Places the file.

  • Defined in:
    • Layout Suite

Parameters

  • alias | string -- The file to place. Supported by the following objects: spread, page, master spread, page item, oval, rectangle, graphic line, polygon, image, EPS, PDF, WMF, PICT, graphic, spline item, EPSText, imported page, media item.
  • [on reference] -- Where to place the file. Can accept: spread, page, master spread, page item or graphic.
  • [place point number | string] -- The point at which to place (only applicable if you are telling a page or spread to place)
  • [destination layer layer] -- The layer on which to place (only applicable if you are telling a page or spread to place)
  • [showing options boolean] -- Whether to display the import options dialog. Can accept: boolean (Default: FALSE).
  • [autoflowing boolean] -- Whether to autoflow placed text. Can accept: boolean (Default: FALSE).
  • [with properties record] -- Initial values for properties of the placed object(s)

Result

  • list of reference -- The placed file. Can return: list of page items, graphics, movies, sounds, stories or texts.
