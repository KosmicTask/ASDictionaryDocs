Command flip item

Description

Flips the object.

  • Defined in:
    • Layout Suite

Parameters

  • reference -- Supported by the following objects: page item, oval, rectangle, graphic line, polygon, group, image, EPS, PDF, WMF, PICT, graphic, spline item, EPSText, imported page, media item.
  • given none / horizontal / vertical / horizontal and vertical / both -- The axis around which to flip the object.
  • [around anything] -- The point around which to flip the object. Can accept: list of 2 numbers or strings or top left anchor/top center anchor/top right anchor/left center anchor/center anchor/right center anchor/bottom left anchor/bottom center anchor/bottom right anchor.
