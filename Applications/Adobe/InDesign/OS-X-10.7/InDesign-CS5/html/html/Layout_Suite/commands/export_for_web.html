Command export for web

Description

Exports the object for the web.

  • Defined in:
    • Layout Suite

Parameters

  • reference -- Supported by the following objects: image, EPS, PDF, WMF, PICT, graphic, imported page.
  • to alias | string -- The full path name of the exported file.

Result

  • list of string -- The file names of the exported object in the following order: copied formatted image, copied optimized image, copied original image.
