Class imported page

expand

Description

An imported InDesign page.

  • Plural name:
    • imported pages
  • Defined in:
    • Layout Suite
  • Inherits from:
    • graphic (Layout Suite) → page item (Layout Suite)

Properties

  • absolute flip none / horizontal / vertical / horizontal and vertical / both -- Indicates whether the imported page has been flipped independently of its parent object and, if yes, the direction in which the imported page was flipped.
  • absolute horizontal scale fixed -- The horizontal scale of the imported page relative to its containing object.
  • absolute rotation angle fixed -- The rotation angle of the imported page relative to its containing object. (Range: -360 to 360)
  • absolute shear angle fixed -- The skewing angle of the imported page relative to its containing object. (Range: -360 to 360)
  • absolute vertical scale fixed -- The vertical scale of the imported page relative to its containing object.
  • all graphics (r/o) list of graphic -- Lists all graphics contained by the imported page.
  • all page items (r/o) list of page item -- Lists all page items contained by the imported page.
  • applied object style object style -- The object style applied to the imported page.
  • associated XML element (r/o) XML item -- The XML element associated with the imported page.
  • clipping path (r/o) clipping path settings -- Clipping path settings.
  • content transparency settings (r/o) content transparency setting -- Transparency settings for the content of the imported page.
  • end join miter end join / round end join / bevel end join -- The corner join applied to the imported page.
  • fill color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the fill of the imported page. . Can also accept: string.
  • fill tint fixed -- The percent of tint to use in the imported page's fill color. (To specify a tint percent, use a number in the range of 0 to 100; to use the inherited or overridden value, use -1.)
  • fill transparency settings (r/o) fill transparency setting -- Transparency settings for the fill applied to the imported page.
  • geometric bounds list of number | list of string -- The bounds of the imported page excluding the stroke width, in the format [y1, x1, y2, x2], which give the coordinates of the top-left and bottom-right corners of the bounding box.
  • gradient fill angle fixed -- The angle of a linear gradient applied to the fill of the imported page. (Range: -180 to 180)
  • gradient fill length number | string -- The length (for a linear gradient) or radius (for a radial gradient) applied to the fill of the imported page.
  • gradient fill start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the fill of the imported page, in the format [x, y].
  • gradient stroke angle fixed -- The angle of a linear gradient applied to the stroke of the imported page. (Range: -180 to 180)
  • gradient stroke length number | string -- The length (for a linear gradient) or radius (for a radial gradient) applied to the stroke of the imported page.
  • gradient stroke start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the stroke of the imported page, in the format [x, y].
  • graphic layer options (r/o) graphic layer option -- Graphic layer option settings.
  • horizontal scale fixed -- The horizontal scaling applied to the imported page.
  • id (r/o) integer -- The unique ID of the imported page.
  • imported page crop (r/o) crop content / crop bleed / crop slug -- Specifies the cropping of the imported InDesign page. Read only for page items.
  • index (r/o) integer -- The index of the imported page within its containing object.
  • item layer layer -- The layer that the imported page is on.
  • local display setting high quality / typical / optimized / default -- Display performance options for the imported page.
  • locked boolean -- If true, the imported page is locked.
  • name string -- The name of the imported page.
  • nonprinting boolean -- If true, the imported page does not print.
  • object reference (r/o) imported page -- An object reference for the object.
  • overprint fill boolean -- If true, the imported page's fill color overprints any underlying objects. If false, the fill color knocks out the underlying colors.
  • overprint stroke boolean -- If true, the imported page's stroke color overprints any underlying objects. If false, the stroke color knocks out the underlying colors.
  • page number (r/o) integer -- Which page of the InDesign document should be imported. Read only for page items.
  • parent (r/o) reference -- The parent of the imported page (a place gun, spline item, polygon, graphic line, rectangle or oval). Can return: place gun, spline item, polygon, graphic line, rectangle or oval.
  • PDF attributes (r/o) PDF attribute -- PDF attribute settings.
  • rotation angle fixed -- The rotatation angle of the imported page. (Range: -360 to 360)
  • shear angle fixed -- The skewing angle applied to the imported page. (Range: -360 to 360)
  • stroke alignment center alignment / inside alignment / outside alignment -- The stroke alignment applied to the imported page.
  • stroke color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the stroke of the imported page. Can also accept: string.
  • stroke corner adjustment none / dashes / gaps / dashes and gaps -- The corner adjustment applied to the imported page.
  • stroke dash and gap list of number | list of string -- The dash and gap measurements that define the pattern of a custom dashed line. Define up to six values (in points) in the format [dash1, gap1, dash2, gap2, dash3, gap3].
  • stroke weight number | string -- The weight (in points) to apply to the imported page's stroke.
  • vertical scale fixed -- The vertical scaling applied to the imported page.
  • visible boolean -- If true, the imported page is visible.
  • visible bounds list of number | list of string -- The bounds of the imported page including the stroke width, in the format [y1, x1, y2, x2], which give the coordinates of the top-left and bottom-right corners of the bounding box.
  • after place (r/o) string -- Dispatched after a imported page is placed. This event bubbles. This event is not cancelable.
  • before place (r/o) string -- Dispatched before a imported page is placed. This event bubbles. This event is cancelable.
