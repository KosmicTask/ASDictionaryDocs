Command intersect path

Description

Creates a new page item by intersecting the object with other objects. Returns an error if the objects do not intersect.

  • Defined in:
    • Layout Suite

Parameters

  • reference -- Supported by the following objects: oval, rectangle, graphic line, polygon, spline item.
  • with page item -- The object(s) with which to intersect.

Result

  • page item -- The new page item.
