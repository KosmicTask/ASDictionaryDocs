Class page

expand

Description

A page.

  • Plural name:
    • pages
  • Defined in:
    • Layout Suite

Properties

  • all graphics (r/o) list of graphic -- Lists all graphics contained by the page.
  • all page items (r/o) list of page item -- Lists all page items contained by the page.
  • applied master master spread -- The master spread applied to the page. Can also accept: nothing.
  • applied section (r/o) section -- The section to which the page belongs.
  • applied trap preset trap preset -- The trap preset applied to the page. Can also accept: string.
  • bounds (r/o) list of number | list of string -- The bounds of the page, in the format [y1, x1, y2, x2].
  • class (r/o) type class -- The class descriptor type.
  • document offset (r/o) integer -- The sequential number of the page within the document.
  • grid data (r/o) grid data information -- Default grid properties. Note: Applies to named, layout, and frame (story) grids.
  • id (r/o) integer -- The unique ID of the page.
  • index (r/o) integer -- The index of the page within its containing object.
  • label string -- A property that can be set to any string.
  • margin preferences (r/o) margin preference -- Margin preference settings.
  • master page items (r/o) list of reference -- The items on a specified document page that originated on the applied master page and have not been overridden or detached. Can return: list of page items, guides, graphics, movies or sounds.
  • master page transform transformation matrix -- The transform applied to the master page before it is applied to page.
  • name (r/o) string -- The name of the page.
  • object reference (r/o) page -- An object reference for the object.
  • page color anything -- The color label of the page, specified either as an array of three doubles, each in the range 0 to 255 and representing R, G, and B values, or as a UI color. Can return: list of 3 fixed (0 - 255) or light blue/red/green/blue/yellow/magenta/cyan/gray/b...
  • parent (r/o) reference -- The parent of the page (a spread or master spread). Can return: spread or master spread.
  • properties record -- A property that allows setting of several properties at the same time.
  • side (r/o) right hand / left hand / single sided -- The side of the binding spine on which to place the page within the spread.
  • tab order list of button -- The order in which the focus moves to different buttons in the PDF when the tab key is pressed.
