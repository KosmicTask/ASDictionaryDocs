Command store

Description

Stores the object in the specified library.

  • Defined in:
    • Layout Suite

Parameters

  • reference -- Supported by the following objects: page item, oval, rectangle, graphic line, polygon, group, image, EPS, PDF, WMF, PICT, graphic, spline item, EPSText, imported page, media item.
  • using library -- The library in which to store the object.
  • [with properties record] -- Initial values for properties of the new object

Result

  • asset -- The new library asset(s).
