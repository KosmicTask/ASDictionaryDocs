Class rectangle

expand

Description

A rectangle.

  • Plural name:
    • rectangles
  • Defined in:
    • Layout Suite
  • Inherits from:
    • spline item (Layout Suite) → page item (Layout Suite)

Properties

  • absolute flip none / horizontal / vertical / horizontal and vertical / both -- Indicates whether the rectangle has been flipped independently of its parent object and, if yes, the direction in which the rectangle was flipped.
  • absolute horizontal scale fixed -- The horizontal scale of the rectangle relative to its containing object.
  • absolute rotation angle fixed -- The rotation angle of the rectangle relative to its containing object. (Range: -360 to 360)
  • absolute shear angle fixed -- The skewing angle of the rectangle relative to its containing object. (Range: -360 to 360)
  • absolute vertical scale fixed -- The vertical scale of the rectangle relative to its containing object.
  • all graphics (r/o) list of graphic -- Lists all graphics contained by the rectangle.
  • all page items (r/o) list of page item -- Lists all page items contained by the rectangle.
  • applied object style object style -- The object style applied to the rectangle.
  • associated XML element (r/o) XML item -- The XML element associated with the rectangle.
  • content transparency settings (r/o) content transparency setting -- Transparency settings for the content of the rectangle.
  • end join miter end join / round end join / bevel end join -- The corner join applied to the rectangle.
  • fill color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the fill of the rectangle. . Can also accept: string.
  • fill tint fixed -- The percent of tint to use in the rectangle's fill color. (To specify a tint percent, use a number in the range of 0 to 100; to use the inherited or overridden value, use -1.)
  • fill transparency settings (r/o) fill transparency setting -- Transparency settings for the fill applied to the rectangle.
  • frame fitting options (r/o) frame fitting option -- The frame fitting option to apply to placed or pasted content. Can be applied to a frame, object style, or document or to the application.
  • geometric bounds list of number | list of string -- The bounds of the rectangle excluding the stroke width, in the format [y1, x1, y2, x2], which give the coordinates of the top-left and bottom-right corners of the bounding box.
  • gradient fill angle fixed -- The angle of a linear gradient applied to the fill of the rectangle. (Range: -180 to 180)
  • gradient fill length number | string -- The length (for a linear gradient) or radius (for a radial gradient) applied to the fill of the rectangle.
  • gradient fill start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the fill of the rectangle, in the format [x, y].
  • gradient stroke angle fixed -- The angle of a linear gradient applied to the stroke of the rectangle. (Range: -180 to 180)
  • gradient stroke length number | string -- The length (for a linear gradient) or radius (for a radial gradient) applied to the stroke of the rectangle.
  • gradient stroke start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the stroke of the rectangle, in the format [x, y].
  • horizontal scale fixed -- The horizontal scaling applied to the rectangle.
  • id (r/o) integer -- The unique ID of the rectangle.
  • InCopy export options (r/o) InCopy export option -- Export options for InCopy INCX document format.
  • index (r/o) integer -- The index of the rectangle within its containing object.
  • item layer layer -- The layer that the rectangle is on.
  • local display setting high quality / typical / optimized / default -- Display performance options for the rectangle.
  • locked boolean -- If true, the rectangle is locked.
  • name string -- The name of the rectangle.
  • nonprinting boolean -- If true, the rectangle does not print.
  • object reference (r/o) rectangle -- An object reference for the object.
  • overprint fill boolean -- If true, the rectangle's fill color overprints any underlying objects. If false, the fill color knocks out the underlying colors.
  • overprint stroke boolean -- If true, the rectangle's stroke color overprints any underlying objects. If false, the stroke color knocks out the underlying colors.
  • parent (r/o) reference -- The parent of the rectangle (a spread, master spread, spline item, polygon, graphic line, rectangle, oval, group, state, character, place gun or snippet). Can return: spread, master spread, spline item, polygon, graphic line, rectangle, oval, group, st...
  • rotation angle fixed -- The rotatation angle of the rectangle. (Range: -360 to 360)
  • shear angle fixed -- The skewing angle applied to the rectangle. (Range: -360 to 360)
  • story title string -- Title for this InCopy story.
  • stroke alignment center alignment / inside alignment / outside alignment -- The stroke alignment applied to the rectangle.
  • stroke color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the stroke of the rectangle. Can also accept: string.
  • stroke corner adjustment none / dashes / gaps / dashes and gaps -- The corner adjustment applied to the rectangle.
  • stroke dash and gap list of number | list of string -- The dash and gap measurements that define the pattern of a custom dashed line. Define up to six values (in points) in the format [dash1, gap1, dash2, gap2, dash3, gap3].
  • stroke weight number | string -- The weight (in points) to apply to the rectangle's stroke.
  • vertical scale fixed -- The vertical scaling applied to the rectangle.
  • visible boolean -- If true, the rectangle is visible.
  • visible bounds list of number | list of string -- The bounds of the rectangle including the stroke width, in the format [y1, x1, y2, x2], which give the coordinates of the top-left and bottom-right corners of the bounding box.
  • after place (r/o) string -- Dispatched after a rectangle is placed. This event bubbles. This event is not cancelable.
  • before place (r/o) string -- Dispatched before a rectangle is placed. This event bubbles. This event is cancelable.
