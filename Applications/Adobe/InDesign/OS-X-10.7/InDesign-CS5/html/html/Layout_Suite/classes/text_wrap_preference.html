Class text wrap preference

expand

Description

Text wrap preferences.

  • Defined in:
    • Layout Suite

Properties

  • apply to master page only boolean -- If true, text wraps on the master spread apply to that spread only, and not to any pages the master spread has been applied to.
  • class (r/o) type class -- The class descriptor type.
  • contour options (r/o) contour option -- The contour of the text wrap. Valid only when when text wrap type is contour.
  • inverse boolean -- If true, inverts the text wrap.
  • object reference (r/o) text wrap preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the text wrap preference (a page item, form field, multi state object, button, media item, sound, movie, EPSText, spline item, polygon, graphic line, rectangle, oval, graphic, imported page, PICT, WMF, PDF, EPS, image, group, text frame, ...
  • properties record -- A property that allows setting of several properties at the same time.
  • text wrap mode none / jump object text wrap / next column text wrap / bounding box text wrap / contour -- The text wrap mode.
  • text wrap offset anything -- The minimum space between text and the edges of the wrapped object. The format for defining text wrap offset values depends on the text wrap type. If text wrap type is jump object text wrap, specify 2 values in the format [top, bottom]. If text wrap ty...
  • text wrap side both sides / left side / right side / side towards spine / side away from spine / largest area -- Text wrap side options.
  • user modified wrap (r/o) boolean -- If true, the text wrap path has been explicitly modified by the user.
