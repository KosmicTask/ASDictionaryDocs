Class graphic

expand

Description

An imported graphic in any graphic file format (including vector, metafile, and bitmap formats).

  • Plural name:
    • graphics
  • Defined in:
    • Layout Suite
  • Inherits from:
    • page item (Layout Suite)

Properties

  • absolute flip none / horizontal / vertical / horizontal and vertical / both -- Indicates whether the graphic has been flipped independently of its parent object and, if yes, the direction in which the graphic was flipped.
  • absolute horizontal scale fixed -- The horizontal scale of the graphic relative to its containing object.
  • absolute rotation angle fixed -- The rotation angle of the graphic relative to its containing object. (Range: -360 to 360)
  • absolute shear angle fixed -- The skewing angle of the graphic relative to its containing object. (Range: -360 to 360)
  • absolute vertical scale fixed -- The vertical scale of the graphic relative to its containing object.
  • all graphics (r/o) list of graphic -- Lists all graphics contained by the graphic.
  • all page items (r/o) list of page item -- Lists all page items contained by the graphic.
  • applied object style object style -- The object style applied to the graphic.
  • associated XML element (r/o) XML item -- The XML element associated with the graphic.
  • content transparency settings (r/o) content transparency setting -- Transparency settings for the content of the graphic.
  • end join miter end join / round end join / bevel end join -- The corner join applied to the graphic.
  • fill color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the fill of the graphic. . Can also accept: string.
  • fill tint fixed -- The percent of tint to use in the graphic's fill color. (To specify a tint percent, use a number in the range of 0 to 100; to use the inherited or overridden value, use -1.)
  • fill transparency settings (r/o) fill transparency setting -- Transparency settings for the fill applied to the graphic.
  • geometric bounds list of number | list of string -- The bounds of the graphic excluding the stroke width, in the format [y1, x1, y2, x2], which give the coordinates of the top-left and bottom-right corners of the bounding box.
  • gradient fill angle fixed -- The angle of a linear gradient applied to the fill of the graphic. (Range: -180 to 180)
  • gradient fill length number | string -- The length (for a linear gradient) or radius (for a radial gradient) applied to the fill of the graphic.
  • gradient fill start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the fill of the graphic, in the format [x, y].
  • gradient stroke angle fixed -- The angle of a linear gradient applied to the stroke of the graphic. (Range: -180 to 180)
  • gradient stroke length number | string -- The length (for a linear gradient) or radius (for a radial gradient) applied to the stroke of the graphic.
  • gradient stroke start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the stroke of the graphic, in the format [x, y].
  • horizontal scale fixed -- The horizontal scaling applied to the graphic.
  • id (r/o) integer -- The unique ID of the graphic.
  • image type name (r/o) string -- The type of the image.
  • index (r/o) integer -- The index of the graphic within its containing object.
  • item layer layer -- The layer that the graphic is on.
  • item link (r/o) link -- The source file of the link.
  • local display setting high quality / typical / optimized / default -- Display performance options for the graphic.
  • locked boolean -- If true, the graphic is locked.
  • name string -- The name of the graphic.
  • nonprinting boolean -- If true, the graphic does not print.
  • object reference (r/o) graphic -- An object reference for the object.
  • overprint fill boolean -- If true, the graphic's fill color overprints any underlying objects. If false, the fill color knocks out the underlying colors.
  • overprint stroke boolean -- If true, the graphic's stroke color overprints any underlying objects. If false, the stroke color knocks out the underlying colors.
  • parent (r/o) reference -- The parent of the graphic (a XML element, sound, place gun, movie, polygon, graphic line, rectangle, oval, spline item, state or snippet). Can return: XML element, sound, place gun, movie, polygon, graphic line, rectangle, oval, spline item, state or s...
  • rotation angle fixed -- The rotatation angle of the graphic. (Range: -360 to 360)
  • shear angle fixed -- The skewing angle applied to the graphic. (Range: -360 to 360)
  • stroke alignment center alignment / inside alignment / outside alignment -- The stroke alignment applied to the graphic.
  • stroke color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the stroke of the graphic. Can also accept: string.
  • stroke corner adjustment none / dashes / gaps / dashes and gaps -- The corner adjustment applied to the graphic.
  • stroke dash and gap list of number | list of string -- The dash and gap measurements that define the pattern of a custom dashed line. Define up to six values (in points) in the format [dash1, gap1, dash2, gap2, dash3, gap3].
  • stroke weight number | string -- The weight (in points) to apply to the graphic's stroke.
  • vertical scale fixed -- The vertical scaling applied to the graphic.
  • visible boolean -- If true, the graphic is visible.
  • visible bounds list of number | list of string -- The bounds of the graphic including the stroke width, in the format [y1, x1, y2, x2], which give the coordinates of the top-left and bottom-right corners of the bounding box.
  • after place (r/o) string -- Dispatched after a graphic is placed. This event bubbles. This event is not cancelable.
  • before place (r/o) string -- Dispatched before a graphic is placed. This event bubbles. This event is cancelable.
