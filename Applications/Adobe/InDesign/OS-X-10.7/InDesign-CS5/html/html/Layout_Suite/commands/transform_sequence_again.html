Command transform sequence again

Description

Transforms the object using the last sequence of transform operations performed on any single object or performed at the same time on any group of objects. Transformations include moving, rotating, shearing, scaling, and flipping.

  • Defined in:
    • Layout Suite

Parameters

  • reference -- Supported by the following objects: page item, oval, rectangle, graphic line, polygon, group, image, EPS, PDF, WMF, PICT, graphic, spline item, EPSText, imported page, media item.

Result

  • list of string -- Playback errors
