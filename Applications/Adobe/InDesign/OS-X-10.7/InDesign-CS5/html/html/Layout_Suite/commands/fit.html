Command fit

Description

Applies the specified fit option to content in a frame.

  • Defined in:
    • Layout Suite

Parameters

  • reference -- Supported by the following objects: page item, oval, rectangle, graphic line, polygon, group, image, EPS, PDF, WMF, PICT, graphic, spline item, EPSText, imported page, media item.
  • given content to frame / center content / proportionally / frame to content / fill proportionally / apply frame fitting options -- The fit option to use.
