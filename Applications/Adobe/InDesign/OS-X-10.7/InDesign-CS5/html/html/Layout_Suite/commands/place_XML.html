Command place XML

Description

Places the XML element onto a page. If the place point is above an existing page item, place the XML element into the page item.

  • Defined in:
    • Layout Suite

Parameters

  • reference -- Supported by the following objects: spread, page, page item, oval, rectangle, graphic line, polygon, group, image, EPS, PDF, WMF, PICT, graphic, spline item, EPSText, imported page, media item.
  • using XML element -- The XML element to place.
  • place point number | string -- The point at which to place the object, specified in the format [x, y].
  • [autoflowing boolean] -- If true, autoflows placed text. Can accept: boolean (Default: FALSE).

Result

  • page item -- The new page item
