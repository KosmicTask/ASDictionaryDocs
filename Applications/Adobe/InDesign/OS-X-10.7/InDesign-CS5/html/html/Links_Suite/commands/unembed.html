Command unembed

Description

Unembeds the source file. If no folder is specified, creates a link to the original source file; if a folder is specified, copies the file to the folder and creates a link to the copied file.

  • Defined in:
    • Links Suite

Parameters

  • link
  • [to alias | string] -- The folder to which to copy the unembedded file.
  • [version comments string] -- The comment for this version.
  • [force save boolean] -- If true, forcibly saves a version. Can accept: boolean (Default: FALSE).
