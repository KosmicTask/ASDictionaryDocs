Class link

expand

Description

A link to a placed file.

  • Plural name:
    • links
  • Defined in:
    • Links Suite

Properties

  • asset ID (r/o) string -- The asset ID of the linked object.
  • asset URL (r/o) string -- The asset URL of the linked object.
  • class (r/o) type class -- The class descriptor type.
  • date (r/o) date -- The date and time the link was created.
  • edited (r/o) boolean -- If true, indicates the linked object has been edited in the current document but the source file has not been updated.
  • editing state (r/o) editing unknown / editing nowhere / editing remotely / editing locally / editing locally locked / editing conflict -- The Version Cue editing state of the file.
  • file path (r/o) string -- The file path (colon delimited on the Mac OS). Can also accept: alias or string.
  • id (r/o) integer -- The unique ID of the link.
  • index (r/o) integer -- The index of the link in the links collection.
  • label string -- A property that can be set to any string.
  • link type (r/o) string -- The file type of the linked object.
  • link xmp (r/o) link metadata -- XMP data for the link source file.
  • name (r/o) string -- The name of the link.
  • needed (r/o) boolean -- If true, indicates a link to a full-resolution version of the source file is needed. If false, indicates the object is embedded.
  • object reference (r/o) link -- An object reference for the object.
  • parent (r/o) reference -- The linked object. Can return: story, graphic, movie or sound.
  • properties record -- A property that allows setting of several properties at the same time.
  • size (r/o) integer -- The size of the link file.
  • status (r/o) normal / link out of date / link missing / link embedded / link inaccessible -- The status of the link.
  • version state (r/o) version unknown / project file newer / local project match / local newer / version conflict / no resource -- The Version Cue version state of the file.
  • after attribute changed (r/o) string -- Dispatched when the value of a property changes on this link. This event bubbles. This event is not cancelable.
  • after delete (r/o) string -- Dispatched after a link is deleted. This event bubbles. This event is not cancelable.
  • after embed (r/o) string -- Dispatched after a link is embedded. This event bubbles. This event is not cancelable.
  • after move (r/o) string -- Dispatched after a link is relocated from one object to another. This event bubbles. This event is not cancelable.
  • after new (r/o) string -- Dispatched after a link is created. This event bubbles. This event is not cancelable.
  • after unembed (r/o) string -- Dispatched after a link is unembedded. This event bubbles. This event is not cancelable.
  • after update (r/o) string -- Dispatched after a link is updated. This event bubbles. This event is not cancelable.
  • before delete (r/o) string -- Dispatched before a link is deleted. This event bubbles. This event is cancelable.
  • before embed (r/o) string -- Dispatched before a link is embedded. This event bubbles. This event is cancelable.
  • before move (r/o) string -- Dispatched before a link is relocated from one object to another. This event bubbles. This event is cancelable.
  • before unembed (r/o) string -- Dispatched before a link is unembedded. This event bubbles. This event is cancelable.
  • before update (r/o) string -- Dispatched before a link is updated. This event bubbles. This event is cancelable.
