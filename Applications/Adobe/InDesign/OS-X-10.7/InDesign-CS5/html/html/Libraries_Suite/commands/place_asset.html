Command place asset

Description

Places the asset in the specified document or text.

  • Defined in:
    • Libraries Suite

Parameters

  • asset
  • on reference -- The document or text object in which to place the asset. Can accept: document or text.

Result

  • list of reference -- The placed item. Can return: list of page items, movies, sounds, images, EPSs, WMFs, PICTs or PDFs.
