Command remove event listener

Description

Removes the event listener.

  • Defined in:
    • Object Styles Suite

Parameters

  • object style
  • event type string -- The registered event type.
  • handler anything -- The registered event handler. Can accept: alias or string or JavaScript Function.
  • [captures boolean] -- This parameter is obsolete. Can accept: boolean (Default: FALSE).

Result

  • boolean -- Returns true if the handler was previously registered for this event.
