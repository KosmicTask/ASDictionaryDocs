Description

Properties

  • class (r/o) type class -- The class descriptor type.
  • description string -- The description of the preflight profile.
  • id (r/o) integer -- The unique ID of the preflight profile.
  • index (r/o) integer -- The index of the preflight profile within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the preflight profile.
  • object reference (r/o) preflight profile -- An object reference for the object.
  • parent (r/o) reference -- The parent of the preflight profile (a application or document). Can return: application or document.
  • properties record -- A property that allows setting of several properties at the same time.
