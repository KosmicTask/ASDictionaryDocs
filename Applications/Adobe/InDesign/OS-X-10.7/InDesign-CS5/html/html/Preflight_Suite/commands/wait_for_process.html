Command wait for process

Description

Waits for the process to finish, up to an optional maximum amount of time. No other processes get cycles during this time.

  • Defined in:
    • Preflight Suite

Parameters

  • preflight process
  • [wait time fixed] -- The maximum time to wait, in seconds; if omitted, waits until completion no matter how long it takes.

Result

  • boolean -- True if the process is finished; false if not. False is only possible when a max time limit is specified.
