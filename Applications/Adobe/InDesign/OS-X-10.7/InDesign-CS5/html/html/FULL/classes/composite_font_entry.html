Class composite font entry

reduce

Description

A composite font entry.

  • Plural name:
    • composite font entries
  • Defined in:
    • Text Suite

Properties

  • applied font anything -- The font applied to the composite font entry, specified as either a font object or the name of font family. Can return: font or string.
  • baseline shift fixed -- The amount of baseline shift.
  • class (r/o) type class -- The class descriptor type.
  • custom characters string -- The characters that the set affects.
  • font style string -- The name of the font style.
  • horizontal scale fixed -- The horizontal scaling applied to the composite font entry.
  • id (r/o) integer -- The unique ID of the composite font entry.
  • index (r/o) integer -- The index of the composite font entry within its containing object.
  • label string -- A property that can be set to any string.
  • locked (r/o) boolean -- If true, the composite font entry is locked.
  • name string -- The name of the composite font entry.
  • object reference (r/o) composite font entry -- An object reference for the object.
  • parent (r/o) composite font -- The parent of the composite font entry (a composite font).
  • properties record -- A property that allows setting of several properties at the same time.
  • relative size fixed -- The size of the entry relative to the base entry. Note: The base entry cannot be modified.
  • scale option boolean -- If true, scales characters from the center.
  • vertical scale fixed -- The vertical scaling applied to the composite font entry.

Elements

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id
