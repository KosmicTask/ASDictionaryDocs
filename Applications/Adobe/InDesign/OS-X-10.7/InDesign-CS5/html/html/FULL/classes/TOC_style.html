Class TOC style

reduce

Description

A TOC style definition.

  • Plural name:
    • TOC styles
  • Defined in:
    • Table Of Contents Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • create bookmarks boolean -- If true, creates bookmarks for TOC entries.
  • id (r/o) integer -- The unique ID of the TOC style.
  • include book documents boolean -- If true, includes the entire book in the TOC. If false, includes only the TOC entries in the current document. Note: Valid when the current document is part of a book.
  • include hidden boolean -- If true, the TOC includes entries from text on hidden layers.
  • index (r/o) integer -- The index of the TOC style within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the TOC style.
  • numbered paragraphs include full paragraph / include numbers only / exclude numbers -- The format for importing numbered paragraphs into the TOC.
  • object reference (r/o) TOC style -- An object reference for the object.
  • parent (r/o) document -- The parent of the TOC style (a document).
  • properties record -- A property that allows setting of several properties at the same time.
  • run in boolean -- If true, the lowest-level TOC entries are placed on the same line as the previous entry.
  • set story direction horizontal / vertical -- The table of contents story direction.
  • title string -- The TOC title.
  • title style paragraph style -- The paragraph style applied to the TOC title.

Elements

contains events , event listeners , TOC style entries

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id
  • TOC style entry -- by index, name, range, relative, test

Notes
