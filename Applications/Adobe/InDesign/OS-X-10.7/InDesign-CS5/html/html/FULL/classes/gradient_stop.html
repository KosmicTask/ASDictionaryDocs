Class gradient stop

reduce

Description

A gradient stop in a gradient.

  • Plural name:
    • gradient stops
  • Defined in:
    • Color Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • index (r/o) integer -- The index of the gradient stop within its containing object.
  • location fixed -- The starting location (as a percentage of the gradient length) of the gradient stop on the gradient. (Range: 0 to 100). Can return: fixed (0 - 100).
  • midpoint fixed -- The mid-point (as a percentage of the gradient length) of the gradient stop. (Range: 13 to 87). Can return: fixed (13 - 87).
  • object reference (r/o) gradient stop -- An object reference for the object.
  • parent (r/o) gradient -- The parent of the gradient stop (a gradient).
  • properties record -- A property that allows setting of several properties at the same time.
  • stop color reference -- The swatch (color, gradient, tint, or mixed ink) applied to the gradient stop. Can return: mixed ink or color.
