Class PDF attribute

reduce

Description

PDF attributes.

  • Defined in:
    • Layout Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • object reference (r/o) PDF attribute -- An object reference for the object.
  • page number (r/o) integer -- The page number of the PDF document page to place.
  • parent (r/o) reference -- The parent of the PDF attribute (a PDF or imported page). Can return: PDF or imported page.
  • PDF crop (r/o) crop art / crop PDF / crop trim / crop bleed / crop media / crop content visible layers / crop content all layers -- The type of cropping to apply.
  • properties record -- A property that allows setting of several properties at the same time.
  • transparent background (r/o) boolean -- If true, the background of the PDF is transparent.

  • event listener -- by index, name, range, relative, test, id
