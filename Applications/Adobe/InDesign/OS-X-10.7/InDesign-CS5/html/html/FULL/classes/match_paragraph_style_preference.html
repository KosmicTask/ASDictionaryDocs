Class match paragraph style preference

reduce

Description

The preferences for a running header/footer (match paragraph style) variable.

  • Defined in:
    • Preferences Suite

Properties

  • applied paragraph style paragraph style -- The paragraph style applied to the text. Can also accept: string.
  • change case none / uppercase / lowercase / titlecase / sentencecase -- The case of the matched text.
  • class (r/o) type class -- The class descriptor type.
  • delete end punctuation boolean -- If true, deletes end punctuation from the matched text.
  • object reference (r/o) match paragraph style preference -- An object reference for the object.
  • parent (r/o) text variable -- The parent of the match paragraph style preference (a text variable).
  • properties record -- A property that allows setting of several properties at the same time.
  • search strategy first on page / last on page -- The starting point and direction in which the search will be conducted.
  • text after string -- The text that follows the value of the variable. (Limit: 128 characters)
  • text before string -- The text that precedes the value of the variable. (Limit: 128 characters)

Elements

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id
