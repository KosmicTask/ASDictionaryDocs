Class spread

reduce

Description

A spread.

  • Plural name:
    • spreads
  • Defined in:
    • Layout Suite

Properties

  • all graphics (r/o) list of graphic -- Lists all graphics contained by the spread.
  • all page items (r/o) list of page item -- Lists all page items contained by the spread.
  • allow page shuffle boolean -- If true, guarantees that when pages are added to a spread it will contain a maximum of two pages. If false, allows pages to be added or moved into existing spreads. For override information, see preserve layout when shuffling.
  • applied master master spread -- The master spread applied to the spread. Can also accept: nothing.
  • class (r/o) type class -- The class descriptor type.
  • flattener override default / none / custom -- The transparency flattener preferences override for the spread.
  • flattener preferences (r/o) flattener preference -- Flattener preference settings.
  • id (r/o) integer -- The unique ID of the spread.
  • IDML component name string -- The IDML component name of the spread.
  • index (r/o) integer -- The index of the spread within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the spread; this is an alias to the spread's label property.
  • object reference (r/o) spread -- An object reference for the object.
  • page transition direction not applicable / down / right to left / left down / left up / left to right / right down / right up / up / in / out / horizontal / vertical / horizontal in / horizontal out / vertical in / vertical out -- The direction of the page transition.
  • page transition duration fast / medium / slow -- The duration of the page transition.
  • page transition type none / blinds transition / box transition / comb transition / cover transition / dissolve transition / fade transition / page turn transition / push transition / split transition / uncover transition / wipe transition / zoom in transition / zoom out transition -- The type of page transition.
  • parent (r/o) document -- The parent of the spread (a document).
  • properties record -- A property that allows setting of several properties at the same time.
  • show master items boolean -- If true, displays master page items on document pages in the spread.
  • timing settings (r/o) timing setting -- The object timing settings.
