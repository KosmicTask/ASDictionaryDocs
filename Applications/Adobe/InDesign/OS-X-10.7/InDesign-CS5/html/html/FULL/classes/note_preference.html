Class note preference

reduce

Description

Note preferences.

  • Defined in:
    • Preferences Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • find and replace note contents boolean -- If true, includes inline notes content when using Find/Change commands (in Galley and Story views only).
  • note background color galley background color / use note color -- The background color for notes.
  • note color anything -- The note color, specified either as an array of three doubles, each in the range 0 to 255 and representing R, G, and B values, or as an InCopy UI color. Can return: list of 3 fixed (0 - 255) or canary/lemon/electrolyte/lime/forest/lichen/jade/aqua/cirr...
  • note color choices use user color / use note pref color -- The color to use for notes.
  • object reference (r/o) note preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the note preference (a application).
  • properties record -- A property that allows setting of several properties at the same time.
  • show note tips boolean -- If true, displays note information and some note content when the mouse pointer hovers over a note anchor in layout view or a note bookend in galley or story view.
  • spell check notes boolean -- If true, includes inline notes content when using Spell Check (in Galley and Story views only).
