Class object style

reduce

Description

An object style.

  • Plural name:
    • object styles
  • Defined in:
    • Object Styles Suite

Properties

  • anchored object settings anchored object setting -- Anchored object settings.
  • applied paragraph style paragraph style -- The paragraph style applied to the text. Can also accept: string.
  • apply next paragraph style boolean -- If true, applies paragraph styles using Next Paragraph Style settings, beginning with the Next Paragraph Style defined in the paragraph style associated with the object style (if any).
  • based on anything -- The style that this style is based on. Can return: object style or string.
  • baseline frame grid options baseline frame grid option -- Baseline frame grid option settings.
  • bottom right corner option none / rounded corner / inverse rounded corner / inset corner / bevel corner / fancy corner -- The shape to apply to the bottom right corner of rectangular shapes.
  • bottom left corner option none / rounded corner / inverse rounded corner / inset corner / bevel corner / fancy corner -- The shape to apply to the bottom left corner of rectangular shapes.
  • bottom left corner radius number | string -- The radius in measurement units of the corner effect applied to the bottom left corner of rectangular shapes
  • bottom right corner radius number | string -- The radius in measurement units of the corner effect applied to the bottom right corner of rectangular shapes
  • class (r/o) type class -- The class descriptor type.
  • content effects enabling settings (r/o) object style content effects category settings -- The content effects enabling settings.
  • content transparency settings (r/o) content transparency setting -- Transparency settings for the content of the object style.
  • enable anchored object options boolean -- If true, the object style will apply an anchored object setting.
  • enable fill boolean -- If true, the object style will apply a fill.
  • enable frame fitting options boolean -- If true, the object style will apply frame fitting options.
  • enable paragraph style boolean -- If true, the object style will apply a paragraph style.
  • enable story options boolean -- If true, the object style will apply story options.
  • enable stroke boolean -- If true, the object style will apply a stroke.
  • enable stroke and corner options boolean -- If true, the object style will apply stroke options and corner options.
  • enable text frame baseline options boolean -- If true, the object style will apply baseline text frame options.
  • enable text frame general options boolean -- If true, the object style will apply general text frame options.
  • enable text wrap and others boolean -- If true, the object style will apply text wrap, contour, and non-printing settings.
  • end cap butt end cap / round end cap / projecting end cap -- The end shape of an open path.
  • end join miter end join / round end join / bevel end join -- The corner join applied to the object style.
  • fill color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the fill of the object style. . Can also accept: string.
  • fill effects enabling settings (r/o) object style fill effects category settings -- The fill effects enabling settings.
  • fill tint fixed -- The percent of tint to use in the object style's fill color. (To specify a tint percent, use a number in the range of 0 to 100; to use the inherited or overridden value, use -1.)
  • fill transparency settings (r/o) fill transparency setting -- Transparency settings for the fill applied to the object style.
  • frame fitting options frame fitting option -- The frame fitting option to apply to placed or pasted content. Can be applied to a frame, object style, or document or to the application.
  • gap color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the gap of a dashed, dotted, or striped stroke. For information, see stroke type.
  • gap tint fixed -- The tint as a percentage of the gap color. (To specify a tint percent, use a number in the range of 0 to 100; to use the inherited or overridden value, use -1.)
  • gradient fill angle fixed -- The angle of a linear gradient applied to the fill of the object style. (Range: -180 to 180)
  • gradient stroke angle fixed -- The angle of a linear gradient applied to the stroke of the object style. (Range: -180 to 180)
  • id (r/o) integer -- The unique ID of the object style.
  • index (r/o) integer -- The index of the object style within its containing object.
  • label string -- A property that can be set to any string.
  • left line end none / simple arrow head / simple wide arrow head / triangle arrow head / triangle wide arrow head / barbed arrow head / curved arrow head / circle arrow head / circle solid arrow head / square arrow head / square solid arrow head / bar arrow head -- The arrowhead applied to the start of the path.
  • miter limit fixed -- The limit of the ratio of stroke width to miter length before a miter (pointed) join becomes a bevel (squared-off) join. Can return: fixed (1 - 500).
  • name string -- The name of the object style.
  • nonprinting boolean -- If true, the object style does not print.
  • object effects enabling settings (r/o) object style object effects category settings -- The object effects enabling settings.
  • object reference (r/o) object style -- An object reference for the object.
  • overprint fill boolean -- If true, the object style's fill color overprints any underlying objects. If false, the fill color knocks out the underlying colors.
  • overprint gap boolean -- If true, the gap color overprints any underlying colors. If false, the gap color knocks out the underlying colors.
  • overprint stroke boolean -- If true, the object style's stroke color overprints any underlying objects. If false, the stroke color knocks out the underlying colors.
  • parent (r/o) reference -- The parent of the object style (a document, application or object style group). Can return: document, application or object style group.
  • properties record -- A property that allows setting of several properties at the same time.
  • right line end none / simple arrow head / simple wide arrow head / triangle arrow head / triangle wide arrow head / barbed arrow head / curved arrow head / circle arrow head / circle solid arrow head / square arrow head / square solid arrow head / bar arrow head -- The arrowhead applied to the end of the path.
  • story preferences story preference -- Story preference settings.
  • stroke alignment center alignment / inside alignment / outside alignment -- The stroke alignment applied to the object style.
  • stroke color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the stroke of the object style. Can also accept: string.
  • stroke effects enabling settings (r/o) object style stroke effects category settings -- The stroke effects enabling settings.
  • stroke tint fixed -- The percent of tint to use in object's stroke color. (To specify a tint percent, use a number in the range of 0 to 100; to use the inherited or overridden value, use -1.)
  • stroke transparency settings (r/o) stroke transparency setting -- Transparency settings for the stroke.
  • stroke type stroke style -- The name of the stroke style to apply. Can also accept: string.
  • stroke weight number | string -- The weight (in points) to apply to the object style's stroke.
  • text frame preferences text frame preference -- Text frame preference settings.
  • text wrap preferences text wrap preference -- The text wrap preference properties that define the default formatting for wrapping text around objects.
  • top left corner option none / rounded corner / inverse rounded corner / inset corner / bevel corner / fancy corner -- The shape to be applied to the top left corner of rectangular shapes and all corners of non-rectangular shapes.Note: corner option differs from end join in which you can set a radius for a corner option, whereas the rounded or beveled effect of an end ...
  • top left corner radius number | string -- The radius in measurement units of the corner effect applied to the top left corner of rectangular shapes and all corners of non-rectangular shapes
  • top right corner option none / rounded corner / inverse rounded corner / inset corner / bevel corner / fancy corner -- The shape to apply to the top right corner of rectangular shapes
  • top right corner radius number | string -- The radius in measurement units of the corner effect applied to the top right corner of rectangular shapes
  • transparency settings (r/o) transparency setting -- Transparency settings.
