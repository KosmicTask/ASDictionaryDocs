Class frame fitting option

reduce

Description

Options for fitting placed or pasted content in a frame.

  • Defined in:
    • Layout Suite

Properties

  • auto fit boolean -- If true, the last saved fitting options will be applied to the contents of a frame when it is resized.
  • bottom crop number | string -- The amount in measurement units to crop the bottom edge of a graphic.
  • class (r/o) type class -- The class descriptor type.
  • fitting alignment top left anchor / top center anchor / top right anchor / left center anchor / center anchor / right center anchor / bottom left anchor / bottom center anchor / bottom right anchor -- The point with which to align the image empty when fitting in a frame. For information, see frame fitting options.
  • fitting on empty frame none / content to frame / proportionally / fill proportionally -- The frame fitting option to apply to placed or pasted content if the frame is empty. Can be applied to a frame, object style, or document or to the application.
  • left crop number | string -- The amount in measurement units to crop the left edge of a graphic.
  • object reference (r/o) frame fitting option -- An object reference for the object.
  • parent (r/o) reference -- The parent of the frame fitting option (a object style, oval, rectangle, polygon, application or document). Can return: object style, oval, rectangle, polygon, application or document.
  • properties record -- A property that allows setting of several properties at the same time.
  • right crop number | string -- The amount in measurement units to crop the right edge of a graphic.
  • top crop number | string -- The amount in measurement units to crop the top edge of a graphic.

  • event listener -- by index, name, range, relative, test, id
