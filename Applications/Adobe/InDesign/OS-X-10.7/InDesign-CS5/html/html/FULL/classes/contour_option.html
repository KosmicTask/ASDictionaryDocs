Class contour option

reduce

Description

A contour option.

  • Defined in:
    • Layout Suite

Properties

  • alpha channel path names (r/o) list of string -- A list of the alpha channels stored in the graphic.
  • class (r/o) type class -- The class descriptor type.
  • contour path name string -- The alpha channel or Photoshop path to use for the contour option. Valid only when the contour options is photoshop path or alpha channel.
  • contour type bounding box / photoshop path / detect edges / alpha channel / graphic frame / same as clipping -- The contour type.
  • include inside edges boolean -- If true, creates interior clipping paths within the surrounding clipping path. Note: Valid only when clipping type is alpha channel or detect edges.
  • object reference (r/o) contour option -- An object reference for the object.
  • parent (r/o) text wrap preference -- The parent of the contour option (a text wrap preference).
  • photoshop path names (r/o) list of string -- A list of the clipping paths stored in the graphic.
  • properties record -- A property that allows setting of several properties at the same time.
