Class named grid

reduce

Description

A named grid.

  • Plural name:
    • named grids
  • Defined in:
    • Text Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • grid data (r/o) grid data information -- Default grid properties. Note: Applies to named, layout, and frame (story) grids.
  • id (r/o) integer -- The unique ID of the named grid.
  • index (r/o) integer -- The index of the named grid within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the named grid.
  • object reference (r/o) named grid -- An object reference for the object.
  • parent (r/o) reference -- The parent of the named grid (a document or application). Can return: document or application.
  • properties record -- A property that allows setting of several properties at the same time.
