Class ink

reduce

Description

An ink.

  • Plural name:
    • inks
  • Defined in:
    • Color Suite

Properties

  • alias ink name string -- The ink object to map this ink to.
  • angle fixed -- The angle of the ink. (Range: 0 to 360). Can return: fixed (0 - 360).
  • class (r/o) type class -- The class descriptor type.
  • convert to process boolean -- Converts spot inks to process inks.
  • frequency fixed -- The frequency of the ink. (Range: 1 to 500). Can return: fixed (1 - 500).
  • id (r/o) integer -- The unique ID of the ink.
  • index (r/o) integer -- The index of the ink within its containing object.
  • ink type normal / opaque / transparent / opaque ignore -- The trapping type of the ink.
  • is process ink (r/o) boolean -- If true, the ink is a process ink.
  • label string -- A property that can be set to any string.
  • name (r/o) string -- The name of the ink.
  • neutral density fixed -- The neutral density of the ink. (Range: 0.001 to 10.0). Can return: fixed (0.001 - 10).
  • object reference (r/o) ink -- An object reference for the object.
  • parent (r/o) reference -- The parent of the ink (a application or document). Can return: application or document.
  • print ink boolean -- If true, prints the ink. Only valid when printing separations.
  • properties record -- A property that allows setting of several properties at the same time.
  • solidity (r/o) fixed -- The solidity value of the ink. (Range: 0.0 to 1.0). Can return: fixed (0 - 1).
  • trap order integer -- The place of the ink in the trapping sequence.
