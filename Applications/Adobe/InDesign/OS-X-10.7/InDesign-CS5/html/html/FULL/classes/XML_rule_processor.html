Class XML rule processor

reduce

Description

An XML rule processor.

  • Plural name:
    • XML rule processors
  • Defined in:
    • XML Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • halted (r/o) boolean -- If true, the XML rule process has halted.
  • id (r/o) integer -- The unique ID of the XML rule processor.
  • index (r/o) integer -- The index of the XML rule processor within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the XML rule processor; this is an alias to the XML rule processor's label property.
  • object reference (r/o) XML rule processor -- An object reference for the object.
  • parent (r/o) reference -- The parent of the XML rule processor (a application).
  • properties record -- A property that allows setting of several properties at the same time.
  • rule paths (r/o) list of string -- The XPath condition paths of the rules in the rule set.

Elements

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id
