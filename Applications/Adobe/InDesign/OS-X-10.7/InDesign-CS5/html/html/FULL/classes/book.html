Properties

  • automatic document conversion boolean -- If true, automatically converts the book content object during repagination and synchronization.
  • automatic pagination boolean -- If true, automatically updates page numbers when pages in book content files are added, deleted, or rearranged.
  • class (r/o) type class -- The class descriptor type.
  • file path (r/o) alias | string -- The full path to the file.
  • full name (r/o) alias | string -- The full path to the book, including the name of the book.
  • index (r/o) integer -- The index of the book within its containing object.
  • insert blank page boolean -- If true, inserts a blank page as necessary to fill in page number gaps caused by the odd or even specification set in the repagination option.
  • label string -- A property that can be set to any string.
  • merge identical layers boolean -- If true, merges identically named layers when exporting to PDF.
  • modified (r/o) boolean -- If true, the book has been modified since it was last saved.
  • name (r/o) string -- The name of the book.
  • object reference (r/o) book -- An object reference for the object.
  • parent (r/o) reference -- The parent of the book (a application).
  • preflight book options (r/o) preflight book option -- Preflight book option settings.
  • print preferences (r/o) print preference -- Print preference settings.
  • properties record -- A property that allows setting of several properties at the same time.
  • repagination option next page / next odd page / next even page -- Page numbering options for book content objects within the book.
  • saved (r/o) boolean -- If true, the book has been saved since it was created.
  • smart match style groups match style path / match style name -- How to match styles with same name while synchronizing a book.
  • style source document reference -- Current style source document. Can return: document or book content.
  • synchronize bullet numbering list boolean -- If true, synchronize bullets and numbering.
  • synchronize cell style boolean -- If true, synchronize cell styles.
  • synchronize character style boolean -- If true, synchronize character styles.
  • synchronize conditional text boolean -- If true, synchronize conditional text.
  • synchronize master page boolean -- If true, synchronize master pages.
  • synchronize object style boolean -- If true, synchronize object styles.
  • synchronize paragraph style boolean -- If true, synchronize paragraph styles.
  • synchronize swatch boolean -- If true, synchronize swatches.
  • synchronize table of content style boolean -- If true, synchronize table of content styles.
  • synchronize table style boolean -- If true, synchronize table styles.
  • synchronize text variable boolean -- If true, synchronize text variables.
  • synchronize trap style boolean -- If true, synchronize trap styles.
