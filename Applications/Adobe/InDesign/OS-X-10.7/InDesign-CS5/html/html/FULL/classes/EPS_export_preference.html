Class EPS export preference

reduce

Description

EPS export preferences.

  • Defined in:
    • Preferences Suite

Properties

  • applied flattener preset flattener preset -- The transparency flattener preset to use.
  • bleed bottom number | string -- The height of the bleed area at the bottom of the page. Note: Valid only when use document bleed to print is true. Can return: number or string (0 - 432 points).
  • bleed inside number | string -- The width of the bleed area at the inside of the page. Note: Valid only when use document bleed to print is true. Can return: number or string (0 - 432 points).
  • bleed outside number | string -- The width of the bleed area at the outside of the page. Note: Valid only when use document bleed to print is true. . Can return: number or string (0 - 432 points).
  • bleed top number | string -- The height of the bleed area at the top of the page. Note: Valid only when use document bleed to print is true. Can return: number or string (0 - 432 points).
  • class (r/o) type class -- The class descriptor type.
  • data format ASCII / binary -- The format in which to send image data to the printer.
  • EPS color RGB / CMYK / gray / unchanged color space / postscript color management -- The color space for representing color in the exported EPS.
  • EPS spreads boolean -- If true, exports facing pages as a single page that has the width of the spread. If false, exports spread pages as separate pages.
  • font embedding none / complete / subset -- Controls how fonts are embedded in the exported EPS.
  • ignore spread overrides boolean -- If true, ignores flattener spread overrides.
  • image data all image data / proxy image data -- The image data to export to the EPS document.
  • object reference (r/o) EPS export preference -- An object reference for the object.
  • omit bitmaps boolean -- If true, replaces bitmap images with OPI links.
  • omit EPS boolean -- If true, replaces EPS images with OPI links.
  • omit PDF boolean -- If true, replaces PDF images with OPI links.
  • OPI image replacement boolean -- If true, prints graphics that are either OPI comments stored in imported EPS files or linked using OPI comments. For information on linking files using OPI comments, see omit EPS, omit PDF, or omit bitmaps.
  • page range anything -- The pages to print, specified either as an enumeration or a string. To specify a range, separate page numbers in the string with a hyphen (-). To specify separate pages, separate page numbers in the string with a comma (,). Can return: all pages/select...
  • parent (r/o) reference -- The parent of the EPS export preference (a application).
  • PostScript level level 2 / level 3 -- The PostScript level of the printer.
  • preview none / TIFF preview / PICT preview -- The file format of the preview image saved with the exported EPS file.
  • properties record -- A property that allows setting of several properties at the same time.

Elements

contains events , event listeners

  • event listener -- by index, name, range, relative, test, id
