Class story window

Description

A story window.

  • Plural name:
    • story windows
  • Defined in:
    • Basics Suite
  • Inherits from:
    • window (Basics Suite)

Properties

  • class (r/o) type class -- The class descriptor type.
  • properties record -- A property that allows setting of several properties at the same time.
  • after selection attribute changed (r/o) string -- Dispatched after an attribute on the active selection changes. This event bubbles. This event is not cancelable.
  • after selection changed (r/o) string -- Dispatched after the active selection changes. This event bubbles. This event is not cancelable.
  • bounds list of integer -- The bounds of the window (specified in pixels) in the form [top, left, bottom, right].
  • index (r/o) integer -- The index of the story window within its containing object.
  • name (r/o) string -- The name of the story window.
  • object reference (r/o) story window -- An object reference for the object.
  • parent (r/o) document -- The parent of the story window (a document).
  • selection list of reference -- The selected object(s). Can also accept: object or nothing.
  • after activate (r/o) string -- Dispatched after the story window becomes active. This event bubbles. This event is not cancelable.
  • after attribute changed (r/o) string -- Dispatched when the value of a property changes on this story window. This event bubbles. This event is not cancelable.
  • after close (r/o) string -- Dispatched when a story window is closing. Since the close has been committed, it can no longer be canceled. This event bubbles. This event is not cancelable.
  • after open (r/o) string -- Dispatched after a story window is opened. This event bubbles. This event is not cancelable.
  • before close (r/o) string -- Dispatched before a story window is closed. This event bubbles. This event is cancelable.
  • before deactivate (r/o) string -- Dispatched before the story window becomes inactive. This event bubbles. This event is not cancelable.
