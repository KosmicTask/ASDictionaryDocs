Properties

  • class (r/o) type class -- The class descriptor type.
  • file path (r/o) string -- The file path (colon delimited on the Mac OS). Can also accept: alias or string.
  • id (r/o) integer -- The unique ID of the assigned story.
  • index (r/o) integer -- The index of the assigned story within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the assigned story.
  • object reference (r/o) assigned story -- An object reference for the object.
  • parent (r/o) assignment -- The parent of the assigned story (a assignment).
  • properties record -- A property that allows setting of several properties at the same time.
  • story reference (r/o) reference -- A reference to the assigned story. Can return: story, page item, oval, rectangle or polygon.
