Class book content

reduce

Description

A document added to a book.

  • Plural name:
    • book contents
  • Defined in:
    • Book Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • date (r/o) date -- The date and time the book content was created.
  • document page range (r/o) string -- The page range of the book content object within the book.
  • file path (r/o) alias | string -- The full path to the file.
  • full name (r/o) alias | string -- The full path to the book content, including the name of the book content.
  • id (r/o) integer -- The unique ID of the book content.
  • index (r/o) integer -- The index of the book content within its containing object.
  • label string -- A property that can be set to any string.
  • name (r/o) string -- The name of the book content.
  • object reference (r/o) book content -- An object reference for the object.
  • parent (r/o) book -- The parent of the book content (a book).
  • properties record -- A property that allows setting of several properties at the same time.
  • size (r/o) integer -- The size of the book content file.
  • status (r/o) normal / missing document / document out of date / document in use / document is open -- The status of the book content object file.
