Class XML export map

reduce

Description

A mapping object that maps a style (paragraph, character, table, or cell) to an XML tag.

  • Plural name:
    • XML export maps
  • Defined in:
    • XML Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • include empty stories boolean -- If true, includes empty stories when mapping styles to tags.
  • include master page stories boolean -- If true, includes stories on master spreads when mapping styles to tags.
  • include pasteboard stories boolean -- If true, includes stories on the pasteboard when mapping styles to tags.
  • index (r/o) integer -- The index of the XML export map within its containing object.
  • mapped style (r/o) reference -- The style mapped to the XML tag. Can return: paragraph style, character style, table style or cell style. Can also accept: string.
  • markup tag XML tag -- The XML tag applied to the element. Can also accept: string.
  • object reference (r/o) XML export map -- An object reference for the object.
  • parent (r/o) reference -- The parent of the XML export map (a application or document). Can return: application or document.
  • properties record -- A property that allows setting of several properties at the same time.

Elements

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id
