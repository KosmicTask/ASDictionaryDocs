Class font

reduce

Description

A font.

  • Plural name:
    • fonts
  • Defined in:
    • Text Suite

Properties

  • allow editable embedding (r/o) boolean -- If true, the font can be embedded.
  • allow outlines (r/o) boolean -- If true, the font can be converted to outlines.
  • allow PDF embedding (r/o) boolean -- If true, the font can be embedded in a PDF document.
  • allow printing (r/o) boolean -- If true, the can font be printed.
  • class (r/o) type class -- The class descriptor type.
  • font family (r/o) string -- The name of the font family
  • font style name (r/o) string -- The name of the font style.
  • font style name native (r/o) string -- The native name of the font style.
  • font type (r/o) type 1 / TrueType / CID / ATC / bitmap / OCF / OpenType CFF / OpenType CID / OpenType TT / unknown -- The type of font.
  • full name (r/o) string -- The full font name.
  • full name native (r/o) string -- The full native name of the font.
  • index (r/o) integer -- The index of the font within its containing object.
  • location (r/o) string -- The full path to the font.
  • name (r/o) string -- The name of the font.
  • object reference (r/o) font -- An object reference for the object.
  • ordering (r/o) string -- The ordering of a CID font.
  • parent (r/o) reference -- The parent of the font (a document or application). Can return: document or application.
  • platform name (r/o) string -- The platform font name.
  • PostScript name (r/o) string -- The PostScript name of the font.
  • properties record -- A property that allows setting of several properties at the same time.
  • registry (r/o) string -- The registry of a CID font.
  • restricted printing (r/o) boolean -- If true, the font allows only restricted printing.
  • status (r/o) installed / not available / fauxed / substituted / unknown -- The status of the font.
  • version (r/o) string -- The font version.
  • writing script (r/o) integer -- The writing script.
