Class image IO preference

reduce

Description

Image I/O preferences.

  • Defined in:
    • Preferences Suite

Properties

  • allow auto embedding boolean -- If true, allows auto embedding.
  • alpha channel name string -- The name of the alpha channel.
  • apply Photoshop clipping path boolean -- If true, applies clipping paths defined in Photoshop to placed images.
  • class (r/o) type class -- The class descriptor type.
  • object reference (r/o) image IO preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the image IO preference (a application or image). Can return: application or image.
  • preview resolution (r/o) integer -- The image resolution in ppi, set when the graphic is imported.
  • properties record -- A property that allows setting of several properties at the same time.

Elements

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id

Notes

  • Contained by:
    • application (Basics Suite)
    • image (Layout Suite)
    • image IO preference (Preferences Suite)
