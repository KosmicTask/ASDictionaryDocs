Class movie behavior

reduce

Description

A movie behavior object.

  • Plural name:
    • movie behaviors
  • Defined in:
    • Interactive Elements Suite
  • Inherits from:
    • behavior (Interactive Elements Suite)

Properties

  • behavior event mouse up / mouse down / mouse enter / mouse exit / on focus / on blur -- The event which triggers the behavior.
  • class (r/o) type class -- The class descriptor type.
  • enable behavior boolean -- If true, the behavior is enabled.
  • label string -- A property that can be set to any string.
  • properties record -- A property that allows setting of several properties at the same time.
  • id (r/o) integer -- The unique ID of the movie behavior.
  • index (r/o) integer -- The index of the movie behavior within its containing object.
  • movie item movie -- The movie page item.
  • name (r/o) string -- The name of the movie behavior.
  • navigation point ID integer -- The id of the navigation point to play from. This property is ignored for all operations other than Play From Navigation Point.
  • object reference (r/o) movie behavior -- An object reference for the object.
  • operation play / play from navigation point / stop / pause / resume / stop all -- The playback mode.
  • parent (r/o) button -- The parent of the movie behavior (a button).

Elements

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id

Notes
