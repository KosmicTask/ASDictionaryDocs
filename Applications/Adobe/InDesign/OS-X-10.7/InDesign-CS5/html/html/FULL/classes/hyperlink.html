  • highlight none / invert / outline / inset -- The hyperlink highlight style.

  • index (r/o) integer -- The index of the hyperlink within its containing object.

  • object reference (r/o) hyperlink -- An object reference for the object.
  • parent (r/o) document -- The parent of the hyperlink (a document).
  • properties record -- A property that allows setting of several properties at the same time.
  • source reference -- The hyperlinked text or page item. Can return: hyperlink page item source, hyperlink text source or cross reference source.
  • visible boolean -- If true, the hyperlink is visible.
  • width thin / medium / thick -- The stroke weight of the hyperlink border.
