Class display setting

reduce

Description

Object-level display settings.

  • Plural name:
    • display settings
  • Defined in:
    • Preferences Suite

Properties

  • antialiasing boolean -- If true, uses anti-aliasing for text and bitmap images.
  • class (r/o) type class -- The class descriptor type.
  • greek below fixed -- The point size below which text is greeked.
  • index (r/o) integer -- The index of the display setting within its containing object.
  • object reference (r/o) display setting -- An object reference for the object.
  • parent (r/o) reference -- The parent of the display setting (a application).
  • properties record -- A property that allows setting of several properties at the same time.
  • raster gray out / proxy / high resolution / default -- The display method for raster images.
  • transparency off / low quality / medium quality / high quality / default -- The display setting for transparencies.
  • vector gray out / proxy / high resolution / default -- The display method for vector images.
