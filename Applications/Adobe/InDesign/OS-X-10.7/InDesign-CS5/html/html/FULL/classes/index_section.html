Class index section

reduce

Description

A section within an index.

  • Plural name:
    • index sections
  • Defined in:
    • Indexing Suite

Properties

  • all topics (r/o) list of topic -- The topics in the specified index section.
  • class (r/o) type class -- The class descriptor type.
  • id (r/o) integer -- The unique ID of the index section.
  • index (r/o) integer -- The index of the index section within its containing object.
  • label string -- A property that can be set to any string.
  • name (r/o) string -- The name of the index section.
  • object reference (r/o) index section -- An object reference for the object.
  • parent (r/o) index -- The parent of the index section (a index).
  • properties record -- A property that allows setting of several properties at the same time.
