Class text variable

reduce

Description

A text variable definition in a document.

  • Plural name:
    • text variables
  • Defined in:
    • Text Suite

Properties

  • associated instances (r/o) list of text variable instance -- Variable instances associated with the text variable.
  • class (r/o) type class -- The class descriptor type.
  • index (r/o) integer -- The index of the text variable within its containing object.
  • name string -- The name of the text variable.
  • object reference (r/o) text variable -- An object reference for the object.
  • parent (r/o) reference -- The parent of the text variable (a application or document). Can return: application or document.
  • properties record -- A property that allows setting of several properties at the same time.
  • variable options (r/o) reference -- The preferences associated with the text variable. Can return: page number variable preference, chapter number variable preference, date variable preference, file name variable preference, match character style preference, match paragraph style prefere...
  • variable type custom text type / file name type / last page number type / chapter number type / output date type / creation date type / modification date type / match character style type / match paragraph style type / xref page number type / xref chapter number type / live caption type -- The variable type.
