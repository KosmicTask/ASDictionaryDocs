Class margin preference

reduce

Description

Margin preferences.

  • Defined in:
    • Preferences Suite

Properties

  • bottom number | string -- The bottom edge of the margin preference.
  • class (r/o) type class -- The class descriptor type.
  • column count integer -- The number of columns to place on the page. . Can return: integer (1 - 216).
  • column direction horizontal / vertical -- The direction of text in the column.
  • column gutter number | string -- The distance between columns. . Can return: number or string (0 - 1440 points).
  • columns positions list of number | list of string -- The distance that each column guide is placed from the left margin, formatted as an array in the format [guide1, guide2, guide3].
  • custom columns (r/o) boolean -- If false, columns are evenly spaced. If true, columns can have custom widths.
  • left number | string -- The left edge of the margin preference.
  • object reference (r/o) margin preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the margin preference (a application, document or page). Can return: application, document or page.
  • properties record -- A property that allows setting of several properties at the same time.
  • right number | string -- The right edge of the margin preference.
  • top number | string -- The top edge of the margin preference.

  • event listener -- by index, name, range, relative, test, id
