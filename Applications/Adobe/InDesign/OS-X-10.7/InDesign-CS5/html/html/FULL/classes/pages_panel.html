Properties

  • class (r/o) type class -- The class descriptor type.
  • properties record -- A property that allows setting of several properties at the same time.
  • visible boolean -- If true, the panel is visible.
  • icon size extra small icon / small icon / medium icon / large icon / extra large icon / jumbo icon -- Page icon size.
  • index (r/o) integer -- The index of the pages panel within its containing object.
  • master icon size extra small icon / small icon / medium icon / large icon / extra large icon / jumbo icon -- Master page icon size.
  • master vertical view boolean -- If true, master page icons will be arranged vertically around the binding spine. If false, master page icons will be arranged horizontally.
  • masters thumbnails boolean -- If true, display thumbnails of the pages in the master pages area of the panel.
  • name (r/o) string -- The name of the pages panel.
  • object reference (r/o) pages panel -- An object reference for the object.
  • pages on top boolean -- If true, the pages area of the panel will be drawn above the master pages area.
  • pages thumbnails boolean -- If true, display thumbnails of the pages in the pages area of the panel.
  • parent (r/o) reference -- The parent of the pages panel (a application).
  • resize behavior proportional resize / pages fixed / masters fixed -- Specifies the behavior of the document and master page areas of the panel when the panel is resized.
  • rotation icons boolean -- If true, a rotation icon will display next to spreads with non-zero rotation applied.
  • transitions icons boolean -- If true, a page transitions icon will display next to spreads with page transitions applied.
  • transparency icons boolean -- If true, a transparency icon will display next to spreads with elements having transparency applied.
  • vertical view boolean -- If true, document page icons will be arranged vertically around the binding spine. If false, page icons will be arranged horizontally.
