Class layout adjustment preference

reduce

Description

Layout adjustment preferences.

  • Defined in:
    • Preferences Suite

Properties

  • allow graphics to resize boolean -- If true, allows graphics to be resized.
  • allow ruler guides to move boolean -- If true, allows ruler guides to move.
  • class (r/o) type class -- The class descriptor type.
  • enable layout adjustment boolean -- If true, layout adjustment is enabled.
  • ignore object or layer locks boolean -- If true, ignores object or layer locks.
  • ignore ruler guide alignments boolean -- If true, ignores ruler guide alignments.
  • object reference (r/o) layout adjustment preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the layout adjustment preference (a application or document). Can return: application or document.
  • properties record -- A property that allows setting of several properties at the same time.
  • snap zone number | string -- The range within which an object snaps to guides. Can return: number or string (0 - 12 points).

Elements

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id
