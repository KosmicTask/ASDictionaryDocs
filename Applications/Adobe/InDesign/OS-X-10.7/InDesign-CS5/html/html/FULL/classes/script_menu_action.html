Properties

  • class (r/o) type class -- The class descriptor type.
  • label string -- A property that can be set to any string.
  • properties record -- A property that allows setting of several properties at the same time.
  • area string -- The menu action area.
  • checked boolean -- If true, the menu item associated with the menu action is checked.
  • enabled boolean -- If true, the script menu action is enabled.
  • id (r/o) integer -- The unique ID of the script menu action.
  • index (r/o) integer -- The index of the script menu action within its containing object.
  • name string -- The name of the script menu action.
  • object reference (r/o) script menu action -- An object reference for the object.
  • parent (r/o) reference -- The parent of the script menu action (a application).
  • title string -- The name of the script menu action for display in the user interface. The title includes any ampersand characters (&), which are used to tell the Windows OS to underline the following character in the name for use with the Alt key to navigate to a menu...
  • after invoke (r/o) string -- Dispatched after the script menu action is invoked. This event does not bubble. This event is not cancelable.
  • before display (r/o) string -- Dispatched before the script menu action is displayed. This event does not bubble. This event is not cancelable.
  • before invoke (r/o) string -- Dispatched before the script menu action is invoked. This event does not bubble. This event is cancelable.
  • on invoke (r/o) string -- Dispatched when the script menu action is invoked. This event does not bubble. This event is not cancelable.
