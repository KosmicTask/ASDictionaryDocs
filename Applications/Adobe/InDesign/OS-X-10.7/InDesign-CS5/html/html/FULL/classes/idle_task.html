Class idle task

reduce

Description

An attachable idle task.

  • Plural name:
    • idle tasks
  • Defined in:
    • Basics Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • id (r/o) integer -- The unique ID of the idle task.
  • index (r/o) integer -- The index of the idle task within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the idle task.
  • object reference (r/o) idle task -- An object reference for the object.
  • parent (r/o) reference -- The parent of the idle task (a application).
  • properties record -- A property that allows setting of several properties at the same time.
  • sleep integer -- Amount of time to sleep (in milliseconds) before calling this task again. Setting this to zero will cause task to be deleted.
  • on idle (r/o) string -- Dispatched at idle time for this idle task. This event does not bubble. This event is not cancelable.

  • event listener -- by index, name, range, relative, test, id
