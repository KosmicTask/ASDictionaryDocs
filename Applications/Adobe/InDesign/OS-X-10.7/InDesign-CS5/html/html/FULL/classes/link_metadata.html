Class link metadata

reduce

Description

MetaData properties for the link source file.

  • Defined in:
    • Links Suite

Properties

  • author (r/o) string -- The author of the document.
  • class (r/o) type class -- The class descriptor type.
  • copyright info URL (r/o) string -- The URL of the file that contains the linked copyright statement.
  • copyright notice (r/o) string -- The text to use as a copyright notice.
  • copyright status (r/o) unknown / yes / no -- The copyright status of the document.
  • creation date (r/o) date -- The creation date of the document.
  • creator (r/o) string -- The name of the application used to create the document.
  • description (r/o) string -- The description of the link metadata.
  • document title (r/o) string -- The title of the document.
  • format (r/o) string -- The format of the document.
  • job name (r/o) string -- The job name.
  • keywords (r/o) list of string -- The list of keywords associated with the document.
  • modification date (r/o) date -- The most recent modification date of the document.
  • object reference (r/o) link metadata -- An object reference for the object.
  • parent (r/o) link -- The parent of the link metadata (a link).
  • properties record -- A property that allows setting of several properties at the same time.
  • server URL (r/o) string -- The location of the document on the asset management server.

Elements

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id

Notes

  • Contained by:
    • link (Links Suite)
    • link metadata (Links Suite)
