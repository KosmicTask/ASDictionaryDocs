Class metadata preference

reduce

Description

Metadata preferences.

  • Defined in:
    • Preferences Suite

Properties

  • author string -- The author of the document.
  • class (r/o) type class -- The class descriptor type.
  • copyright info URL string -- The URL of the file that contains the linked copyright statement.
  • copyright notice string -- The text to use as a copyright notice.
  • copyright status unknown / yes / no -- The copyright status of the document.
  • creation date (r/o) date -- The creation date of the document.
  • creator (r/o) string -- The name of the application used to create the document.
  • description string -- The description of the metadata preference.
  • document title string -- The title of the document.
  • format (r/o) string -- The format of the document.
  • job name string -- The job name.
  • keywords list of string -- The list of keywords associated with the document.
  • modification date (r/o) date -- The most recent modification date of the document.
  • object reference (r/o) metadata preference -- An object reference for the object.
  • parent (r/o) document -- The parent of the metadata preference (a document).
  • properties record -- A property that allows setting of several properties at the same time.
  • server URL (r/o) string -- The location of the document on the asset management server.

Elements

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id

Notes

  • Contained by:
    • document (Basics Suite)
    • metadata preference (Preferences Suite)
