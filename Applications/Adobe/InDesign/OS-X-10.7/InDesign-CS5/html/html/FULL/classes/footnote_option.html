Class footnote option

reduce

Description

Options for specifying default footnote formatting.

  • Defined in:
    • Preferences Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • continuing rule color anything -- The swatch (color, gradient, tint, or mixed ink) applied to the rule above continued footnote text. Note: Valid when continuing rule on is true. Can return: swatch or string.
  • continuing rule gap color anything -- The swatch (color, gradient, tint, or mixed ink) applied to the stroke gap of the rule above continued footnote text. Note: Valid when continuing rule type is not solid. Can return: swatch or string.
  • continuing rule gap overprint boolean -- If true, overprints the gap color of the rule above continued footnote text. Note: Valid when continuing rule type is not solid.
  • continuing rule gap tint fixed -- The tint (as a percentage) of the gap color of the rule above continued footnote text. (Range: 0 to 100) Note: Valid when continuing rule type is not solid. Can return: fixed (0 - 100).
  • continuing rule left indent number | string -- The amount to left indent the rule above continued footnote text. Note: Valid when continuing rule on is true. Can return: number or string (-103680 - 103680 points).
  • continuing rule line weight number | string -- The stroke weight of the rule above continued footnote text. (Range: 0 to 1000) Note: Valid when continuing rule on is true. Can return: number or string (0 - 1000 points).
  • continuing rule offset number | string -- The vertical offset of the rule above continued footnote text. Note: Valid when continuing rule on is true. Can return: number or string (-15552 - 15552 points).
  • continuing rule on boolean -- If true, draws a rule above footnote text that continues from a previous column. Note: Valid when no splitting is false or undefined.
  • continuing rule overprint boolean -- If true, overprints the rule above continued footnote text. Note: Valid when continuing rule on is true.
  • continuing rule tint fixed -- The tint (as a percentage) of the rule above continued footnote text. (Range: 0 to 100) Note: Valid when continuing rule type is not solid. . Can return: fixed (0 - 100).
  • continuing rule type anything -- The stroke type of the rule above continued footnote text. Note: Valid when continuing rule on is true. Can return: stroke style or string.
  • continuing rule width number | string -- The length of the rule above continued footnote text. Note: Valid when continuing rule on is true. Can return: number or string (0 - 103680 points).
  • eos placement boolean -- If true, footnotes at the end of the story are placed just below the text. If false, footnotes at the end of the story are placed at the bottom of the column.
  • footnote first baseline offset ascent offset / cap height / leading offset / embox height / x height / fixed height -- The distance between the top of the footnote container and the footnote text.
  • footnote marker style character style -- The character style to apply to footnote reference numbers in the main text.
  • footnote minimum first baseline offset number | string -- The minimum distance between the baseline of the text and the top of the footnote container. Can return: number or string (0 - 103680 points).
  • footnote numbering style anything -- The footnote numbering style. Can return: upper roman/lower roman/upper letters/lower letters/arabic/symbols/kanji/full width arabic/single leading zeros/double leading zeros/asterisks/arabic alif ba tah/arabic abjad/hebrew biblical/hebrew non standard...
  • footnote text style paragraph style -- The paragraph style to apply to footnotes. Note: The space before and after the paragraph defined in the paragraph style is ignored for footnotes. To define space above and between footnotes, see spacer and space between.
  • marker positioning anything -- The position of footnote reference numbers in the main text. Can return: normal marker/superscript marker/subscript marker/ruby marker or string.
  • no splitting boolean -- If true, footnotes cannot split across columns. If false, footnotes flow into succeeding columns when the footnote text causes the footnote area to expand upward to reach the footnote reference number in the main text.
  • object reference (r/o) footnote option -- An object reference for the object.
  • parent (r/o) reference -- The parent of the footnote option (a application or document). Can return: application or document.
  • prefix string -- The prefix text of the footnote. (Limit: 0 to 100 characters)
  • properties record -- A property that allows setting of several properties at the same time.
  • restart numbering anything -- The point at which to restart footnote numbering. Can return: dont restart/page restart/spread restart/section restart or string.
  • rule color anything -- The swatch (color, gradient, tint, or mixed ink) applied to the stroke of the rule above the first footnote in the column. Note: Valid when rule on is true. Can return: swatch or string.
  • rule gap color anything -- The swatch (color, gradient, tint, or mixed ink) applied to the stroke gap of the rule above the first footnote in the column. Note: Valid when rule type is not solid. Can return: swatch or string.
  • rule gap overprint boolean -- If true, overprints the gap color of the rule above the first footnote in the column. Note: Valid when rule type is not solid.
  • rule gap tint fixed -- The tint (as a percentage) of the gap color of the rule above the first footnote in the column. (Range: 0 to 100) Note: Valid when rule type is not solid. Can return: fixed (0 - 100).
  • rule left indent number | string -- The amount to left indent the rule above the first footnote in the column. Note: Valid when rule on is true. Can return: number or string (-103680 - 103680 points).
  • rule line weight number | string -- The stroke weight of the rule above the first footnote in the column. (Range: 0 to 1000) Note: Valid when rule on is true. Can return: number or string (0 - 1000 points).
  • rule offset number | string -- The vertical offset of the rule above the first footnote in the column. Note: Valid when rule on is true. Can return: number or string (-15552 - 15552 points).
  • rule on boolean -- If true, draws a rule between the text and the first footnote in the column.
  • rule overprint boolean -- If true, overprints the rule above the first footnote in the column. Note: Valid when rule on is true.
  • rule tint fixed -- The tint (as a percentage) of the rule above the first footnote in the column. (Range: 0 to 100) Note: Valid when rule on is true. Can return: fixed (0 - 100).
  • rule type anything -- The stroke type of the rule above the first footnote in a column. Note: Valid when rule on is true. Can return: stroke style or string.
  • rule width number | string -- The length of the rule above the first footnote in the column. Note: Valid when rule on is true. Can return: number or string (0 - 103680 points).
  • separator text string -- The text to insert between the footnote marker number and the footnote text. (Range: 0 to 100 characters)
  • show prefix suffix anything -- The position of the footnote prefix and/or suffix. Can return: no prefix suffix/prefix suffix reference/prefix suffix marker/prefix suffix both or string.
  • space between number | string -- The amount of vertical space between footnotes. Note: The space before and space after defined for the paragraph style applied to the footnote is ignored. Can return: number or string (0 - 864 points).
  • spacer number | string -- The minimum amount of vertical space between the bottom of the text column and the first footnote. Note: The space before amount defined in the paragraph style applied to the footnote is ignored for the first footnote. Can return: number or string (0 -...
  • start at integer -- The number at which to start footnote numbering. Can return: integer (1 - 100000).
  • suffix string -- The suffix text of the footnote. (Limit: 0 to 100 characters)
