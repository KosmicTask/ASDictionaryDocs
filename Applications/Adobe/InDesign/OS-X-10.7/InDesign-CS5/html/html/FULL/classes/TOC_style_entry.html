Class TOC style entry

reduce

Description

A TOC entry definition.

  • Plural name:
    • TOC style entries
  • Defined in:
    • Table Of Contents Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • format style anything -- The paragraph style applied to the TOC entry. Can return: paragraph style or string.
  • index (r/o) integer -- The index of the TOC style entry within its containing object.
  • level small integer -- The indent level of the entry in the TOC.
  • name string -- The name of the TOC style entry.
  • object reference (r/o) TOC style entry -- An object reference for the object.
  • page number position after entry / before entry / none -- The page number placement for the TOC entry style.
  • page number style anything -- The character style applied to the page number of the entry. Can return: character style or string.
  • parent (r/o) TOC style -- The parent of the TOC style entry (a TOC style).
  • properties record -- A property that allows setting of several properties at the same time.
  • separator string -- The string to insert between the entry text and the page numbers.
  • separator style anything -- The character style applied to the string separating the TOC entry text and the page numbers. Can return: character style or string.
  • sort alphabet boolean -- If true, sorts the entries alphabetically.

  • event listener -- by index, name, range, relative, test, id
