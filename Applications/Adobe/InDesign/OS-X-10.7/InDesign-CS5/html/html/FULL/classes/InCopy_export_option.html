Class InCopy export option

reduce

Description

Export options for InCopy INCX document format.

  • Defined in:
    • Preferences Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • include all resources boolean -- If true, export all resources (styles etc), otherwise export resources used by the story.
  • include graphic proxies boolean -- If true, include graphic proxy data.
  • object reference (r/o) InCopy export option -- An object reference for the object.
  • parent (r/o) reference -- The parent of the InCopy export option (a application, story, xml story, oval, rectangle or polygon). Can return: application, story, xml story, oval, rectangle or polygon.
  • properties record -- A property that allows setting of several properties at the same time.

Elements

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id
