Class layer

reduce

Description

A layer.

  • Plural name:
    • layers
  • Defined in:
    • Layout Suite

Properties

  • all graphics (r/o) list of graphic -- Lists all graphics contained by the layer.
  • all page items (r/o) list of page item -- Lists all page items contained by the layer.
  • class (r/o) type class -- The class descriptor type.
  • id (r/o) integer -- The unique ID of the layer.
  • ignore wrap boolean -- If true, text wrap settings applied to objects on the layer will not affect text on other layers when the layer is hidden.
  • index (r/o) integer -- The index of the layer within its containing object.
  • label string -- A property that can be set to any string.
  • layer color anything -- The color of the layer, specified either as an array of three doubles, each in the range 0 to 255 and representing R, G, and B values, or as a UI color. Can return: list of 3 fixed (0 - 255) or light blue/red/green/blue/yellow/magenta/cyan/gray/black/o...
  • lock guides boolean -- If true, the guide positions on the layer are locked.
  • locked boolean -- If true, the layer is locked.
  • name string -- The name of the layer.
  • object reference (r/o) layer -- An object reference for the object.
  • parent (r/o) document -- The parent of the layer (a document).
  • printable boolean -- If true, the layer will print.
  • properties record -- A property that allows setting of several properties at the same time.
  • show guides boolean -- If true, guides are visible on the layer.
  • visible boolean -- If true, the layer is visible.
