Class XML view preference

reduce

Description

XML view preferences

  • Defined in:
    • Preferences Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • object reference (r/o) XML view preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the XML view preference (a application or document). Can return: application or document.
  • properties record -- A property that allows setting of several properties at the same time.
  • show attributes boolean -- If true, displays attributes as well as elements in the structure view. Note: Valid only when show structure is true.
  • show structure boolean -- If true, displays the structure view.
  • show tag markers boolean -- If true, displays XML tags.
  • show tag options (r/o) boolean -- If true, displays the tag options dialog when tagging any item whose parent is not tagged.
  • show tagged frames boolean -- If true, displays XML tags in tagged frames.
  • show text snippets boolean -- If true, the structure view displays text snippets of element content. Note: Valid only when show structure is true.
