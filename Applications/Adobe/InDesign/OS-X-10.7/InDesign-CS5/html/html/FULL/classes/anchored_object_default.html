Class anchored object default

reduce

Description

Anchored object default settings.

  • Defined in:
    • Preferences Suite

Properties

  • anchor content unassigned / graphic type / text type -- The initial frame type of a new anchored object.
  • anchored object style object style -- The initial object style of a new anchored object.
  • anchored paragraph style paragraph style -- The initial paragraph style of a new anchored object. Note: Valid when anchor content is text.
  • class (r/o) type class -- The class descriptor type.
  • initial anchor height number | string -- The initial height of a new anchored object.
  • initial anchor width number | string -- The initial width of a new anchored object.
  • object reference (r/o) anchored object default -- An object reference for the object.
  • parent (r/o) reference -- The parent of the anchored object default (a application or document). Can return: application or document.
  • properties record -- A property that allows setting of several properties at the same time.
