Description

An XML processing instruction.

  • Plural name:
    • XML instructions
  • Defined in:

  • Inherits from:

Properties

  • class (r/o) type class -- The class descriptor type.
  • properties record -- A property that allows setting of several properties at the same time.
  • data string -- A value that tells the application reading the exported XML file what to do with the processing instruction.
  • id (r/o) integer -- The unique ID of the XML instruction.
  • index (r/o) integer -- The index of the XML instruction within its containing object.
  • object reference (r/o) XML instruction -- An object reference for the object.
  • parent (r/o) reference -- The parent of the XML instruction (a document or XML element). Can return: document or XML element.
  • story offset (r/o) insertion point -- The insertion point before the table in the story containing the table.
  • target string -- A name that identifies the processing instruction to an application reading the exported XML file.
