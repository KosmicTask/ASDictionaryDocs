Class master spread

reduce

Description

A master spread.

  • Plural name:
    • master spreads
  • Defined in:
    • Layout Suite

Properties

  • all graphics (r/o) list of graphic -- Lists all graphics contained by the master spread.
  • all page items (r/o) list of page item -- Lists all page items contained by the master spread.
  • applied master master spread -- The master spread applied to the master spread. Can also accept: nothing.
  • base name string -- The name of the master spread.
  • class (r/o) type class -- The class descriptor type.
  • id (r/o) integer -- The unique ID of the master spread.
  • IDML component name string -- The IDML component name of the master spread.
  • index (r/o) integer -- The index of the master spread within its containing object.
  • label string -- A property that can be set to any string.
  • name (r/o) string -- The name of the master spread.
  • name prefix string -- The prefix of the master spread name.
  • object reference (r/o) master spread -- An object reference for the object.
  • page color anything -- The color label of the master spread, specified either as an array of three doubles, each in the range 0 to 255 and representing R, G, and B values, or as a UI color. Can return: list of 3 fixed (0 - 255) or light blue/red/green/blue/yellow/magenta/cy...
  • parent (r/o) document -- The parent of the master spread (a document).
  • properties record -- A property that allows setting of several properties at the same time.
  • show master items boolean -- If true, displays master page items on document pages in the spread.
  • timing settings (r/o) timing setting -- The object timing settings.
