Description

Generic term for an object in a dialog column or dialog row, including dialog controls such as radio button groups, checkboxes, editboxes, dropdowns, and comboboxes; static text objects; enabling groups; border panels; and nested dialog columns and dia...

Properties

  • class (r/o) type class -- The class descriptor type.
  • id (r/o) integer -- The unique ID of the widget.
  • index (r/o) integer -- The index of the widget within its containing object.
  • min width fixed -- The width of the control. For an editbox or combobox, specifies the minimum width of the box.
  • object reference (r/o) widget -- An object reference for the object.
  • parent (r/o) reference -- The parent of the widget (a dialog column, dialog row, enabling group, border panel or radiobutton group). Can return: dialog column, dialog row, enabling group, border panel or radiobutton group.
  • properties record -- A property that allows setting of several properties at the same time.
