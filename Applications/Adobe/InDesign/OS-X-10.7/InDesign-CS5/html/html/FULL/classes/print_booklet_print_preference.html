Class print booklet print preference

reduce

Description

Print booklet preferences.

  • Defined in:
    • Preferences Suite

Properties

  • active printer preset anything -- The current printer preset type. Can return: default/custom or printer preset.
  • all printer marks boolean -- If true, prints all printer marks. If false, prints specified printer marks.
  • bitmap printing boolean -- If true, uses bitmap printing.
  • bitmap resolution integer -- The resolution for bitmap printing. (Range: 72 to 1200) Note: Valid when bitmap printing is true. Can return: integer (72 - 1200).
  • black angle fixed -- The angle override for black ink. (Range: 0 to 360)
  • black frequency fixed -- The frequency override for black ink. (Range: 1 to 500)
  • bleed bottom number | string -- The height of the bleed area at the bottom of the page. Note: Valid only when use document bleed to print is true. Can return: number or string (0 - 432 points).
  • bleed chain boolean -- If true, forces all bleed area settings to be the same, using the most recent bleed measurement setting. If false, allows bleed top, bleed bottom, bleed inside, and bleed outside to have different measurements.
  • bleed inside number | string -- The width of the bleed area at the inside of the page. Note: Valid only when use document bleed to print is true. Can return: number or string (0 - 432 points).
  • bleed marks boolean -- If true, print bleed marks.
  • bleed outside number | string -- The width of the bleed area at the outside of the page. Note: Valid only when use document bleed to print is true. . Can return: number or string (0 - 432 points).
  • bleed top number | string -- The height of the bleed area at the top of the page. Note: Valid only when use document bleed to print is true. Can return: number or string (0 - 432 points).
  • class (r/o) type class -- The class descriptor type.
  • collating boolean -- If true, collate printed copies.
  • color bars boolean -- If true, add small squares of color representing the CMYK inks and tints of gray in 10% increments.
  • color output composite leave unchanged / composite gray / composite RGB / composite CMYK / separations / inRIP separations -- The color output mode for composites. Note: Not valid when a device-independent PPD is specified.
  • composite angle fixed -- The screen angle to use when printing composites. (Range: 0 to 360) Note: Valid only for PostScript or PDF files that use custom screening.
  • composite frequency fixed -- The screen frequency to use when printing composites. (Range: 1 to 500) Note: Valid only for PostScript or PDF files that use custom screening.
  • copies integer -- The number of copies to print. Note: Not valid when printer is PostScript File.
  • CRD anything -- The color-rendering dictionary (CRD), specified as a CRD name or an enumeration value. Note: Valid only when use color management is true. Can return: default/use document/working or string.
  • crop marks boolean -- Prints crop marks that define where the page should be trimmed.
  • cyan angle fixed -- The angle override for cyan ink. (Range: 0 to 360)
  • cyan frequency fixed -- The frequency override for cyan ink. (Range: 1 to 500)
  • data format ASCII / binary -- The format in which to send image data to the printer.
  • download PPD fonts boolean -- If true, downloads all fonts listed in the selected PPD. Valid only when font downloading is complete or subset.
  • flattener preset name string -- The name of the transparency flattener preset.
  • flip none / horizontal / vertical / horizontal and vertical / both -- The direction in which to flip the printed image.
  • font downloading none / complete / subset / subset large -- Controls how fonts are downloaded to the printer.
  • ignore spread overrides boolean -- If true, ignores flattener spread overrides.
  • intent use color settings / perceptual / saturation / relative colorimetric / absolute colorimetric -- The rendering intent. Note: Valid only when use color management is true.
  • magenta angle fixed -- The angle override for magenta ink. (Range: 0 to 360)
  • magenta frequency fixed -- The frequency override for magenta ink. (Range: 1 to 500)
  • mark line weight p125pt / p25pt / p50pt / p05mm / p07mm / p10mm / p15mm / p20mm / p30mm -- The stroke weight (in points) for printer marks.
  • mark offset number | string -- The distance to offset the page marks from the edge of the page.
  • mark type anything -- The type of printer marks, either an enum value or the name of a custom marks file. Can return: default/J mark with circle/J mark without circle or string.
  • negative boolean -- If true, prints the document as a negative.
  • object reference (r/o) print booklet print preference -- An object reference for the object.
  • omit bitmaps boolean -- If true, replaces bitmap images with OPI links.
  • omit EPS boolean -- If true, replaces EPS images with OPI links.
  • omit PDF boolean -- If true, replaces PDF images with OPI links.
  • OPI image replacement boolean -- If true, prints graphics that are either OPI comments stored in imported EPS files or linked using OPI comments. For information on linking files using OPI comments, see omit EPS, omit PDF, or omit bitmaps.
  • page information marks boolean -- If true, prints the filename, page number, current date and time, and color separation name.
  • page position upper left / center horizontally / center vertically / centered -- The position of the page on the printing medium. Note: Valid only when tile is false.
  • paper gap number | string -- The space between document pages on the printing medium.
  • paper height anything -- The paper height. Note: Valid only when paper size is custom or scale mode is scale width height. Can return: auto or number or string.
  • paper offset number | string -- The amount of space to offset the page from the left edge of the imageable area.
  • paper size anything -- The paper size, specified as either a string or an enumeration. For information on paper size names, see paper size list. Can return: defined by driver/custom or string.
  • paper size list (r/o) list of string -- A list of the available paper sizes.
  • paper transverse boolean -- If true, uses transverse orientation.
  • paper width anything -- The paper width. Note: Valid only when paper size is custom or scale mode is scale width height. Can return: auto or number or string.
  • parent (r/o) document -- The parent of the print booklet print preference (a document).
  • PostScript level level 2 / level 3 -- The PostScript level of the printer.
  • PPD anything -- The PPD, specified as a PPD name or an enumeration. Can return: device independent or string.
  • PPD list (r/o) list of string -- Available PPDs.
  • preserve color numbers boolean -- If true, preserves uncalibrated color numbers.
  • print black boolean -- If true, prints the black ink. Note: Valid only when trapping is off.
  • print blank pages boolean -- If true, prints blank pages. Note: Valid only when trapping is off.
  • print cyan boolean -- If true, prints the cyan ink. Note: Valid only when trapping is off.
  • print file alias | string -- The PostScript file to print to. Note: Valid only when the current printer is defined as postscript file.
  • print guides grids boolean -- If true, prints visible guides and baseline grids. Note: Valid only when trapping is off.
  • print layers all layers / visible layers / visible printable layers -- The layers to print.
  • print magenta boolean -- If true, prints the magenta ink. Note: Valid only when trapping is off.
  • print nonprinting boolean -- If true, prints non-printing objects. Note: Valid only when trapping is off.
  • print page orientation portrait / landscape / reverse portrait / reverse landscape -- The orientation of the printed page.
  • print yellow boolean -- If true, prints the yellow ink. Note: Valid only when trapping is off.
  • printer anything -- The current printer. Can return: postscript file or string.
  • printer list (r/o) list of string -- Available printers.
  • profile anything -- The color profile. Can return: PostScript CMS/use document/working/No CMS or string.
  • properties record -- A property that allows setting of several properties at the same time.
  • registration marks boolean -- If true, prints small targets outside the page area for aligning color separations.
  • reverse order boolean -- If true, prints pages in reverse order.
  • scale height fixed -- The amount (as a percentage) that the page height is scaled during printing. (Range: 0 to 1000) Note: Valid only when scale mode is scale width height.
  • scale mode scale width height / scale to fit -- The policy for scaling the page. Note: Valid only when printing from Layout view.
  • scale proportional boolean -- If true, constrains the proportions of the scaling; uses the most recent value for either scale width or scale height to define both values. Note: Valid only when scale mode is scale width height.
  • scale width fixed -- The amount (as a percentage) that the page width is scaled during printing. (Range: 0 to 1000) Note: Valid only when scale mode is scale width height.
  • screening anything -- The ink screening settings for composite gray output in PostScript or PDF format. . Can return: default/custom or string.
  • screening list (r/o) list of string -- Lists the ink screenings available in the PPD. Note: Valid only when color output is separations or in rip separations.
  • send image data all image data / optimized subsampling / proxy image data / none -- The image data sent to the printer or file.
  • simulate overprint boolean -- If true, simulates the effects of overprinting spot inks with different neutral density values by converting spot colors to process colors for printing. Note: Not valid when the color output mode is defined to leave color profiles unchanged.
  • source space use document / proof space -- The source of the color management system. Note: Valid only when use color management is true.
  • text as black boolean -- If true, prints all text as black unless text has the color None or Paper or a color value that equals white. If false, prints colored text, such as blue hyperlinks, in halftone patterns. Note: Valid only when trapping is off.
  • trapping off / application builtin / Adobe inRIP -- The type of trapping.
  • use document bleed to print boolean -- If true, uses the bleed area set for the document.
  • yellow angle fixed -- The angle override for yellow ink. (Range: 0 to 360)
  • yellow frequency fixed -- The frequency override for yellow ink. (Range: 1 to 500)
