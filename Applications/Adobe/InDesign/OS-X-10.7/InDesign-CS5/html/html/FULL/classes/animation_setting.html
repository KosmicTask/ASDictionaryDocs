Class animation setting

reduce

Description

Animation settings.

  • Defined in:
    • Preferences Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • design option from current appearance / to current appearance / to current location -- The animation design option.
  • duration fixed -- The duration in second for this animation. Can return: fixed (0.125 - 60).
  • ease type no ease / ease in / ease out / ease in out / custom ease -- The ease type.
  • has custom settings (r/o) boolean -- Determines if this animated object has custom settings.
  • hidden after boolean -- Determines if this object is hidden after its animation is played in an exported SWF file.
  • initially hidden boolean -- Determines if this object is initially hidden when displayed in an exported SWF file.
  • motion path list of anything -- The list of motion path points and key frames for this animation. Can return: Ordered list containing key frame:integer, path point:Ordered list containing anchor:list of 2 fixed, left direction:list of 2 fixed, right direction:list of 2 fixed.
  • motion path points anything -- The list of motion path points for this animation. Can return: Ordered list containing path point array:list of Ordered list containing anchor:list of 2 fixed, left direction:list of 2 fixed, right direction:list of 2 fixed, path open:boolean.
  • object reference (r/o) animation setting -- An object reference for the object.
  • opacity array list of anything -- The list of opacity key frames for this animation. Can return: Ordered list containing key frame:integer, value:fixed.
  • parent (r/o) reference -- The parent of the animation setting (a page item, form field, multi state object, button, media item, sound, movie, EPSText, spline item, polygon, graphic line, rectangle, oval, graphic, imported page, PICT, WMF, PDF, EPS, image, group or text frame). ...
  • plays integer -- The number of times this animation plays. Can return: integer (1 - 100).
  • plays loop boolean -- Set to true if animation play loops.
  • preset anything -- The base animation preset applied. Can return: Motion Preset, string or nothing.
  • properties record -- A property that allows setting of several properties at the same time.
  • rotation array list of anything -- The list of rotation key frames for this animation. Can return: Ordered list containing key frame:integer, value:fixed.
  • scale x array list of anything -- The list of scale x key frames for this animation. Can return: Ordered list containing key frame:integer, value:fixed.
  • scale y array list of anything -- The list of scale y key frames for this animation. Can return: Ordered list containing key frame:integer, value:fixed.
  • transform offsets list of fixed -- The tranform offset percentage from the target object bounding box's left-top corner.

Elements

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id

Notes

  • Contained by:
    • animation setting (Preferences Suite)
