Class JPEG export preference

reduce

Description

JPEG export preferences.

  • Defined in:
    • Preferences Suite

Properties

  • anti alias boolean -- If true, use anti-aliasing for text and vectors during export.
  • class (r/o) type class -- The class descriptor type.
  • embed color profile boolean -- True to embed the color profile, false otherwise.
  • export resolution fixed -- The export resolution expressed as a real number instead of an integer. (Range: 1.0 to 2400.0). Can return: fixed (1 - 2400).
  • Exporting Spread boolean -- If true, exports each spread as a single JPEG file. If false, exports facing pages as separate files and appends sequential numbers to each file name.
  • jpeg color space RGB / CMYK / gray -- One of RGB, CMYK or Gray
  • JPEG export range export range / export all -- The page range to export.
  • JPEG Quality low / medium / high / maximum -- The compression quality.
  • JPEG Rendering style baseline encoding / progressive encoding -- The rendering style.
  • object reference (r/o) JPEG export preference -- An object reference for the object.
  • Page String string -- The page(s) to export, specified as a page number or an array of page numbers. Note: Valid when JPEG export range is not all.
  • parent (r/o) reference -- The parent of the JPEG export preference (a application).
  • properties record -- A property that allows setting of several properties at the same time.
  • simulate overprint boolean -- If true, simulates the effects of overprinting spot and process colors in the same way they would occur when printing.
  • use document bleeds boolean -- If true, uses the document's bleed settings in the exported JPEG.

  • event listener -- by index, name, range, relative, test, id
