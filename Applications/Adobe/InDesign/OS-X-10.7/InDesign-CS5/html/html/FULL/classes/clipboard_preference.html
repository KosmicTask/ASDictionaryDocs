Class clipboard preference

reduce

Description

Clipboard preferences.

  • Defined in:
    • Preferences Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • copy PDF to clipboard boolean -- If true, copies PDF to the clipboard.
  • object reference (r/o) clipboard preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the clipboard preference (a application).
  • paste remembers layers boolean -- If true, objects cut or copied from different layers retain their layer assignment when pasted.
  • prefer PDF when pasting boolean -- If true, pastes PDF if available.
  • prefer styled text when pasting boolean -- If true, includes text attributes when pasting text.
  • preserve pdf clipboard at quit boolean -- If true, preserves PDF data on the system clipboard when the application exits.
  • properties record -- A property that allows setting of several properties at the same time.

Elements

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id
