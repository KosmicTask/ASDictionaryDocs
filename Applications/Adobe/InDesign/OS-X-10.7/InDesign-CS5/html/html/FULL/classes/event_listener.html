Class event listener

reduce

Description

An event listener.

  • Plural name:
    • event listeners
  • Defined in:
    • Basics Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • event type (r/o) string -- The name of the event.
  • handler (r/o) anything -- The handler to invoke when the event occurs. Can return: alias or string or JavaScript Function.
  • id (r/o) integer -- The unique ID of the event listener.
  • index (r/o) integer -- The index of the event listener within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the event listener; this is an alias to the event listener's label property.
  • object reference (r/o) event listener -- An object reference for the object.
  • parent (r/o) reference -- The parent of the event listener (a object, UID based object, cell style group, cell style, table style group, condition set, hidden text, condition, Motion Preset, assigned story, assignment, object style group, object style, numbering list, snippet, ...
  • properties record -- A property that allows setting of several properties at the same time.
