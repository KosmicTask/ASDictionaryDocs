Properties

  • class (r/o) type class -- The class descriptor type.
  • properties record -- A property that allows setting of several properties at the same time.
  • angle anything -- The angle of the gradient feather. Can return: fixed (-180 - 180) or nothing.
  • applied anything -- If true, the gradient feather effect is applied. Can return: boolean or nothing.
  • gradient start anything -- The center point (for a radial gradient) or starting point (for a linear gradient) applied to the fill, as page coordinates in the format [x, y]. Can return: list of 2 numbers or strings or nothing.
  • hilite angle anything -- The hilite angle of the radial gradient feather. Can return: fixed or nothing.
  • hilite length anything -- The hilite length of the radial gradient feather. Can return: number or string or nothing.
  • length anything -- The length of the axial gradient, or radius of the radial gradient. Can return: number or string or nothing.
  • object reference (r/o) find change gradient feather setting -- An object reference for the object.
