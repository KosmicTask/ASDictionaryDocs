Description

Properties

  • anti alias type gray antialiasing / color antialiasing / thicker antialiasing -- The type of text anti-aliasing to use in story and galley views.
  • background color anything -- The background color, specified either as an array of three doubles, each in the range 0 to 255 and representing R, G, and B values, or as an InCopy UI color. . Can return: list of 3 fixed (0 - 255) or canary/lemon/electrolyte/lime/forest/lichen/jade/a...
  • blink cursor boolean -- If true, the cursor blinks.
  • class (r/o) type class -- The class descriptor type.
  • cursor type standard cursor / thick cursor / barbell cursor / block cursor -- The cursor type for galley and story views.
  • display font string -- Font family name to use for text display.
  • display font size number | string -- Size to use for text display.
  • info column width number | string -- Info column width.
  • line spacing value single space / one and half space / double space / triple space -- Amount of spacing between lines.
  • object reference (r/o) galley preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the galley preference (a application or document). Can return: application or document.
  • properties record -- A property that allows setting of several properties at the same time.
  • show depth ruler boolean -- If true, displays the depth ruler.
  • show info column boolean -- If true, display the Info column.
  • show paragraph break marks boolean -- If true, show paragraph break marks.
  • show paragraph style names boolean -- If true, show paragraph style names.
  • smooth text boolean -- If true, galley text is anti-aliased.
  • text color anything -- The text color, specified either as an array of three doubles, each in the range 0 to 255 and representing R, G, and B values, or as an InCopy UI color. Can return: list of 3 fixed (0 - 255) or canary/lemon/electrolyte/lime/forest/lichen/jade/aqua/cirr...
