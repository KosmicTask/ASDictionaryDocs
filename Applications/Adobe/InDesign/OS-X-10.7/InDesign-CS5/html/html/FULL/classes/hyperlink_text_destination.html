Class hyperlink text destination

reduce

Description

A hyperlink destination that is either text or an insertion point.

  • Plural name:
    • hyperlink text destinations
  • Defined in:
    • Hyperlinks Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • destination text insertion point -- The text or insertion point that the hyperlink points to. Can also accept: text.
  • hidden (r/o) boolean -- If true, the hyperlink is hidden.
  • id (r/o) integer -- The unique ID of the hyperlink text destination.
  • index (r/o) integer -- The index of the hyperlink text destination within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the hyperlink text destination.
  • object reference (r/o) hyperlink text destination -- An object reference for the object.
  • parent (r/o) document -- The parent of the hyperlink text destination (a document).
  • properties record -- A property that allows setting of several properties at the same time.
