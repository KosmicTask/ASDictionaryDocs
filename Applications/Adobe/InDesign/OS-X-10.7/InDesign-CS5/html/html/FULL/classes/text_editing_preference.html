Class text editing preference

reduce

Description

Text editing preferences.

  • Defined in:
    • Preferences Suite

Properties

  • allow drag and drop text in story boolean -- If true, allows text to be dragged and dropped in the story editor or galley view.
  • class (r/o) type class -- The class descriptor type.
  • drag and drop text in layout boolean -- If true, allows text to be dragged and dropped on a layout page.
  • object reference (r/o) text editing preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the text editing preference (a application).
  • properties record -- A property that allows setting of several properties at the same time.
  • single click converts frames to text frames boolean -- If true, a single click (with the Type tool) converts non-text frames to text frames.
  • smart cut and paste boolean -- If true, automatically adjusts spacing among words and between words and punctuation marks when cutting and pasting text.
  • triple click selects line boolean -- If true, a triple click selects a line of text. If false, a triple click selects a paragraph.

Elements

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id
