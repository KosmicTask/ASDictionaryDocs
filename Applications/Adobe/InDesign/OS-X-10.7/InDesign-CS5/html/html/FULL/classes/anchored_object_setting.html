Class anchored object setting

reduce

Description

The settings for an anchored object.

  • Defined in:
    • Preferences Suite

Properties

  • anchor point top left anchor / top center anchor / top right anchor / left center anchor / center anchor / right center anchor / bottom left anchor / bottom center anchor / bottom right anchor -- The point in the anchored object to position.
  • anchor space above number | string -- The space above an above-line anchored object.
  • anchor xoffset number | string -- The horizontal (x) offset of the anchored object.
  • anchor yoffset number | string -- The vertical (y) offset of the anchored object. Corresponds to the space after property for above line positioning.
  • anchored position inline position / above line / anchored -- The position of the anchored object relative to the anchor.
  • class (r/o) type class -- The class descriptor type.
  • horizontal alignment right align / left align / center align / text align -- When anchored position is above line, the position of the anchored object is relative to the text area. When anchored position is custom, the horizontal alignment of the anchored object is set by the horizontal reference point. Note: Not valid when anc...
  • horizontal reference point column edge / text frame / page margins / page edge / anchor location -- The horizontal reference point on the page. Valid only when anchored position is custom.
  • lock position boolean -- If true, prevents manual positioning of the anchored object.
  • object reference (r/o) anchored object setting -- An object reference for the object.
  • parent (r/o) reference -- The parent of the anchored object setting (a application, document, EPSText, spline item, polygon, graphic line, rectangle, oval, group, text frame, button, form field, multi state object or object style). Can return: application, document, EPSText, sp...
  • pin position boolean -- If true, pins the position of the anchored object within the text frame top and bottom.
  • properties record -- A property that allows setting of several properties at the same time.
  • spine relative boolean -- If true, the position of the anchored object is relative to the binding spine of the page or spread.
  • vertical alignment top align / bottom align / center align -- The vertical alignment of the anchored object reference point with the vertical reference point on the page. Notes: Valid only when anchored position is custom.
  • vertical reference point column edge / text frame / page margins / page edge / line baseline / line xheight / line ascent / capheight / top of leading / embox top / embox middle / embox bottom -- The vertical reference point on the page. Valid when anchored position is custom.
