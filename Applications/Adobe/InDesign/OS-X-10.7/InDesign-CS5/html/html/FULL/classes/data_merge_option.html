Class data merge option

reduce

Description

Data merge options.

  • Defined in:
    • Preferences Suite

Properties

  • center image boolean -- If true, centers the image in the frame; preserves the frame size as well as content size and proportions. Note: If the content is larger than the frame, content around the edges is obscured by the bounding box of the frame.
  • class (r/o) type class -- The class descriptor type.
  • create new document boolean -- If true, creates a new document when records are merged.
  • document size integer -- The maximum number of pages per document.
  • fitting option proportional / fit content to frame / fit frame to content / preserve sizes / fill proportional -- Instructions for fitting content in a frame.
  • link images boolean -- If true, links images to the target document. If false, embeds images in the target document.
  • object reference (r/o) data merge option -- An object reference for the object.
  • parent (r/o) reference -- The parent of the data merge option (a application or document). Can return: application or document.
  • properties record -- A property that allows setting of several properties at the same time.
  • remove blank lines boolean -- If true, removes blank lines caused by empty fields.

contains events , event listeners

  • event listener -- by index, name, range, relative, test, id
