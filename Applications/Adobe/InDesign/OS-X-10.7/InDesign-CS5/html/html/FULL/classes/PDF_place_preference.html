Description

PDF place preferences.

Properties

  • class (r/o) type class -- The class descriptor type.
  • object reference (r/o) PDF place preference -- An object reference for the object.
  • open document password string -- The password to enter when opening the PDF document. Valid only when use security is true. Note: A script can set but not get this value.
  • page number integer -- The page number of the PDF document page to place.
  • parent (r/o) reference -- The parent of the PDF place preference (a application).
  • PDF crop crop art / crop PDF / crop trim / crop bleed / crop media / crop content visible layers / crop content all layers -- The type of cropping to apply.
  • properties record -- A property that allows setting of several properties at the same time.
  • transparent background boolean -- If true, the background of the PDF is transparent.
