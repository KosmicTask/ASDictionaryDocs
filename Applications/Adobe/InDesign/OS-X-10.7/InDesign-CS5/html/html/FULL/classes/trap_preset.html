Class trap preset

reduce

Description

A trap preset.

  • Plural name:
    • trap presets
  • Defined in:
    • Color Suite

Properties

  • black color threshold fixed -- The minimum amount (as a percentage) of black ink required before the black width setting is applied. (Range: 0 to 100). Can return: fixed (0 - 100).
  • black density fixed -- The neutral density value at or above which an ink is considered black. (Range: .001 to 10). Can return: fixed (0 - 10).
  • black width number | string -- The black width. (Range depends on unit. For points: 0.0 to 8.0; picas: 0p0 to 0p8; inches: 0 to 0.1111; mm: 0 to 2.822; cm: 0 to .2822; ciceros: 0c0 to 0c7.507). Can return: number or string (0 - 8 points).
  • class (r/o) type class -- The class descriptor type.
  • color reduction fixed -- The degree (as a percentage) to which components from abutting colors are used to reduce the trap color. (Range: 0 to 100) Note: 0% makes a trap whose neutral density is equal to the neutral density of the darker color. Can return: fixed (0 - 100).
  • default trap width number | string -- The default width for trapping all colors except those involving solid black. (Range depends on unit. For points: 0.0 to 8.0; picas: 0p0 to 0p8; inches: 0 to 0.1111; mm: 0 to 2.822; cm: 0 to .2822; ciceros: 0c0 to 0c7.507). Can return: number or string...
  • id (r/o) integer -- The unique ID of the trap preset.
  • image placement center edges / choke / image neutral density / images over spread -- The trap placement between vector objects and bitmap images.
  • images to images boolean -- If true, turns on trapping along the boundary of overlapping or abutting bitmap images.
  • index (r/o) integer -- The index of the trap preset within its containing object.
  • internal images boolean -- If true, turns on trapping among colors within individual bitmap images.
  • label string -- A property that can be set to any string.
  • name string -- The name of the trap preset.
  • object reference (r/o) trap preset -- An object reference for the object.
  • objects to images boolean -- If true, ensures that vector objects overlap bitmap images.
  • one bit images boolean -- If true, ensures that one-bit images trap to abutting objects.
  • parent (r/o) reference -- The parent of the trap preset (a application or document). Can return: application or document.
  • properties record -- A property that allows setting of several properties at the same time.
  • sliding trap threshold fixed -- The difference (as a percentage) between the neutral densities of abutting colors at which the trap is moved from the darker side of a color edge toward the centerline. (Range: 0 to 100). Can return: fixed (0 - 100).
  • step threshold fixed -- The amount (as a percentage) that components of abutting colors must vary before a trap is created. (Range: 1 to 100). Can return: fixed (1 - 100).
  • trap end miter trap ends / overlap trap ends -- The shape to use at the intersection of three-way traps.
  • trap join miter end join / round end join / bevel end join -- The join type of the trap preset.
