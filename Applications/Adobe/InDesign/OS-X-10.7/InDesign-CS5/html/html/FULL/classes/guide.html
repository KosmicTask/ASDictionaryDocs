Class guide

reduce

Description

A guide.

  • Plural name:
    • guides
  • Defined in:
    • Layout Suite

Properties

  • allow overrides boolean -- If true, the master page item can be overridden.
  • class (r/o) type class -- The class descriptor type.
  • fit to page boolean -- If true, horizontal orientation guides stop at the edges of the specified page. If false, the guides extends across the width of the spread and into the pasteboard area.
  • guide color anything -- The color of the guide, specified either as an array of three doubles, each in the range 0 to 255 and representing R, G, and B values, or as a UI color. Can return: list of 3 fixed (0 - 255) or light blue/red/green/blue/yellow/magenta/cyan/gray/black/o...
  • id (r/o) integer -- The unique ID of the guide.
  • index (r/o) integer -- The index of the guide within its containing object.
  • item layer layer -- The layer that the guide is on.
  • label string -- A property that can be set to any string.
  • location number | string -- The location at which to place the guide relative to the current ruler zero point.
  • locked boolean -- If true, the guide is locked.
  • name string -- The name of the guide; this is an alias to the guide's label property.
  • object reference (r/o) guide -- An object reference for the object.
  • orientation horizontal / vertical -- The orientation of the guide.
  • overridden (r/o) boolean -- If true, the object originated on a master spread and was overridden. If false, the object either originated on a master spread and was not overridden, or the object did not originate on a master page.
  • overridden master page item (r/o) reference -- An object that originated on a master page and has been overridden. Can return: page item, guide, graphic, movie or sound.
  • parent (r/o) reference -- The parent of the guide (a spread or master spread). Can return: spread or master spread.
  • parent page (r/o) page -- The page on which this page item appears.
  • properties record -- A property that allows setting of several properties at the same time.
  • view threshold fixed -- The view magnification as a percentage below which guides are no longer displayed. (Range: 5.0 to 4000.0). Can return: fixed (5 - 4000).
  • after place (r/o) string -- Dispatched after a guide is placed. This event bubbles. This event is not cancelable.
