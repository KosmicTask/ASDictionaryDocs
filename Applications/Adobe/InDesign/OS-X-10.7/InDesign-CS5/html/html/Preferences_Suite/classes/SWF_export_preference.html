Class SWF export preference

expand

Description

SWF export settings for the application object.

  • Defined in:
    • Preferences Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • curve quality minimum / low / medium / high / maximum -- The SWF curve quality.
  • dynamic document export reader spreads boolean -- If true, each spread in the exported document is combined into a single page that has spread's original width.
  • dynamic media handling include all media / appearance only -- The dynamic media handling options.
  • fit method fit predefined settings / fit given width and height / fit given scale percentage -- The fitting method to use.
  • fit option fit1280x800 / fit1240x620 / fit1024x768 / fit984x588 / fit800x600 / fit760x420 / fit640x480 / fit600x300 -- The options for fitting to predefined dimension.
  • fit scale fixed -- The size fits to given scale percentage. Can return: fixed (1 - 1000).
  • fit width and height list of integer -- The size fits to given width and height.
  • flatten transparency boolean -- Flatten transparency when exporting.
  • frame rate integer -- The frame rate in frames per second. Can return: integer (1 - 120).
  • generate HTML boolean -- Flag indicates whether to generate HTML.
  • include interactive page curl boolean -- Flag indicates whether to include interactive page curl when export to SWF.
  • JPEG quality options minimum / low / medium / high / maximum -- The JPEG quality options.
  • object reference (r/o) SWF export preference -- An object reference for the object.
  • page range anything -- The pages to print, specified either as an enumeration or a string. To specify a range, separate page numbers in the string with a hyphen (-). To specify separate pages, separate page numbers in the string with a comma (,). Can return: all pages/select...
  • page transition override from document / none / blinds transition / box transition / comb transition / cover transition / dissolve transition / fade transition / page turn transition / push transition / split transition / uncover transition / wipe transition / zoom in transition / zoom out transition -- The name of the page transition to use for all pages.
  • parent (r/o) reference -- The parent of the SWF export preference (a application).
  • properties record -- A property that allows setting of several properties at the same time.
  • raster compression JPEG compression / lossless compression / automatic compression -- The SWF raster compression options.
  • raster resolution integer -- The raster resolution. Can return: integer (72 - 2400).
  • rasterize pages boolean -- If true, all the pages in the exported document will be rasterized.
  • resampling threshold integer -- The resampling threshold.
  • resampling type none / downsample / subsample / bicubic downsample -- The resampling type.
  • SWF background transparent SWF background / paper color -- SWF background setting
  • text export policy live / rasterize / vectorize / tlf -- The text export policy.
  • use network access boolean -- Flag indicates whether to allow to use network access when export to SWF.
  • view SWF after exporting boolean -- Flag indicates to whether to view the SWF after exporting.
