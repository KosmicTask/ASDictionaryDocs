Class XML export preference

expand

Description

XML export preferences.

  • Defined in:
    • Preferences Suite

Properties

  • allow transform boolean -- If true, transforms the XML using an XSLT file.
  • character references boolean -- If true, replaces special characters with character references.
  • class (r/o) type class -- The class descriptor type.
  • copy formatted images boolean -- If true, copies formatted images to the images subfolder.
  • copy optimized images boolean -- If true, copies optimized images to the images subfolder.
  • copy original images boolean -- If true, copies original images to the images subfolder.
  • exclude dtd boolean -- If true, excludes the DTD from the exported XML content.
  • export from selected boolean -- If true, exports XML content from the selected XML element. If false, exports the entire document.
  • export untagged tables format none / CALS -- The export format for untagged tables in tagged stories.
  • file encoding UTF8 / UTF16 / shift JIS -- The file encoding type for exporting XML content.
  • GIF options interlaced boolean -- If true, generates interlaced GIFs. Note: Not valid when image conversion is JPEG.
  • GIF options palette adaptive palette / macintosh palette / web palette / windows palette -- The color palette for GIF conversion. Note: Not valid when image conversion is JPEG.
  • image conversion automatic / JPEG / GIF -- The file format to use for converted images. Note: Valid only when copy optimized images and/or copy formatted images is true.
  • JPEG options format baseline encoding / progressive encoding -- The formatting method for converted JPEG images. Note: Not valid when image conversion is GIF.
  • JPEG options quality low / medium / high / maximum -- The quality of converted JPEG images. Note: Not valid when image conversion is GIF.
  • object reference (r/o) XML export preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the XML export preference (a application or document). Can return: application or document.
  • preferred browser anything -- The preferred browser for viewing XML. Can return: alias or string or nothing.
  • properties record -- A property that allows setting of several properties at the same time.
  • ruby boolean -- If true, includes Ruby text in the exported XML content.
  • transform filename anything -- The name of the XSLT file. Note: Valid when allow transform is true. Can return: alias or string or stylesheet in XML.
  • view after export boolean -- If true, displays exported XML content in a specified viewer.
