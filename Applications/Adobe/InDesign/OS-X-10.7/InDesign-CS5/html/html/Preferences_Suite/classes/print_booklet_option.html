Class print booklet option

expand

Description

Print booklet options.

  • Defined in:
    • Preferences Suite

Properties

  • auto adjust margins boolean -- If true, automatically adjust margins to fit the specified printer's marks and bleed area.
  • bleed between pages number | string -- The amount of bleed between pages. Can return: number or string (0 - 144 points).
  • booklet type two up saddle stitch / two up perfect bound / two up consecutive / three up consecutive / four up consecutive -- The type of booklet.
  • bottom margin number | string -- Bottom margin of the printed booklet. Can return: number or string (0 - 288 points).
  • class (r/o) type class -- The class descriptor type.
  • creep number | string -- The amount of creep (binding adjustment based on paper thickness) to add. Can return: number or string (-144 - 144 points).
  • left margin number | string -- Left margin of the printed booklet. Can return: number or string (0 - 288 points).
  • margins uniform size boolean -- If true, make all margins equal to the top margin.
  • object reference (r/o) print booklet option -- An object reference for the object.
  • page range anything -- The pages to print, specified either as an enumeration or a string. To specify a range, separate page numbers in the string with a hyphen (-). To specify separate pages, separate page numbers in the string with a comma (,). Can return: all pages/select...
  • parent (r/o) document -- The parent of the print booklet option (a document).
  • print blank printer spreads boolean -- If true, print blank spreads.
  • properties record -- A property that allows setting of several properties at the same time.
  • right margin number | string -- Right margin of the printed booklet. Can return: number or string (0 - 288 points).
  • signature size signature size 4 / signature size 8 / signature size 12 / signature size 16 / signature size 32 -- The signature size of booklet (for perfect binding).
  • space between pages number | string -- The amount of space between pages. Can return: number or string (0 - 288 points).
  • top margin number | string -- Top margin of the printed booklet. Can return: number or string (0 - 288 points).
