Class find change text option

expand

Description

Find/change text options.

  • Defined in:
    • Preferences Suite

Properties

  • case sensitive boolean -- If true, finds strings whose use of case matches the find text string. If false, finds strings that match the find text string regardless of case.
  • class (r/o) type class -- The class descriptor type.
  • ignore diacritics boolean -- If true, ignore diacs in the find/change query.
  • ignore kashidas boolean -- If true, ignore kashidas in the find/change query.
  • include footnotes boolean -- If true, includes footnotes in the find/change query.
  • include hidden layers boolean -- If true, includes hidden layers in the find/change query.
  • include locked layers for find boolean -- If true, includes locked layers in the find query.
  • include locked stories for find boolean -- If true, includes locked stories in the find query.
  • include master pages boolean -- If true, includes master pages in the find/change query.
  • kana sensitive boolean -- If true, finds only text that matches the specified kana type.
  • object reference (r/o) find change text option -- An object reference for the object.
  • parent (r/o) reference -- The parent of the find change text option (a application).
  • properties record -- A property that allows setting of several properties at the same time.
  • whole word boolean -- If true, finds only the complete find text string. If false, also finds strings that contain the find text string.
  • width sensitive boolean -- If true, finds only text that matches the specified character width.
