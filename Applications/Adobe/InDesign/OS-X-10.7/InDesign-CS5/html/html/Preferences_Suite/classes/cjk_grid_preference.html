Class cjk grid preference

expand

Description

Character grid preferences.

  • Defined in:
    • Preferences Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • color every nth cell small integer -- Applies the grid color to every nth cell, where n is the value of this property.
  • ICF mode boolean -- If true, uses ICF mode for grid cells. If false, uses virtual body mode.
  • layout grid color index anything -- The layout grid color, specified either as an array of three doubles, each in the range 0 to 255 and representing R, G, and B values, or as a UI color. Can return: list of 3 fixed (0 - 255) or light blue/red/green/blue/yellow/magenta/cyan/gray/black/or...
  • minimum scale fixed -- The view magnification (as a percentage) less than which grids do not appear. (Range: 5 to 4000)
  • object reference (r/o) cjk grid preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the cjk grid preference (a application or document). Can return: application or document.
  • properties record -- A property that allows setting of several properties at the same time.
  • show all frame grids boolean -- If true, displays the frame (story) grids.
  • show all layout grids boolean -- If true, displays the layout grids.
  • show character count boolean -- If true, displays the character count for the frame.
  • single line color mode boolean -- If true, applies the grid color from the the edge of the line. If false, applies the grid color from the corner of the frame.
  • snap to layout grid boolean -- If true, objects snap to the layout grid.
  • use circular cells boolean -- If true, cell shape is circular. If false, cell shape is rectangular.
