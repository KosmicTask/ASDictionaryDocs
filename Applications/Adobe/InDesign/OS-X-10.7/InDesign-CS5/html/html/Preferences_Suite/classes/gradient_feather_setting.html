Class gradient feather setting

expand

Description

Gradient feather effect settings.

  • Defined in:
    • Preferences Suite

Properties

  • angle fixed -- The angle of the gradient feather. Can return: fixed (-180 - 180).
  • applied boolean -- If true, the gradient feather effect is applied.
  • class (r/o) type class -- The class descriptor type.
  • gradient start list of number | list of string -- The center point (for a radial gradient) or starting point (for a linear gradient) applied to the fill, as page coordinates in the format [x, y].
  • hilite angle fixed -- The hilite angle of the radial gradient feather.
  • hilite length number | string -- The hilite length of the radial gradient feather.
  • length number | string -- The length of the axial gradient, or radius of the radial gradient.
  • object reference (r/o) gradient feather setting -- An object reference for the object.
  • parent (r/o) reference -- The parent of the gradient feather setting (a transparency setting, stroke transparency setting, fill transparency setting or content transparency setting). Can return: transparency setting, stroke transparency setting, fill transparency setting or con...
  • properties record -- A property that allows setting of several properties at the same time.
  • type linear / radial -- The type of gradient.
