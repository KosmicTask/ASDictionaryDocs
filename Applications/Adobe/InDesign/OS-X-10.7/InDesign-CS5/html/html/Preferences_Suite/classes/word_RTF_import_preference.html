Class word RTF import preference

expand

Description

Word RTF import preferences.

  • Defined in:
    • Preferences Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • convert bullets and numbers to text boolean -- If true, bullets and numbers will be converted to embedded characters during import. If false, bullets and numbers will be rendered by InDesign.
  • convert page breaks none / page break / column break -- The option for handling manual page breaks.
  • convert tables to Unformatted Table / Unformatted Tabbed text -- The policy for converting tables whose formatting has been removed. Note: Valid when remove formatting is true.
  • import endnotes boolean -- If true, imports endnotes.
  • import footnotes boolean -- If true, imports footnotes.
  • import index boolean -- If true, imports the index.
  • import TOC boolean -- If true, imports the table of contents.
  • import unused styles boolean -- If true, imports unused styles.
  • object reference (r/o) word RTF import preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the word RTF import preference (a application).
  • preserve graphics boolean -- If true, preserves inline graphics.
  • preserve local overrides boolean -- If true, maintains character formatting in text whose formatting has been removed. Note: Valid when remove formatting is true.
  • preserve track changes boolean -- If true, preserves comments and edits in the imported file.
  • properties record -- A property that allows setting of several properties at the same time.
  • remove formatting boolean -- If true, removes text and table formatting.
  • resolve character style clash resolve clash use existing / resolve clash auto rename / resolve clash use new -- The option for handling style name conflicts.
  • resolve paragraph style clash resolve clash use existing / resolve clash auto rename / resolve clash use new -- The option for resolving conflicts that arise when paragraph styles have matching names.
  • use typographers quotes boolean -- If true, convert straight quotes and apostrophes in the imported text to typographic quotation marks and apostrophes.
