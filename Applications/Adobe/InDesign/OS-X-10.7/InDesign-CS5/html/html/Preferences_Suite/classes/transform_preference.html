Class transform preference

expand

Description

Transform preferences.

  • Defined in:
    • Preferences Suite

Properties

  • adjust stroke weight when scaling boolean -- If true, strokes are scaled when objects are scaled.
  • class (r/o) type class -- The class descriptor type.
  • dimensions include stroke weight boolean -- If true, includes the stroke weight when displaying object dimensions. If false, measures objects from the path or frame.
  • object reference (r/o) transform preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the transform preference (a application).
  • properties record -- A property that allows setting of several properties at the same time.
  • show content offset boolean -- If true, measures the x and y values of the object relative to the containing frame. If false, measures the x and y values relative to the rulers.
  • transformations are totals boolean -- If true, transformation values are relative to the parent object. If false, the transformation values are absolute values.
  • when scaling apply to content / adjust scaling percentage -- The method used to scale a page item.
