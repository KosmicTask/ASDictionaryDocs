Class find change feather setting

expand

Description

Feather effect settings.

  • Defined in:
    • Preferences Suite
  • Inherits from:
    • feather setting (Preferences Suite)

Properties

  • choke amount anything -- The amount to choke the feather (as a percentage of the feather width). (Range: 0 to 100). Can return: fixed (0 - 100) or nothing.
  • corner type anything -- The corner effect applied to the feather. Can return: sharp/rounded/diffusion or nothing.
  • mode anything -- The feather mode. Can return: none/standard or nothing.
  • noise anything -- The amount (as a percentage) of noise applied to the feather. (Range: 0 to 100). Can return: fixed (0 - 100) or nothing.
  • object reference (r/o) find change feather setting -- An object reference for the object.
  • parent (r/o) reference -- The parent of the find change feather setting (a find change transparency setting, find change stroke transparency setting, find change fill transparency setting or find change content transparency setting). Can return: find change transparency setting...
  • width anything -- The feather width. (Range depends on the unit type. For points: 0 to 1000; for picas: 0 to 83p4; for inches: 0 to 13.8889; for mm: 0 to 352.778; for cm: 0 to 35.277; for ciceros: 0 to 78c2.389.). Can return: number or string (0 - 1000 points) or nothing.
