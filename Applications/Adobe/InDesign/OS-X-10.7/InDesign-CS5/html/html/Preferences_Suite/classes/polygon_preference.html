Class polygon preference

expand

Description

Default settings to use when creating a polygon.

  • Defined in:
    • Preferences Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • inset percentage integer -- The star inset percentage for the sides of a polygon. (Range: 0 to 100). Can return: integer (0 - 100).
  • number of sides integer -- The number of sides for a polygon. (Range: 3 to 100). Can return: integer (3 - 100).
  • object reference (r/o) polygon preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the polygon preference (a application).
  • properties record -- A property that allows setting of several properties at the same time.
