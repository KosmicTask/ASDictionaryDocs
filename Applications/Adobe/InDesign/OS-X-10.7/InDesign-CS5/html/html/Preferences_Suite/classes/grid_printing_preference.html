Class grid printing preference

expand

Description

Grid printing and exporting preferences.

  • Defined in:
    • Preferences Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • frame grid printing boolean -- If true, displays frame (story) grids in the printed or exported document.
  • frame grid stroke weight fixed -- The stroke weight (in points) of the frame grid.
  • layout grid printing boolean -- If true, displays layout grids in the printed or exported document.
  • layout grid stroke weight fixed -- The stroke weight (in points) of the layout grid.
  • object reference (r/o) grid printing preference -- An object reference for the object.
  • page item printing boolean -- If true, displays page items other than text, frame grids, and layout grids in the printed or exported document. For information on printing and exporting text and grids, see text printing, frame grid printing, and layout grid printing.
  • parent (r/o) reference -- The parent of the grid printing preference (a application).
  • properties record -- A property that allows setting of several properties at the same time.
  • text printing boolean -- If true, displays text in the printed or exported document.
