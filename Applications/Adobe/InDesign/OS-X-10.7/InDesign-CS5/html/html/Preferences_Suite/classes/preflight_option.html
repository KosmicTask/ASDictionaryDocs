Class preflight option

expand

Description

Preflight options.

  • Defined in:
    • Preferences Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • object reference (r/o) preflight option -- An object reference for the object.
  • parent (r/o) reference -- The parent of the preflight option (a application or document). Can return: application or document.
  • preflight embed working profile boolean -- If true, embed working profile when creating new document.
  • preflight include nonprinting objects boolean -- If true, include objects that do not print when preflighting.
  • preflight include objects on pasteboard boolean -- If true, include objects on pasteboard when preflighting.
  • preflight off boolean -- If true, preflight is turned off for all documents or for this document.
  • preflight profile policy use embedded profile / use working profile -- The policy for preflighting applied when opening a document or book, whether to use embedded profile or the another profile when the preflight is turned on.
  • preflight scope anything -- The pages or documents to preflight, specified either as an enumeration or a string. To specify a range, separate page numbers in the string with a hyphen (-). To specify separate pages, separate page numbers in the string with a comma (,). Can return:...
  • preflight which layers preflight all layers / preflight visible layers / preflight visible printable layers -- Which layers to preflight.
  • preflight Working Profile anything -- The working preflight profile. Can return: preflight profile or string.
  • properties record -- A property that allows setting of several properties at the same time.
