Class data merge preference

expand

Description

Data merge preferences.

  • Defined in:
    • Preferences Suite

Properties

  • arrange by rows first / columns first -- The order in which to arrange multiple records on the page in the target document.
  • bottom margin number | string -- The offset value of the bottom margin in the target document.
  • class (r/o) type class -- The class descriptor type.
  • column spacing number | string -- The amount of space between columns of records in the target document.
  • left margin number | string -- The offset value of the left margin in the target document.
  • object reference (r/o) data merge preference -- An object reference for the object.
  • parent (r/o) data merge -- The parent of the data merge preference (a data merge).
  • properties record -- A property that allows setting of several properties at the same time.
  • record number integer -- The number of the record to merge. Note: Valid only when record selection is one record.
  • record range string -- The range of records to merge. Note: Valid only when record selection is range.
  • record selection all records / one record / range -- The records to merge.
  • records per page single record / multiple record -- The number of records to place on each page in the document.
  • right margin number | string -- The offset value of the right margin in the target document.
  • row spacing number | string -- The amount of space between rows of records in the target document.
  • top margin number | string -- The offset value of the top margin in the target document.
