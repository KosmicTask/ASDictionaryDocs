Command create container item

Description

Creates an empty container.

  • Defined in:
    • Preferences Suite

Parameters

  • metadata preference
  • namespace string -- The namespace of the container.
  • path string -- The path to the container.
  • [index integer] -- The index of the item within the container. Specified values must be 1 or greater. To append the item to the end of the index and allow the next available value to be assigned, use 0. Can accept: integer (Default: 0).
  • [container bag / seq / alt] -- The container type. Note: Required when the new item is the first item added to the container. Can accept: bag/seq/alt (Default: bag).
