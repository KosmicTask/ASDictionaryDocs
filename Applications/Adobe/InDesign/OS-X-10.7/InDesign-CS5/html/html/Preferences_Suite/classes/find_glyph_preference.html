Class find glyph preference

expand

Description

Find glyph preferences.

  • Defined in:
    • Preferences Suite

Properties

  • applied font anything -- The font applied to the find glyph preference, specified as either a font object or the name of font family. Can return: font, string or nothing.
  • class (r/o) type class -- The class descriptor type.
  • font style anything -- The name of the font style. Can return: string or nothing.
  • glyph ID anything -- The GID/CID of the glyph. Can return: integer or nothing.
  • object reference (r/o) find glyph preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the find glyph preference (a application).
  • properties record -- A property that allows setting of several properties at the same time.
  • ros font group anything -- The Registry Ordered font group. Can return: list of 2 strings or nothing.
