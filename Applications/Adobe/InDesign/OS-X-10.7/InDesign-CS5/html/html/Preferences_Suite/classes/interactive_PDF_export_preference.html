Class interactive PDF export preference

expand

Description

Interactive PDF export settings for the application object.

  • Defined in:
    • Preferences Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • export layers boolean -- If true, saves each layer as an Acrobat layer within the PDF document.
  • export reader spreads boolean -- If true, each spread in the exported document is combined into a single page that has spread's original width.
  • flip pages boolean -- Automatically flip pages in the exported PDF.
  • flip pages speed integer -- The speed that the pages flip. Can return: integer (0 - 32767).
  • generate thumbnails boolean -- If true, generates thumbnail images for each page or spread.
  • include structure boolean -- If true, creates a tagged PDF file. Note: If acrobat compatibility is acrobat 6 or higher, tags are visible only when the PDF is opened in Acrobat 6 or higher.
  • interactive PDF interactive elements option include all media / appearance only -- How to draw interactive elements.
  • object reference (r/o) interactive PDF export preference -- An object reference for the object.
  • open in full screen boolean -- Open PDF in full screen mode.
  • page range anything -- The pages to print, specified either as an enumeration or a string. To specify a range, separate page numbers in the string with a hyphen (-). To specify separate pages, separate page numbers in the string with a comma (,). Can return: all pages/select...
  • page transition override from document / none / blinds transition / box transition / comb transition / cover transition / dissolve transition / fade transition / page turn transition / push transition / split transition / uncover transition / wipe transition / zoom in transition / zoom out transition -- The name of the page transition to use for all pages.
  • parent (r/o) reference -- The parent of the interactive PDF export preference (a application).
  • PDF JPEG quality minimum / low / medium / high / maximum -- The PDF JPEG quality options.
  • pdf magnification default / actual size / fit page / fit width / fit height / fit visible / twenty five percent / fifty percent / seventy five percent / one hundred percent -- The export PDF magnification options.
  • pdf page layout default / single page / single page continuous / two up facing / two up facing continuous / two up cover page / two up cover page continuous -- The export PDF page layout.
  • PDF raster compression JPEG compression / lossless compression / automatic compression -- The PDF raster compression options.
  • properties record -- A property that allows setting of several properties at the same time.
  • raster resolution anything -- The raster resolution. Can return: seventy two ppi/ninety six ppi/one hundred forty four ppi or integer (72 - 300).
  • view PDF boolean -- If true, automatically opens the PDF file after exporting.
