Class layout grid data information

expand

Description

Default properties specific to layout grids.

  • Defined in:
    • Preferences Suite

Properties

  • applied font anything -- The font applied to the layout grid data information, specified as either a font object or the name of font family. Can return: font or string.
  • character aki fixed -- The amount of white space between characters.
  • class (r/o) type class -- The class descriptor type.
  • font style string -- The name of the font style.
  • horizontal scale fixed -- The horizontal scaling applied to the layout grid data information.
  • line aki fixed -- The amount of white space between lines.
  • object reference (r/o) layout grid data information -- An object reference for the object.
  • parent (r/o) reference -- The parent of the layout grid data information (a application or document). Can return: application or document.
  • point size number | string -- The text size.
  • properties record -- A property that allows setting of several properties at the same time.
  • vertical scale fixed -- The vertical scaling applied to the layout grid data information.
