Class graphic layer

expand

Description

A layer in a PSD image or PDF file.

  • Plural name:
    • graphic layers
  • Defined in:
    • Preferences Suite

Properties

  • adjustment layer (r/o) boolean -- If true, the layer is an adjustment layer. Note: Must occur in the script before overriding the visibility state of the layer with a current visibility statement.
  • class (r/o) type class -- The class descriptor type.
  • current visibility boolean -- If true, the layer is visible in the document.
  • export state (r/o) boolean -- If true, the export state is on.
  • FX layer (r/o) boolean -- If true, layer effects have been applied to the layer. Note: Must occur in the script before overriding the visibility state of the layer with a current visibility statement.
  • has export state (r/o) boolean -- If true, the layer has an export state.
  • has print state (r/o) boolean -- If true, the layer has a print state.
  • has view state (r/o) boolean -- If true, the layer has a view state.
  • id (r/o) integer -- The unique ID of the graphic layer.
  • index (r/o) integer -- The index of the graphic layer within its containing object.
  • locked (r/o) boolean -- If true, the graphic layer is locked.
  • name (r/o) string -- The name of the graphic layer.
  • object reference (r/o) graphic layer -- An object reference for the object.
  • original visibility (r/o) boolean -- Returns the visibility setting set for the layer before the graphic file was imported. Note: Must occur in the script before overriding the visibility state with a current visibility statement.
  • parent (r/o) reference -- The parent of the graphic layer (a graphic layer option or graphic layer). Can return: graphic layer option or graphic layer.
  • print state (r/o) boolean -- If true, the print state is on.
  • properties record -- A property that allows setting of several properties at the same time.
  • section divider layer (r/o) boolean -- If true, the layer is a section divider layer.
  • separator layer (r/o) boolean -- If true, the layer is a separator layer.
  • view state (r/o) boolean -- If true, the view state is on.
