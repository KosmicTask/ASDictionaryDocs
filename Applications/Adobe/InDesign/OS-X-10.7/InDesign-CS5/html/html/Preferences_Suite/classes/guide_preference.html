Class guide preference

expand

Description

Guide preferences.

  • Defined in:
    • Preferences Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • guides in back boolean -- If true, places guides behind all other objects on the spread.
  • guides locked boolean -- If true, guides cannot be moved, added, or deleted.
  • guides shown boolean -- If true, displays the guides.
  • guides snapto boolean -- If true, an object within the specified range snaps to the nearest guide when the object is created, moved, or resized. For range information, see guide snapto zone.
  • object reference (r/o) guide preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the guide preference (a application or document). Can return: application or document.
  • properties record -- A property that allows setting of several properties at the same time.
  • ruler guides color anything -- The color of the guide, specified either as an array of three doubles, each in the range 0 to 255 and representing R, G, and B values, or as a UI color. . Can return: list of 3 fixed (0 - 255) or light blue/red/green/blue/yellow/magenta/cyan/gray/black...
  • ruler guides view threshold fixed -- The magnification (as a percentage) less than which ruler guides do not appear. (Range: 5 to 4000)

Elements

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id
