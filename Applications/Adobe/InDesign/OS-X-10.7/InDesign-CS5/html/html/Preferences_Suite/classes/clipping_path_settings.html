Class clipping path settings

expand

Description

Clipping path settings.

  • Defined in:
    • Preferences Suite

Properties

  • alpha channel path names (r/o) list of string -- A list of the alpha channels stored in the graphic.
  • applied path name string -- The name of the Photoshop path or alpha channel to use as a clipping path.
  • class (r/o) type class -- The class descriptor type.
  • clipping type none / detect edges / alpha channel / photoshop path / user modified path -- The clipping path type.
  • include inside edges boolean -- If true, creates interior clipping paths within the surrounding clipping path. Note: Valid only when clipping type is alpha channel or detect edges.
  • inset frame number | string -- Shrinks the area enclosed by the clipping path by the specified amount. (Range depends on the unit. For points: -10000 to 10000; picas: -833p4 to 833p4; inches: -138.8889 to 138.8889; mm: -3527.778 to 3527.778; cm: -352.7778 to 352.7778; ciceros: -781c...
  • invert path boolean -- If true, inverts the clipping path.
  • object reference (r/o) clipping path settings -- An object reference for the object.
  • parent (r/o) reference -- The parent of the clipping path settings (a image, EPS, WMF, PICT, PDF or imported page). Can return: image, EPS, WMF, PICT, PDF or imported page.
  • photoshop path names (r/o) list of string -- A list of the clipping paths stored in the graphic.
  • properties record -- A property that allows setting of several properties at the same time.
  • restrict to frame boolean -- If true, truncates the clipping path at the edge of the frame containing the graphic. Note: Valid only when clipping type is alpha channel or detect edges.
  • threshold integer -- The lowest value (darkest) pixel to allow in the image. All pixels in the image whose values are greater than (lighter than) the threshold value are clipped (obscured). (Range: 0 to 255) Note: Valid only when clipping type is detect edges or alpha chan...
  • tolerance fixed -- Specifies how similar a pixel's intensity value can be to the threshold value before the pixel is obscured by the clipping path. (Range: 0 to 10) Note: Valid only when clipping type is detect edges or alpha channel. Can return: fixed (0 - 10).
  • use high resolution image boolean -- If true, uses the high-resolution version of the graphic to create the clipping path. If false, calculates the clipping path based on screen-display resolution. Note: Valid only when clipping type is detect edges.
