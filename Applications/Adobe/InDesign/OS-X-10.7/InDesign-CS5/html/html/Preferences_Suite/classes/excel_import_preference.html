Class excel import preference

expand

Description

Excel import preferences.

  • Defined in:
    • Preferences Suite

Properties

  • alignment style spreadsheet / left align / right align / center align -- The cell alignment for the imported document.
  • class (r/o) type class -- The class descriptor type.
  • decimal places integer -- The number of decimal places to include. Note: Valid only when alignment style is decimal.
  • error code integer -- The import error code. (Key: 0=Success; 1=Empty Sheet; 2=Invalid sheet; 3=Invalid range; 4=Invalid View; 5=Misc. Error)
  • object reference (r/o) excel import preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the excel import preference (a application).
  • preserve graphics boolean -- If true, preserves inline graphics.
  • properties record -- A property that allows setting of several properties at the same time.
  • range name string -- The range of cells to import. Note: Use a colon (:) to separate the start and end cell names in the range.
  • sheet index integer -- The worksheet's index. For information, see sheet name.
  • sheet name string -- The worksheet to import.
  • show hidden cells boolean -- If true, shows hidden cells.
  • table formatting excel formatted table / excel unformatted table / excel unformatted tabbed text / excel format only once -- The format for imported spreadsheets.
  • use typographers quotes boolean -- If true, convert straight quotes and apostrophes in the imported text to typographic quotation marks and apostrophes.
  • view name string -- The stored custom or personal view(s) to import with the file.

contains events , event listeners

  • event listener -- by index, name, range, relative, test, id
