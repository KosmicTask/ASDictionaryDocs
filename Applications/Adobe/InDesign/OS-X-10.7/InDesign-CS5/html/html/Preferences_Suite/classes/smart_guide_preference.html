Class smart guide preference

expand

Description

Smart Guide preferences.

  • Defined in:
    • Preferences Suite

Properties

  • align to object center boolean -- If true, smart alignment to object centers is enabled.
  • align to object edges boolean -- If true, smart alignment to object edges is enabled.
  • class (r/o) type class -- The class descriptor type.
  • enabled boolean -- If true, smart guides are enabled.
  • guide color anything -- The color of the guide, specified either as an array of three doubles, each in the range 0 to 255 and representing R, G, and B values, or as a UI color. Can return: list of 3 fixed (0 - 255) or light blue/red/green/blue/yellow/magenta/cyan/gray/black/o...
  • object reference (r/o) smart guide preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the smart guide preference (a application).
  • properties record -- A property that allows setting of several properties at the same time.
  • smart dimensions boolean -- If true, smart dimensions guides are enabled.
  • smart spacing boolean -- If true, smart spacing guides are enabled.
