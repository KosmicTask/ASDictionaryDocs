Command append

Description

Uses metadata from the specified external file to define any undefined metadata properties in the document.

  • Defined in:
    • Preferences Suite

Parameters

  • metadata preference
  • from alias | string -- The path to the external file that contains the metadata.
  • [affect all boolean] -- If true, also replaces existing metadata with data from the external file. If false, does not replace existing metadata. Note: Defaults to false. Can accept: boolean (Default: FALSE).
