Class story grid data information

expand

Description

Default properties specific to frame grids.

  • Defined in:
    • Preferences Suite

Properties

  • applied font anything -- The font applied to the story grid data information, specified as either a font object or the name of font family. Can return: font or string.
  • character aki fixed -- The amount of white space between characters.
  • character alignment align baseline / align em top / align em center / align em bottom / align ICF top / align ICF bottom -- The alignment of small characters to the largest character in the line.
  • character count location none / top align / left align / bottom align / right align -- The character count location. Note: Valid when show character count is true.
  • character count size fixed -- The character size for the character count display.
  • class (r/o) type class -- The class descriptor type.
  • font style string -- The name of the font style.
  • grid alignment none / align baseline / align em top / align em center / align em bottom / align ICF top / align ICF bottom -- The alignment to the frame grid or baseline grid.
  • grid view grid view enum / zn view enum / align view enum / grid and zn view enum -- The grid view setting.
  • horizontal scale fixed -- The horizontal scaling applied to the story grid data information.
  • line aki fixed -- The amount of white space between lines.
  • line alignment left or top line align / center line align / right or bottom line align / left or top line justify / center line justify / right or bottom line justify / full line justify -- The line justification.
  • object reference (r/o) story grid data information -- An object reference for the object.
  • parent (r/o) reference -- The parent of the story grid data information (a application or document). Can return: application or document.
  • point size number | string -- The text size.
  • properties record -- A property that allows setting of several properties at the same time.
  • vertical scale fixed -- The vertical scaling applied to the story grid data information.

Elements

contains events , event listeners

  • event listener -- by index, name, range, relative, test, id
