Class blending setting

Description

Basic object blending settings.

  • Defined in:
    • Preferences Suite

Properties

  • blend mode normal / multiply / screen / overlay / soft light / hard light / color dodge / color burn / darken / lighten / difference / exclusion / hue / saturation / color / luminosity -- The blending mode for controlling how the base color interacts with the color of the blending setting.
  • class (r/o) type class -- The class descriptor type.
  • isolate blending boolean -- If true, blending is applied only to the group. If false, blending includes all objects beneath the group.
  • knockout group boolean -- If true, the blending setting is a knockout group.
  • object reference (r/o) blending setting -- An object reference for the object.
  • opacity fixed -- The fill opacity of the blending setting (as a percentage). (Range: 0 to 100). Can return: fixed (0 - 100).
