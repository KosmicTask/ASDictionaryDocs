Class transparency preference

expand

Description

Transparency preferences.

  • Defined in:
    • Preferences Suite

Properties

  • blending space default / RGB / CMYK -- The color space used for blending the colors of transparent objects.
  • class (r/o) type class -- The class descriptor type.
  • global light altitude fixed -- The altitude of the global light. (Range: 0 to 90). Can return: fixed (0 - 100).
  • global light angle fixed -- The angle of the global light. (Range: -360 to 360). Can return: fixed (-180 - 180).
  • object reference (r/o) transparency preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the transparency preference (a application or document). Can return: application or document.
  • properties record -- A property that allows setting of several properties at the same time.

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id
