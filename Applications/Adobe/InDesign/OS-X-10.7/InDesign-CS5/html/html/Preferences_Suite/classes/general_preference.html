Class general preference

expand

Description

General application preferences.

  • Defined in:
    • Preferences Suite

Properties

  • application bar shown boolean -- If true, application bar is shown.
  • auto collapse icon panels boolean -- If true, panel drawers close automatically.
  • auto show hidden panels boolean -- If true, vertical reveal strips appear when palette UI is hidden.
  • class (r/o) type class -- The class descriptor type.
  • complete font download glyph limit integer -- The threshold at which to trigger font subsetting based on the number of glyphs the font contains.
  • enable floating window docking boolean -- If true, floating windows can be docked by user as tabs.
  • enable multi touch gestures boolean -- Controls whether or not multi-touch gestures are enabled.
  • include preview boolean -- If true, includes a preview in saved documents.
  • object reference (r/o) general preference -- An object reference for the object.
  • objects move with page boolean -- Controls whether page items move when a page is repositioned from the UI. The option/alt key temporarily reverses this property
  • open documents as tabs boolean -- If true, documents open as tabs.
  • open recent length integer -- Number of items to show in the Open Recent menu list. Range: 0 to 30. Can return: integer (0 - 30).
  • page numbering section / absolute -- The page numbering method.
  • parent (r/o) reference -- The parent of the general preference (a application).
  • place cursor uses thumbnails boolean -- Controls whether or not to show thumbnails of imported files in the Place icon.
  • prevent selecting locked items boolean -- Controls whether or not you can select and interact with a locked item. When this is off, only position is locked.
  • preview size small / medium / large / extra large -- The preview size. Note: Valid when include preview is true.
  • properties record -- A property that allows setting of several properties at the same time.
  • show content grabber boolean -- Controls whether or not the content grabber adornment is shown.
  • show live corners boolean -- Controls whether or not the live corners grabber adornment is shown.
  • show master page overlay boolean -- Controls whether or not to show the master page overlay when a page is selected using the Page Tool.
  • show transformation values boolean -- Controls whether or not to dynamically display transformation information as part of the cursor while manipulating page items.
  • temporary folder alias | string -- The location in which to store temporary files.
  • tool tips normal / none / fast -- Tool tip behavior.
  • tools panel single column / double column / single row -- Controls the appearance of the Tools panel.
  • use application frame boolean -- If true, application lives in a frame.
