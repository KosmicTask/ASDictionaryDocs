Description

Placed InDesign page attribute.

  • Defined in:

Properties

  • class (r/o) type class -- The class descriptor type.
  • imported page crop crop content / crop bleed / crop slug -- Specifies the cropping of the imported InDesign page. Read only for page items.
  • object reference (r/o) imported page attribute -- An object reference for the object.
  • page number integer -- Which page of the InDesign document should be imported. Read only for page items.
  • parent (r/o) reference -- The parent of the imported page attribute (a application).
  • properties record -- A property that allows setting of several properties at the same time.
