Class dictionary preference

expand

Description

User dictionary preferences.

  • Defined in:
    • Preferences Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • composition use user dictionary / use document / both -- The hyphenation exception list to use when composing text.
  • merge user dictionary boolean -- If true, merges the spelling and hyphenation exceptions lists in the external user dictionary with the lists stored within the document.
  • object reference (r/o) dictionary preference -- An object reference for the object.
  • parent (r/o) reference -- The parent of the dictionary preference (a application or document). Can return: application or document.
  • properties record -- A property that allows setting of several properties at the same time.
  • recompose when changed boolean -- If true, recomposes all stories when the compose using settings are changed, or when words are added to or removed from the user dictionary.

Elements

contains events , event listeners

  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id
