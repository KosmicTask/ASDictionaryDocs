Class paragraph

expand

Description

A paragraph.

  • Plural name:
    • paragraphs
  • Defined in:
    • Text Suite
  • Inherits from:
    • text (Text Suite)

Properties

  • all graphics (r/o) list of graphic -- Lists all graphics contained by the paragraph.
  • all page items (r/o) list of page item -- Lists all page items contained by the paragraph.
  • applied conditions list of condition -- The applied conditions.
  • applied font anything -- The font applied to the paragraph, specified as either a font object or the name of font family. Can return: font or string.
  • applied nested styles (r/o) list of character style -- The applied character styles dictated by nested styles.
  • associated XML elements (r/o) list of XML item -- The XML elements associated with the paragraph.
  • bullets and numbering result text (r/o) string -- The current bullets or numbering string.
  • fill color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the fill of the paragraph. . Can also accept: string.
  • fill tint fixed -- The tint (as a percentage) of the fill color of the paragraph. (To specify a tint percentage, use a number in the range of 0 to 100; to use the inherited or overridden value, use -1.)
  • gradient fill start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the fill of the text, in the format [x, y].
  • gradient stroke start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the stroke of the text, in the format [x, y].
  • horizontal scale fixed -- The horizontal scaling applied to the paragraph.
  • numbering result number (r/o) integer -- The current level number value.
  • object reference (r/o) paragraph -- An object reference for the object.
  • OpenType features list of anything -- OpenType features. Can return: list of list of 2 strings or integers.
  • parent (r/o) reference -- The parent of the paragraph (a xml story, text path, text frame, text, insertion point, text style range, paragraph, text column, line, word, character, story, cell, XML element, footnote, note, change or hidden text). Can return: xml story, text path,...
  • parent text frames (r/o) list of reference -- An array of the text frames that contain the text. Can return: list of text frames or text paths.
  • skew fixed -- The skew angle of the paragraph.
  • stroke color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the stroke of the paragraph. Can also accept: string.
  • stroke tint fixed -- The tint (as a percentage) of the stroke color of the paragraph. (To specify a tint percentage, use a number in the range of 0 to 100; to use the inherited or overridden value, use -1.)
  • tab list list of anything -- A list of the tab stops in the paragraph. Can return: list of record.
  • vertical scale fixed -- The vertical scaling applied to the paragraph.
