Class change

expand

Description

A tracked change made to a story.

  • Plural name:
    • changes
  • Defined in:
    • Text Suite

Properties

  • change type (r/o) inserted text / deleted text / moved text -- The type of tracked change. Note: Valid only when track changes is true.
  • class (r/o) type class -- The class descriptor type.
  • date (r/o) date -- The date on which the tracked change was made. Note: Valid only when track changes is true.
  • index (r/o) integer -- The index of the change within its containing object.
  • object reference (r/o) change -- An object reference for the object.
  • parent (r/o) reference -- The parent of the change (a story, xml story or cell). Can return: story, xml story or cell.
  • properties record -- A property that allows setting of several properties at the same time.
  • story offset (r/o) insertion point -- The location of the first insertion point in the object (relative to the beginning of the story).
  • user name (r/o) string -- The user who made the change. Note: Valid only when track changes is true.
