Command add exception

Description

Adds the specified words to the hyphenation exceptions list.

  • Defined in:
    • Text Suite

Parameters

  • hyphenation exception
  • added exceptions string -- The list of words to add.
  • [removed list boolean] -- If true, adds the words to the removed exceptions list. If false or unspecified, adds the words to the added exceptions list. Can accept: boolean (Default: FALSE).
