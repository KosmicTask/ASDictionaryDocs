Command place XML

Description

Places XML content into the specified object. Note: Replaces any existing content.

  • Defined in:
    • Text Suite

Parameters

  • reference -- Supported by the following objects: text frame, story.
  • using XML element -- The XML element whose content you want to place.
  • [place point number | string] -- The point at which to place the object, specified in the format [x, y]. Only applicable for page and spread.
  • [autoflowing boolean] -- If true, autoflows placed text. Only applicable for page and spread. Can accept: boolean (Default: FALSE).
