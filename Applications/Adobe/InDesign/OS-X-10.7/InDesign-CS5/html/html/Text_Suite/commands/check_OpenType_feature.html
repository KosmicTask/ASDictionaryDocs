Command check OpenType feature

Description

Checks whether the font supports the specified OpenType feature.

  • Defined in:
    • Text Suite

Parameters

  • font
  • using anything -- The OpenType feature to check for, specified either as an OpenType feature or a string. Can accept: low/discretionary ligatures feature/fractions feature/ordinal feature/swash feature/titling feature/contextual alternates feature/all small caps feature...

Result

  • boolean -- If true, the font supports the specified feature.
