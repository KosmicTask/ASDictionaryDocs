Class hidden text

expand

Description

A hidden text object.

  • Plural name:
    • hidden texts
  • Defined in:
    • Text Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • id (r/o) integer -- The unique ID of the hidden text.
  • index (r/o) integer -- The index of the hidden text within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the hidden text; this is an alias to the hidden text's label property.
  • object reference (r/o) hidden text -- An object reference for the object.
  • parent (r/o) reference -- The parent of the hidden text (a story, xml story, text frame, insertion point, note, cell or footnote). Can return: story, xml story, text frame, insertion point, note, cell or footnote.
  • properties record -- A property that allows setting of several properties at the same time.
  • story offset (r/o) insertion point -- The insertion point before the table in the story containing the table.
