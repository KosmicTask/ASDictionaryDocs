Class paragraph style group

expand

Description

A paragraph style group.

  • Plural name:
    • paragraph style groups
  • Defined in:
    • Text Suite

Properties

  • all paragraph styles (r/o) list of paragraph style -- Lists all paragraph styles (regardless of their group).
  • class (r/o) type class -- The class descriptor type.
  • id (r/o) integer -- The unique ID of the paragraph style group.
  • index (r/o) integer -- The index of the paragraph style group within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the paragraph style group.
  • object reference (r/o) paragraph style group -- An object reference for the object.
  • parent (r/o) reference -- The parent of the paragraph style group (a document, application or paragraph style group). Can return: document, application or paragraph style group.
  • properties record -- A property that allows setting of several properties at the same time.
