Class text frame

Description

A text frame.

  • Plural name:
    • text frames
  • Defined in:
    • Text Suite
  • Inherits from:
    • page item (Layout Suite)

Properties

  • absolute flip none / horizontal / vertical / horizontal and vertical / both -- Indicates whether the text frame has been flipped independently of its parent object and, if yes, the direction in which the text frame was flipped.
  • absolute horizontal scale fixed -- The horizontal scale of the text frame relative to its containing object.
  • absolute rotation angle fixed -- The rotation angle of the text frame relative to its containing object. (Range: -360 to 360)
  • absolute shear angle fixed -- The skewing angle of the text frame relative to its containing object. (Range: -360 to 360)
  • absolute vertical scale fixed -- The vertical scale of the text frame relative to its containing object.
  • all graphics (r/o) list of graphic -- Lists all graphics contained by the text frame.
  • all page items (r/o) list of page item -- Lists all page items contained by the text frame.
  • anchored object settings (r/o) anchored object setting -- Anchored object settings.
  • applied object style object style -- The object style applied to the text frame.
  • associated XML element (r/o) XML item -- The XML element associated with the text frame.
  • baseline frame grid options (r/o) baseline frame grid option -- Baseline frame grid option settings.
  • content transparency settings (r/o) content transparency setting -- Transparency settings for the content of the text frame.
  • content type unassigned / graphic type / text type -- The type of content that a frame can contain.
  • contents anything -- The contents of the text frame. Can return: string, placeholder text or auto page number/next page number/previous page number/section marker/bullet character/copyright symbol/degree symbol/ellipsis character/forced line break/paragraph symbol/register...
  • end join miter end join / round end join / bevel end join -- The corner join applied to the text frame.
  • end text frame (r/o) reference -- The last text frame in the thread. Can return: text frame or text path.
  • fill color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the fill of the text frame. . Can also accept: string.
  • fill tint fixed -- The percent of tint to use in the text frame's fill color. (To specify a tint percent, use a number in the range of 0 to 100; to use the inherited or overridden value, use -1.)
  • fill transparency settings (r/o) fill transparency setting -- Transparency settings for the fill applied to the text frame.
  • geometric bounds list of number | list of string -- The bounds of the text frame excluding the stroke width, in the format [y1, x1, y2, x2], which give the coordinates of the top-left and bottom-right corners of the bounding box.
  • gradient fill angle fixed -- The angle of a linear gradient applied to the fill of the text frame. (Range: -180 to 180)
  • gradient fill length number | string -- The length (for a linear gradient) or radius (for a radial gradient) applied to the fill of the text frame.
  • gradient fill start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the fill of the text frame, in the format [x, y].
  • gradient stroke angle fixed -- The angle of a linear gradient applied to the stroke of the text frame. (Range: -180 to 180)
  • gradient stroke length number | string -- The length (for a linear gradient) or radius (for a radial gradient) applied to the stroke of the text frame.
  • gradient stroke start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the stroke of the text frame, in the format [x, y].
  • grid data (r/o) grid data information -- Default grid properties. Note: Applies to named, layout, and frame (story) grids.
  • horizontal scale fixed -- The horizontal scaling applied to the text frame.
  • id (r/o) integer -- The unique ID of the text frame.
  • index (r/o) integer -- The index of the text frame within its containing object.
  • item layer layer -- The layer that the text frame is on.
  • local display setting high quality / typical / optimized / default -- Display performance options for the text frame.
  • locked boolean -- If true, the text frame is locked.
  • name string -- The name of the text frame.
  • next text frame reference -- The next text frame in the thread. Can return: text frame or text path. Can also accept: nothing.
  • nonprinting boolean -- If true, the text frame does not print.
  • object reference (r/o) text frame -- An object reference for the object.
  • overflows (r/o) boolean -- If true, the story has overset text.
  • overprint fill boolean -- If true, the text frame's fill color overprints any underlying objects. If false, the fill color knocks out the underlying colors.
  • overprint stroke boolean -- If true, the text frame's stroke color overprints any underlying objects. If false, the stroke color knocks out the underlying colors.
  • parent (r/o) reference -- The parent of the text frame (a spread, master spread, spline item, polygon, graphic line, rectangle, oval, group, state, character, place gun or snippet). Can return: spread, master spread, spline item, polygon, graphic line, rectangle, oval, group, s...
  • parent story (r/o) story -- The story that contains the text.
  • previous text frame reference -- The previous text frame in the thread. Can return: text frame or text path. Can also accept: nothing.
  • rotation angle fixed -- The rotatation angle of the text frame. (Range: -360 to 360)
  • shear angle fixed -- The skewing angle applied to the text frame. (Range: -360 to 360)
  • start text frame (r/o) reference -- The first text frame in the thread. Can return: text frame or text path.
  • stroke alignment center alignment / inside alignment / outside alignment -- The stroke alignment applied to the text frame.
  • stroke color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the stroke of the text frame. Can also accept: string.
  • stroke corner adjustment none / dashes / gaps / dashes and gaps -- The corner adjustment applied to the text frame.
  • stroke dash and gap list of number | list of string -- The dash and gap measurements that define the pattern of a custom dashed line. Define up to six values (in points) in the format [dash1, gap1, dash2, gap2, dash3, gap3].
  • stroke weight number | string -- The weight (in points) to apply to the text frame's stroke.
  • text frame index (r/o) integer -- The index of the text frame within the story.
  • text frame preferences (r/o) text frame preference -- Text frame preference settings.
  • vertical scale fixed -- The vertical scaling applied to the text frame.
  • visible boolean -- If true, the text frame is visible.
  • visible bounds list of number | list of string -- The bounds of the text frame including the stroke width, in the format [y1, x1, y2, x2], which give the coordinates of the top-left and bottom-right corners of the bounding box.
  • after place (r/o) string -- Dispatched after a text frame is placed. This event bubbles. This event is not cancelable.
  • before place (r/o) string -- Dispatched before a text frame is placed. This event bubbles. This event is cancelable.
