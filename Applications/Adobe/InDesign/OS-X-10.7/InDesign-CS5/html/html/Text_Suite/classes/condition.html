Class condition

expand

Description

A condition for conditional text.

  • Plural name:
    • conditions
  • Defined in:
    • Text Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • id (r/o) integer -- The unique ID of the condition.
  • index (r/o) integer -- The index of the condition within its containing object.
  • indicator color anything -- The color for the condition indicator; specified either as an array of three doubles representing RGB values in the range 0 to 255, or as a UI color. Can return: list of 3 fixed (0 - 255) or light blue/red/green/blue/yellow/magenta/cyan/gray/black/oran...
  • indicator method use underline / use highlight -- The condition indicator method.
  • label string -- A property that can be set to any string.
  • name string -- The name of the condition.
  • object reference (r/o) condition -- An object reference for the object.
  • parent (r/o) reference -- The parent of the condition (a application or document). Can return: application or document.
  • properties record -- A property that allows setting of several properties at the same time.
  • underline indicator appearance wavy / solid / dashed -- The condition underline indicator appearance.
  • visible boolean -- If true, the condition is visible.
