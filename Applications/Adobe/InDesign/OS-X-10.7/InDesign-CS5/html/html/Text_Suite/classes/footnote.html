Class footnote

expand

Description

A footnote.

  • Plural name:
    • footnotes
  • Defined in:
    • Text Suite

Properties

  • all graphics (r/o) list of graphic -- Lists all graphics contained by the footnote.
  • all page items (r/o) list of page item -- Lists all page items contained by the footnote.
  • class (r/o) type class -- The class descriptor type.
  • contents anything -- The text contents of the footnote. Can return: string, auto page number/next page number/previous page number/section marker/bullet character/copyright symbol/degree symbol/ellipsis character/forced line break/paragraph symbol/registered trademark/sect...
  • id (r/o) integer -- The unique ID of the footnote.
  • index (r/o) integer -- The index of the footnote within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the footnote; this is an alias to the footnote's label property.
  • object reference (r/o) footnote -- An object reference for the object.
  • parent (r/o) reference -- The parent of the footnote (a xml story, story, text frame or insertion point). Can return: xml story, story, text frame or insertion point.
  • properties record -- A property that allows setting of several properties at the same time.
  • story offset (r/o) insertion point -- The location of the footnote marker in the parent story.
