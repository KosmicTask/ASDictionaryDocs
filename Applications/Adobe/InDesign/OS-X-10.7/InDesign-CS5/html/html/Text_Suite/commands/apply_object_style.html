Command apply object style

Description

Applies the specified object style.

  • Defined in:
    • Text Suite

Parameters

  • text frame
  • using object style -- The object style to apply.
  • [clearing overrides boolean] -- If true, clears the object's existing attributes before applying the style. Can accept: boolean (Default: TRUE).
  • [clearing overrides through root object style boolean] -- If true, clears attributes and formatting applied to the object that are not defined in the object style. Can accept: boolean (Default: FALSE).
