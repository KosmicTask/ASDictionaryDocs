Class note

expand

Description

A note in a story.

  • Plural name:
    • notes
  • Defined in:
    • Text Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • collapsed boolean -- If true, the note is collapsed in galley view.
  • creation date (r/o) date -- The date and time the note was created.
  • id (r/o) integer -- The unique ID of the note.
  • index (r/o) integer -- The index of the note within its containing object.
  • label string -- A property that can be set to any string.
  • modification date (r/o) date -- The date and time the note was last modified.
  • name string -- The name of the note; this is an alias to the note's label property.
  • object reference (r/o) note -- An object reference for the object.
  • parent (r/o) reference -- The parent of the note (a xml story, story, text frame, insertion point or cell). Can return: xml story, story, text frame, insertion point or cell.
  • properties record -- A property that allows setting of several properties at the same time.
  • story offset (r/o) insertion point -- The insertion point before the table in the story containing the table.
  • user name (r/o) string -- The user who made the note.
