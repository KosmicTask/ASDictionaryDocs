Class text path

expand

Description

A text object that is on a path.

  • Plural name:
    • text paths
  • Defined in:
    • Text Suite

Properties

  • center bracket (r/o) fixed -- The halfway point between the start bracket and the end bracket.
  • class (r/o) type class -- The class descriptor type.
  • contents anything -- The contents of the text frame. Can return: string, placeholder text or auto page number/next page number/previous page number/section marker/bullet character/copyright symbol/degree symbol/ellipsis character/forced line break/paragraph symbol/register...
  • end bracket fixed -- The end of the type on a path. Note: Additional text becomes overset text unless the text is linked to another path or text frame.
  • end text frame (r/o) reference -- The last text frame in the thread. Can return: text frame or text path.
  • flip path effect not flipped / flipped / undefined flip value -- The flip effect applied to the type on a path.
  • id (r/o) integer -- The unique ID of the text path.
  • index (r/o) integer -- The index of the text path within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the text path; this is an alias to the text path's label property.
  • next text frame reference -- The next text frame in the thread. Can return: text frame or text path. Can also accept: nothing.
  • object reference (r/o) text path -- An object reference for the object.
  • overflows (r/o) boolean -- If true, the story has overset text.
  • parent (r/o) reference -- The parent of the text path (a spline item, polygon, graphic line, rectangle, oval, text frame or EPSText). Can return: spline item, polygon, graphic line, rectangle, oval, text frame or EPSText.
  • parent story (r/o) story -- The story that contains the text.
  • path alignment top path alignment / bottom path alignment / center path alignment -- The alignment of the type on a path.
  • path effect rainbow path effect / skew path effect / ribbon path effect / stair step path effect / gravity path effect -- The effect applied to the type on a path.
  • path spacing integer -- The spacing applied to the type on a path.
  • previous text frame reference -- The previous text frame in the thread. Can return: text frame or text path. Can also accept: nothing.
  • properties record -- A property that allows setting of several properties at the same time.
  • start bracket fixed -- The location of the start of the type on the path, expressed in points. Note: 0 is the first point on the path.
  • start text frame (r/o) reference -- The first text frame in the thread. Can return: text frame or text path.
  • text alignment ascender text alignment / descender text alignment / center text alignment / baseline text alignment / above right em box text alignment / below left em box text alignment / above right icf box text alignment / below left icf box text alignment -- The alignment applied to the type on a path.
  • text frame index (r/o) integer -- The index of the text frame within the story.
  • after place (r/o) string -- Dispatched after a text path is placed. This event bubbles. This event is not cancelable.
  • before place (r/o) string -- Dispatched before a text path is placed. This event bubbles. This event is cancelable.
