Class xml story

expand

Description

XML text content that has not yet been placed in the layout.

  • Plural name:
    • xml stories
  • Defined in:
    • XML Suite
  • Inherits from:
    • story (Text Suite)

Properties

  • all graphics (r/o) list of graphic -- Lists all graphics contained by the xml story.
  • all page items (r/o) list of page item -- Lists all page items contained by the xml story.
  • applied font anything -- The font applied to the xml story, specified as either a font object or the name of font family. Can return: font or string.
  • applied nested styles (r/o) list of character style -- The applied character styles dictated by nested styles.
  • associated XML element (r/o) XML item -- The XML element associated with the xml story.
  • contents (r/o) anything -- The contents of the text. Can return: string or auto page number/next page number/previous page number/section marker/bullet character/copyright symbol/degree symbol/ellipsis character/forced line break/paragraph symbol/registered trademark/section sym...
  • fill color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the fill of the xml story. . Can also accept: string.
  • fill tint fixed -- The tint (as a percentage) of the fill color of the xml story. (To specify a tint percentage, use a number in the range of 0 to 100; to use the inherited or overridden value, use -1.)
  • gradient fill start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the fill of the text, in the format [x, y].
  • gradient stroke start list of number | list of string -- The starting point (in page coordinates) of a gradient applied to the stroke of the text, in the format [x, y].
  • horizontal scale fixed -- The horizontal scaling applied to the xml story.
  • id (r/o) integer -- The unique ID of the xml story.
  • index (r/o) integer -- The index of the xml story within its containing object.
  • name string -- The name of the xml story; this is an alias to the xml story's label property.
  • object reference (r/o) xml story -- An object reference for the object.
  • OpenType features list of anything -- OpenType features. Can return: list of list of 2 strings or integers.
  • parent (r/o) document -- The parent of the xml story (a document).
  • skew fixed -- The skew angle of the xml story.
  • stroke color swatch -- The swatch (color, gradient, tint, or mixed ink) applied to the stroke of the xml story. Can also accept: string.
  • stroke tint fixed -- The tint (as a percentage) of the stroke color of the xml story. (To specify a tint percentage, use a number in the range of 0 to 100; to use the inherited or overridden value, use -1.)
  • tab list list of anything -- A list of the tab stops in the paragraph. Can return: list of record.
  • text containers (r/o) list of reference -- Array of text frames or text paths. Can return: list of text frames or text paths.
  • vertical scale fixed -- The vertical scaling applied to the xml story.
  • after place (r/o) string -- Dispatched after a xml story is placed. This event bubbles. This event is not cancelable.
  • before place (r/o) string -- Dispatched before a xml story is placed. This event bubbles. This event is cancelable.

Notes

  • Contained by:
    • document (Basics Suite)
