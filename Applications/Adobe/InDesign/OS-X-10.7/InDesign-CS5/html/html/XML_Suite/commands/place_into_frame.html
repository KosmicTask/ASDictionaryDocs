Command place into frame

Description

Places the XML element into a new rectangular page item. If the XML element was already associated with a page item, that page item is deleted.

  • Defined in:
    • XML Suite

Parameters

  • XML element
  • on reference -- The page or spread on which to create the new page item. Can accept: spread, page or master spread.
  • geometric bounds number | string -- The bounds of the page item excluding the stroke width, in the format [y1, x1, y2, x2].

Result

  • page item -- The new frame.
