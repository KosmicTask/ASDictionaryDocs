Command convert to element

Description

Converts the XML attribute to a child element of its parent element.

  • Defined in:
    • XML Suite

Parameters

  • XML attribute
  • [located element start / element end] -- The location of the new XML element within the parent XML element of the XML attribute. Can accept: element start/element end (Default: element start).
  • [markup tag XML tag] -- The XML tag to apply to the new XML element.

Result

  • XML element -- The new element.
