Properties

  • contents (r/o) anything -- The contents of the text. Can return: string or auto page number/next page number/previous page number/section marker/bullet character/copyright symbol/degree symbol/ellipsis character/forced line break/paragraph symbol/registered trademark/section sym...
  • id (r/o) integer -- The unique ID of the DTD.
  • index (r/o) integer -- The index of the DTD within its containing object.
  • object reference (r/o) DTD -- An object reference for the object.
  • parent (r/o) document -- The parent of the DTD (a document).
  • public id (r/o) string -- The public ID of the DOCTYPE declaration. Note: Valid only when the DTD is an external subset.
  • root tag XML tag -- The tag of the root object.
  • system id (r/o) string -- The system ID of the DOCTYPE declaration. Note: Valid only when the DTD is an external subset.
