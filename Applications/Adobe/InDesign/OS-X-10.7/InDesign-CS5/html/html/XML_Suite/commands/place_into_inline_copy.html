Command place into inline copy

Description

Associates an existing page item with the XML element and places it into an inline frame.

  • Defined in:
    • XML Suite

Parameters

  • XML element
  • copy item page item -- The page item to copy.
  • [retain existing frame boolean] -- If true, moves the existing page item. If false, moves a copy of the page item. Can accept: boolean (Default: FALSE).

Result

  • page item -- The page item.
