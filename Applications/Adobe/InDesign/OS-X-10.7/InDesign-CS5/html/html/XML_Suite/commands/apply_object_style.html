Command apply object style

Description

Applies an object style to the frame associated with the object.

  • Defined in:
    • XML Suite

Parameters

  • XML element
  • using anything -- The object style to apply. Can accept: string or object style.
  • [clearing overrides boolean] -- If true, removes local formatting before applying the object style. Can accept: boolean (Default: TRUE).
  • [clearing overrides through root object style boolean] -- If true, clears unchecked category attributes through the root style. Can accept: boolean (Default: FALSE).
