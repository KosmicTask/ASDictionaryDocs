Command place into copy

Description

Associates the XML element with a copy of a page item.

  • Defined in:
    • XML Suite

Parameters

  • XML element
  • on reference -- The page or spread on which to create the new page item. Can accept: spread, page or master spread.
  • place point number | string -- The page coordinates of the top left corner of the page item, in the format [y1, x1]
  • copy item page item -- The page item to copy.
  • [retain existing frame boolean] -- If true, associates the XML element with the existing page item and moves the page item (rather than a copy of the page item). Can accept: boolean (Default: FALSE).

Result

  • page item -- The new page item.
