Properties

  • contents anything -- The contents of the text. Can return: string or auto page number/next page number/previous page number/section marker/bullet character/copyright symbol/degree symbol/ellipsis character/forced line break/paragraph symbol/registered trademark/section sym...
  • id (r/o) integer -- The unique ID of the XML element.
  • index (r/o) integer -- The index of the XML element within its containing object.
  • markup tag XML tag -- The XML tag applied to the element. Can also accept: string.
  • object reference (r/o) XML element -- An object reference for the object.
  • parent (r/o) reference -- The parent of the XML element (a document or XML element). Can return: document or XML element.
  • parent story (r/o) story -- The story that contains the text.
  • story offset (r/o) insertion point -- The insertion point before the table in the story containing the table.
  • XML content (r/o) reference -- The text content or page item referred to by the element. Can return: text, story, page item, movie, sound, graphic, table or cell.
