Command asynchronous export file

Description

asynchronously exports the object(s) to a file.

  • Defined in:
    • XML Suite

Parameters

  • reference -- Supported by the following objects: XML element, xml story.
  • format anything -- The export format, specified as an enumeration value or as an extension that appears in the Save as type or Format menu in the Export dialog. Can accept: tagged text/PDF type/EPS type/RTF/text type/XML/JPG/interactive PDF/packaged XFL/SWF/InDesign snip...
  • to alias | string -- The path to the export file.
  • [showing options boolean] -- If true, displays the export options dialog. Can accept: boolean (Default: FALSE).
  • [using PDF export preset] -- The export style.
  • [version comments string] -- The comment for this version.
  • [force save boolean] -- If true, forcibly saves a version. Can accept: boolean (Default: FALSE).

Result

  • background task -- object representing the asynchronous export
