Description

A bookmark.

  • Plural name:
    • bookmarks
  • Defined in:

Properties

  • class (r/o) type class -- The class descriptor type.
  • destination (r/o) reference -- The destination that the hyperlink points to. Can return: hyperlink text destination, hyperlink page destination, hyperlink external page destination or page.
  • id (r/o) integer -- The unique ID of the bookmark.
  • indent (r/o) integer -- The indent level of the bookmark.
  • index (r/o) integer -- The index of the bookmark within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the bookmark.
  • object reference (r/o) bookmark -- An object reference for the object.
  • parent (r/o) reference -- The parent of the bookmark (a document or bookmark). Can return: document or bookmark.
  • properties record -- A property that allows setting of several properties at the same time.
