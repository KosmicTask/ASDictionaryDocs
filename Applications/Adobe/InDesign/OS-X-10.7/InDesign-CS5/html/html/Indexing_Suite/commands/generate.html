Command generate

Description

Generates a new index story.

  • Defined in:
    • Indexing Suite

Parameters

  • index
  • [on reference] -- The spread or page on which to place the story. Can accept: page, spread or master spread.
  • [place point number | string] -- The coordinates of the upper left corner of the story bounding box, in the format [x, y].
  • [destination layer layer] -- The layer on which to place the story.
  • [autoflowing boolean] -- If true, allows the story to flow onto subsequent pages if the story does not fit on the specified page. If no subsequent pages exist in the document, creates the necessary pages. Can accept: boolean (Default: FALSE).
  • [include overset boolean] -- If true, includes topics in overset text in the story. . Can accept: boolean (Default: FALSE).

Result

  • list of story -- The story containing the generated index.
