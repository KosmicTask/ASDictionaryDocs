Command capitalize

Description

Makes the initial letter for the specified index topic or group of index topics upper case.

  • Defined in:
    • Indexing Suite

Parameters

  • reference -- Supported by the following objects: index, topic.
  • [capitalization option selected entry / include subentries / all level 1 entries / all entries] -- The entry or entries to capitalize. Can accept: selected entry/include subentries/all level 1 entries/all entries (Default: all entries).
