Class cross reference

expand

Description

A cross reference to a different index topic. (For cross references in text, use the 'cross reference source' and 'hyperlink' objects.)

  • Plural name:
    • cross references
  • Defined in:
    • Indexing Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • cross reference type see or also bracket / see / see also / see herein / see also herein / custom cross reference / custom cross reference before / custom cross reference after -- The text that precedes or follows the referenced topic.
  • custom type string string -- The text used for a custom cross reference type. Note: Valid only for custom cross reference types.
  • id (r/o) integer -- The unique ID of the cross reference.
  • index (r/o) integer -- The index of the cross reference within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the cross reference; this is an alias to the cross reference's label property.
  • object reference (r/o) cross reference -- An object reference for the object.
  • parent (r/o) topic -- The parent of the cross reference (a topic).
  • properties record -- A property that allows setting of several properties at the same time.
  • referenced topic topic -- The topic to which this cross reference points.
