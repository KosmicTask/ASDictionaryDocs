Class page reference

expand

Description

The page reference for an index topic.

  • Plural name:
    • page references
  • Defined in:
    • Indexing Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • id (r/o) integer -- The unique ID of the page reference.
  • index (r/o) integer -- The index of the page reference within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the page reference; this is an alias to the page reference's label property.
  • object reference (r/o) page reference -- An object reference for the object.
  • page number style override character style -- The character style applied to page numbers.
  • page reference limit anything -- The paragraph style or number of paragraphs or pages that defines the last page in a page range. Note: Valid only when page reference type specifies the next use of a paragraph style or a number of paragraphs or pages. . Can return: paragraph style or ...
  • page reference type current page / to next style change / to next use of style / to end of story / to end of document / to end of section / for next n paragraphs / for next n pages / suppress page numbers -- Options for index page references.
  • parent (r/o) topic -- The parent of the page reference (a topic).
  • properties record -- A property that allows setting of several properties at the same time.
  • source text (r/o) text -- The hyperlinked text or insertion point.
