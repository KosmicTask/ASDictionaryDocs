Class topic

expand

Description

An index topic.

  • Plural name:
    • topics
  • Defined in:
    • Indexing Suite

Properties

  • class (r/o) type class -- The class descriptor type.
  • index (r/o) integer -- The index of the topic within its containing object.
  • name string -- The name of the topic.
  • object reference (r/o) topic -- An object reference for the object.
  • parent (r/o) reference -- The parent of the topic (a index section, topic or index). Can return: index section, topic or index.
  • properties record -- A property that allows setting of several properties at the same time.
  • sort order string -- The string by which the topic is sorted instead of the topic name is not used. Note: The actual topic text, rather than the sort order text, appears in the index.
