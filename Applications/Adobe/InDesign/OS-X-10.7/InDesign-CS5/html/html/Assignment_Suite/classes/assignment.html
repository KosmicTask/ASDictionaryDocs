Description

An assignment.

  • Plural name:
    • assignments
  • Defined in:
    • Assignment Suite

Properties

  • assignment file status (r/o) assignment up to date / assignment out of date / assignment file missing -- The status of the assignment file.
  • class (r/o) type class -- The class descriptor type.
  • document path (r/o) alias | string -- The path to the document that the hyperlink destination points to.
  • export options empty frames / assigned spreads / everything -- The content to export in the assignment.
  • file path (r/o) string -- The file path (colon delimited on the Mac OS). Can also accept: alias or string.
  • frame color anything -- The color of the assignment's frames. Can return: list of 3 fixed (0 - 255) or light blue/red/green/blue/yellow/magenta/cyan/gray/black/orange/dark green/teal/tan/brown/violet/gold/dark blue/pink/lavender/brick red/olive green/peach/burgundy/grass gree...
  • id (r/o) integer -- The unique ID of the assignment.
  • include links when package boolean -- If true, includes linked files when packaging the assignment.
  • index (r/o) integer -- The index of the assignment within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the assignment.
  • object reference (r/o) assignment -- An object reference for the object.
  • package up to date (r/o) boolean -- If true, the assignment package is up to date.
  • packaged (r/o) boolean -- If true, the assignment is packaged.
  • parent (r/o) document -- The parent of the assignment (a document).
  • properties record -- A property that allows setting of several properties at the same time.
  • user name string -- The user name to assign to tracked changes and notes.
