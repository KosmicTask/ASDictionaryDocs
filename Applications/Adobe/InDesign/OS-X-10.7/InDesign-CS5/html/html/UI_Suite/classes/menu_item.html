Properties

  • associated menu action (r/o) menu action -- The menu action that implements the menu item.
  • checked (r/o) boolean -- If true, the menu item associated with the menu action is checked.
  • enabled (r/o) boolean -- If true, the menu item is enabled.
  • id (r/o) integer -- The unique ID of the menu item.
  • index (r/o) integer -- The index of the menu item within its containing object.
  • name (r/o) string -- The name of the menu item.
  • object reference (r/o) menu item -- An object reference for the object.
  • parent (r/o) reference -- The parent of the menu item (a menu or submenu). Can return: menu or submenu.
