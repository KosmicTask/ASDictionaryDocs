Class radiobutton control

expand

Description

An individual radiobutton control in a radiobutton group.

  • Plural name:
    • radiobutton controls
  • Defined in:
    • UI Suite
  • Inherits from:
    • widget (UI Suite)

Properties

  • checked state boolean -- If true, the control is checked by default in the dialog.
  • id (r/o) integer -- The unique ID of the radiobutton control.
  • index (r/o) integer -- The index of the radiobutton control within its containing object.
  • object reference (r/o) radiobutton control -- An object reference for the object.
  • parent (r/o) radiobutton group -- The parent of the radiobutton control (a radiobutton group).
  • static label string -- Text that appears in the radiobutton control.
