Class dialog

expand

Description

A dialog.

  • Plural name:
    • dialogs
  • Defined in:
    • UI Suite

Properties

  • can cancel boolean -- If true, creates a Cancel button in the dialog that allows users to close the dialog without saving any selections. If false, the dialog contains an OK button but no Cancel button.
  • class (r/o) type class -- The class descriptor type.
  • id (r/o) integer -- The unique ID of the dialog.
  • index (r/o) integer -- The index of the dialog within its containing object.
  • label string -- A property that can be set to any string.
  • name string -- The name of the dialog.
  • object reference (r/o) dialog -- An object reference for the object.
  • parent (r/o) reference -- The parent of the dialog (a application).
  • properties record -- A property that allows setting of several properties at the same time.

Elements

contains dialog columns , events , event listeners

  • dialog column -- by index, name, range, relative, test, id
  • event -- by index, name, range, relative, test, id
  • event listener -- by index, name, range, relative, test, id
