Class menu action

Description

A menu action.

  • Plural name:
    • menu actions
  • Defined in:
    • UI Suite

Properties

  • area (r/o) string -- The menu action area.
  • checked (r/o) boolean -- If true, the menu item associated with the menu action is checked.
  • class (r/o) type class -- The class descriptor type.
  • enabled (r/o) boolean -- If true, the menu action is enabled.
  • id (r/o) integer -- The unique ID of the menu action.
  • index (r/o) integer -- The index of the menu action within its containing object.
  • label string -- A property that can be set to any string.
  • name (r/o) string -- The name of the menu action.
  • object reference (r/o) menu action -- An object reference for the object.
  • parent (r/o) reference -- The parent of the menu action (a application).
  • properties record -- A property that allows setting of several properties at the same time.
  • title (r/o) string -- The name of the menu action for display in the user interface. The title includes any ampersand characters (&), which are used to tell the Windows OS to underline the following character in the name for use with the Alt key to navigate to a menu item. ...
  • after invoke (r/o) string -- Dispatched after the menu action is invoked. This event does not bubble. This event is not cancelable.
  • before invoke (r/o) string -- Dispatched before the menu action is invoked. This event does not bubble. This event is cancelable.
