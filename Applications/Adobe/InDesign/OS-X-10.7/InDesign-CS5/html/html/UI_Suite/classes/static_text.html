Class static text

expand

Description

A static text control (typically a label for another control or a set of controls).

  • Plural name:
    • static texts
  • Defined in:
    • UI Suite
  • Inherits from:
    • widget (UI Suite)

Properties

  • id (r/o) integer -- The unique ID of the static text.
  • index (r/o) integer -- The index of the static text within its containing object.
  • object reference (r/o) static text -- An object reference for the object.
  • parent (r/o) reference -- The parent of the static text (a dialog column, dialog row, enabling group or border panel). Can return: dialog column, dialog row, enabling group or border panel.
  • static label string -- Text that appears in the static text.
