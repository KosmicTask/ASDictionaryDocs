Class enabling group

expand

Description

A bordered panel that contains an enabling checkbox. A user makes the enabling group active or inactive by clicking the checkbox. An enabling group can contain any type and number of controls.

  • Plural name:
    • enabling groups
  • Defined in:
    • UI Suite
  • Inherits from:
    • widget (UI Suite)

Properties

  • checked state boolean -- If true, the control is checked by default in the dialog.
  • id (r/o) integer -- The unique ID of the enabling group.
  • index (r/o) integer -- The index of the enabling group within its containing object.
  • object reference (r/o) enabling group -- An object reference for the object.
  • parent (r/o) reference -- The parent of the enabling group (a dialog column, dialog row, enabling group or border panel). Can return: dialog column, dialog row, enabling group or border panel.
  • static label string -- Text that appears in the enabling group.
