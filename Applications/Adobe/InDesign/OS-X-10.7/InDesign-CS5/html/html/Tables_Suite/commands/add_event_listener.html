Command add event listener

Description

Adds an event listener.

  • Defined in:
    • Tables Suite

Parameters

  • reference -- Supported by the following objects: cell, table, column, row, table style, table style group, cell style, cell style group.
  • event type string -- The event type.
  • handler anything -- The event handler. Can accept: alias or string or JavaScript Function.
  • [captures boolean] -- This parameter is obsolete. Can accept: boolean (Default: FALSE).

Result

  • event listener -- The new event listener.
