Class tab

reduce

Description

A Safari window tab.

  • Plural name:
    • tabs
  • Defined in:
    • Safari suite
  • Inherits from:
    • item (Standard Suite)

Properties

  • class (r/o) type class -- The class of the object.
  • properties record -- All of the object's properties.
  • index (r/o) integer -- The index of the tab, ordered left to right.
  • name (r/o) Unicode text -- The name of the tab.
  • source (r/o) Unicode text -- The HTML source of the web page currently loaded in the tab.
  • text (r/o) text -- The text of the web page currently loaded in the tab. Modifications to text aren't reflected on the web page.
  • URL Unicode text -- The current URL of the tab.
  • visible (r/o) boolean -- Whether the tab is currently visible.

Notes

  • Contained by:
    • window (Safari suite)
