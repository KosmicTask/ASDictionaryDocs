Command add reading list item

Description

Add a new Reading List item with the given URL. Allows a custom title and preview text to be specified.

  • Defined in:
    • Safari suite

Parameters

  • Unicode text -- URL of the Reading List item
  • [and preview text Unicode text] -- Preview text for the Reading List item, usually the first few sentences of the article
  • [with title Unicode text] -- Title of the Reading List item
