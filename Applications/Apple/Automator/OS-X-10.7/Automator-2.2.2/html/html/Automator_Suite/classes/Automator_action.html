Class Automator action

expand

Description

A single step in a workflow

  • Plural name:
    • Automator actions
  • Defined in:
    • Automator Suite
  • Inherits from:
    • item (Standard Suite)

Properties

  • bundle id (r/o) Unicode text -- The bundle identifier for the action
  • category (r/o) list -- The category that contains the action
  • comment Unicode text -- The comment for the name of the action
  • enabled boolean -- Is the action enabled?
  • execution error message (r/o) Unicode text -- The text error message generated by execution of the action
  • execution error number (r/o) integer -- The numeric error code generated by execution of the action
  • execution result (r/o) anything -- The result of the action, passed as input to the next action
  • icon name (r/o) Unicode text -- The icon name of the action
  • id (r/o) Unicode text -- The unique identifier for the action
  • ignores input boolean -- Shall the action ignore its input when it is run?
  • index integer -- The index of the action
  • input types (r/o) list -- The input types accepted by the action
  • keywords (r/o) list -- The keywords that describe the action
  • name (r/o) Unicode text -- The localized name of the action
  • output types (r/o) list -- The output types produced by the action
  • parent workflow (r/o) workflow -- The workflow that contains the action
  • path (r/o) alias -- The path of the file that contains the action
  • show action when run boolean -- Shall the action show its user interface when it is run?
  • target application (r/o) list -- The application with which the action communicates
  • version (r/o) Unicode text -- The version of the action
  • warning action (r/o) Unicode text -- The action suggested by the warning, if any
  • warning level (r/o) irreversible / none / reversible -- The level of the warning, increasing in likelihood of data loss
  • warning message (r/o) Unicode text -- The message that accompanies the warning, if any

Elements

contains required resources , settings

  • required resource -- by name, index, relative, range, test
  • setting -- by name, index, relative, range, test

Notes

  • Contained by:
    • application (Automator Suite)
