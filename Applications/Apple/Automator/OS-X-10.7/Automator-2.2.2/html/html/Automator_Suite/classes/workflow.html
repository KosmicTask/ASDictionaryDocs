Class workflow

expand

Description

A series of actions stored in a file

  • Plural name:
    • workflows
  • Defined in:
    • Automator Suite
  • Inherits from:
    • document (Standard Suite) → item (Standard Suite)

Properties

  • current action (r/o) Automator action -- The current or most recent action of the workflow
  • execution error message (r/o) Unicode text -- The text error message generated by the most recent execution
  • execution error number (r/o) integer -- The numeric error code generated by the most recent execution
  • execution id (r/o) Unicode text -- The identifier of the current or most recent execution
  • execution result (r/o) anything -- The result of the most recent execution; the output of the last action of that execution
  • name (r/o) Unicode text -- The name of the workflow

Elements

contains Automator actions , variables

  • Automator action -- by name, index, relative, range, test, id
  • variable -- by name, index, relative, range, test, id

Notes

  • Contained by:
    • application (Automator Suite)
    • Automator action (Automator Suite)
