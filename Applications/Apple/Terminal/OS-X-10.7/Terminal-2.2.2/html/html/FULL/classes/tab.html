Class tab

reduce

Description

A tab.

  • Plural name:
    • tabs
  • Defined in:
    • Terminal Suite

Properties

  • number of rows integer -- The number of rows displayed in the tab.
  • number of columns integer -- The number of columns displayed in the tab.
  • contents (r/o) text -- The currently visible contents of the tab.
  • history (r/o) text -- The contents of the entire scrolling buffer of the tab.
  • busy (r/o) boolean -- Whether the tab is busy running a process.
  • processes (r/o) list of text -- The processes currently running in the tab.
  • selected boolean -- Whether the tab is selected.
  • title displays custom title boolean -- Whether the title contains a custom title.
  • custom title text -- The tab’s custom title.
  • tty (r/o) text -- The tab’s TTY device.
  • current settings settings set -- The set of settings which control the tab’s behavior and appearance.
  • cursor color color -- The cursor color for the tab.
  • background color color -- The background color for the tab.
  • normal text color color -- The normal text color for the tab.
  • bold text color color -- The bold text color for the tab.
  • clean commands list of text -- The processes which will be ignored when checking whether a tab can be closed without showing a prompt.
  • title displays device name boolean -- Whether the title contains the device name.
  • title displays shell path boolean -- Whether the title contains the shell path.
  • title displays window size boolean -- Whether the title contains the tab’s size, in rows and columns.
  • title displays file name boolean -- Whether the title contains the file name.
  • font name text -- The name of the font used to display the tab’s contents.
  • font size integer -- The size of the font used to display the tab’s contents.
  • font antialiasing boolean -- Whether the font used to display the tab’s contents is antialiased.

Notes

  • Contained by:
    • window (Standard Suite)
