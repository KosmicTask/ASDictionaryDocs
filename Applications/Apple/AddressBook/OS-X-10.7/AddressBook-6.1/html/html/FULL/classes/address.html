Class address

reduce

Description

Address for the given record.

  • Plural name:
    • addresses
  • Defined in:
    • Address Book Script Suite

Properties

  • city text or missing value -- City part of the address.
  • formatted address (r/o) text or missing value -- properly formatted string for this address.
  • street text or missing value -- Street part of the address, multiple lines separated by carriage returns.
  • id text -- unique identifier for this address.
  • zip text or missing value -- Zip or postal code of the address.
  • country text or missing value -- Country part of the address.
  • label text or missing value -- Label.
  • country code text or missing value -- Country code part of the address (should be a two character iso country code).
  • state text or missing value -- State, Province, or Region part of the address.

Notes

  • Contained by:
    • person (Address Book Script Suite)
