Address Book Rollover Suite

These event definitions are used for constructing Address Book Rollover plug-ins. They would not normally appear in a typical end user script.

Commands

  • action property -- RollOver - Which property this roll over is associated with (Properties can be one of maiden name, phone, email, url, birth date, custom date, related name, aim, icq, jabber, msn, yahoo, address.)
  • action title -- RollOver - Returns the title that will be placed in the menu for this roll over
  • perform action -- RollOver - Performs the action on the given person and value
  • should enable action -- RollOver - Determines if the rollover action should be enabled for the given person and value
