Standard Suite

Common classes and commands for all applications.

Classes

  • application -- The application's top-level scripting object.
  • document -- A document.
  • window -- A window.

Commands

  • open -- Open a document.
  • close -- Close a document.
  • print -- Print a document.
  • quit -- Quit the application.
  • count -- Return the number of elements of a particular class within an object.
  • delete -- Delete an object.
  • duplicate -- Copy an object.
  • exists -- Verify that an object exists.
  • make -- Create a new object.
  • move -- Move an object to a new location.

iCal

iCal classes and commands

Classes

  • calendar -- This class represents a calendar.
  • display alarm -- This class represents a message alarm.
  • mail alarm -- This class represents a mail alarm.
  • sound alarm -- This class represents a sound alarm.
  • open file alarm -- This class represents an 'open file' alarm.
  • attendee -- This class represents a attendee.
  • todo -- This class represents a task.
  • event -- This class represents an event.

Commands

  • create calendar -- Creates a new calendar (obsolete, will be removed in next release)
  • reload calendars -- Tell the application to reload all calendar files contents
  • switch view -- Show calendar on the given view
  • view calendar -- Show calendar on the given date
  • GetURL -- Subscribe to a remote calendar through a webcal or http URL
  • show -- Show the event or to-do in the calendar window
