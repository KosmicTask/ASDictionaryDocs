Class desktop window

reduce

Description

the desktop window

  • Defined in:
    • Window classes
  • Inherits from:
    • Finder window (Window classes) → window (Window classes)

Properties

  • id (r/o) unsigned integer -- the unique id for this window
  • position point -- the upper left position of the window
  • bounds bounding rectangle -- the boundary rectangle for the window
  • titled (r/o) boolean -- Does the window have a title bar?
  • name (r/o) Unicode text -- the name of the window
  • index integer -- the number of the window in the front-to-back layer ordering
  • closeable (r/o) boolean -- Does the window have a close box?
  • floating (r/o) boolean -- Does the window have a title bar?
  • modal (r/o) boolean -- Is the window modal?
  • resizable (r/o) boolean -- Is the window resizable?
  • zoomable (r/o) boolean -- Is the window zoomable?
  • zoomed boolean -- Is the window zoomed?
  • visible (r/o) boolean -- Is the window visible (always true for open Finder windows)?
  • collapsed boolean -- Is the window collapsed
  • properties record -- every property of a window
  • target reference -- the container at which this file viewer is targeted
  • current view icon view / list view / column view / group view / flow view -- the current view for the container window
  • icon view options (r/o) icon view options -- the icon view options for the container window
  • list view options (r/o) list view options -- the list view options for the container window
  • column view options (r/o) column view options -- the column view options for the container window
  • toolbar visible boolean -- Is the window's toolbar visible?
  • statusbar visible boolean -- Is the window's status bar visible?
  • sidebar width integer -- the width of the sidebar for the container window
