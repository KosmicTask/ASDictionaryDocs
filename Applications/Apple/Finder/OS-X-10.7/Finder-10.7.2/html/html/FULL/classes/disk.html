Class disk

reduce

Description

A disk

  • Plural name:
    • disks
  • Defined in:
    • Containers and folders
  • Inherits from:
    • container (Containers and folders) → item (Finder items)

Properties

  • name Unicode text -- the name of the item
  • displayed name (r/o) Unicode text -- the user-visible name of the item
  • name extension Unicode text -- the name extension of the item (such as “txt”)
  • extension hidden boolean -- Is the item's extension hidden from the user?
  • index (r/o) integer -- the index in the front-to-back ordering within its container
  • container (r/o) reference -- the container of the item
  • disk (r/o) reference -- the disk on which the item is stored
  • position point -- the position of the item within its parent window (can only be set for an item in a window viewed as icons or buttons)
  • desktop position point -- the position of the item on the desktop
  • bounds bounding rectangle -- the bounding rectangle of the item (can only be set for an item in a window viewed as icons or buttons)
  • label index integer -- the label of the item
  • locked boolean -- Is the file locked?
  • kind (r/o) Unicode text -- the kind of the item
  • description (r/o) Unicode text -- a description of the item
  • comment Unicode text -- the comment of the item, displayed in the “Get Info” window
  • size (r/o) double integer -- the logical size of the item
  • physical size (r/o) double integer -- the actual space used by the item on disk
  • creation date (r/o) date -- the date on which the item was created
  • modification date date -- the date on which the item was last modified
  • icon icon family -- the icon bitmap of the item
  • URL (r/o) Unicode text -- the URL of the item
  • owner Unicode text -- the user that owns the container
  • group Unicode text -- the user or group that has special access to the container
  • owner privileges read only / read write / write only / none
  • group privileges read only / read write / write only / none
  • everyones privileges read only / read write / write only / none
  • information window (r/o) reference -- the information window for the item
  • properties record -- every property of an item
  • class (r/o) type class -- the class of the item
  • entire contents (r/o) reference -- the entire contents of the container, including the contents of its children
  • expandable (r/o) boolean -- (NOT AVAILABLE YET) Is the container capable of being expanded as an outline?
  • expanded boolean -- (NOT AVAILABLE YET) Is the container opened as an outline? (can only be set for containers viewed as lists)
  • completely expanded boolean -- (NOT AVAILABLE YET) Are the container and all of its children opened as outlines? (can only be set for containers viewed as lists)
  • container window (r/o) reference -- the container window for this folder
  • id (r/o) integer -- the unique id for this disk (unchanged while disk remains connected and Finder remains running)
  • capacity (r/o) double integer -- the total number of bytes (free or used) on the disk
  • free space (r/o) double integer -- the number of free bytes left on the disk
  • ejectable (r/o) boolean -- Can the media be ejected (floppies, CDs, and so on)?
  • local volume (r/o) boolean -- Is the media a local volume (as opposed to a file server)?
  • startup (r/o) boolean -- Is this disk the boot disk?
  • format (r/o) Mac OS format / Mac OS Extended format / UFS format / NFS format / audio format / ProDOS format / MSDOS format / NTFS format / ISO 9660 format / High Sierra format / QuickTake format / Apple Photo format / AppleShare format / UDF format / WebDAV format / FTP format / Packet written UDF format / Xsan format / unknown format -- the filesystem format of this disk
  • journaling enabled (r/o) boolean -- Does this disk do file system journaling?
  • ignore privileges boolean -- Ignore permissions on this disk?

Elements

contains items , containers , folders , files , alias files , application files , document files , internet location files , clippings , packages

  • alias file -- by index, name
  • application file -- by index, name, id
  • document file -- by index, name
  • internet location file -- by index, name
