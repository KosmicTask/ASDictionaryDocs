Class information window

reduce

Description

An inspector window (opened by “Show Info”)

  • Defined in:
    • Window classes
  • Inherits from:
    • window (Window classes)

Properties

  • id (r/o) unsigned integer -- the unique id for this window
  • position point -- the upper left position of the window
  • bounds bounding rectangle -- the boundary rectangle for the window
  • titled (r/o) boolean -- Does the window have a title bar?
  • name (r/o) Unicode text -- the name of the window
  • index integer -- the number of the window in the front-to-back layer ordering
  • closeable (r/o) boolean -- Does the window have a close box?
  • floating (r/o) boolean -- Does the window have a title bar?
  • modal (r/o) boolean -- Is the window modal?
  • resizable (r/o) boolean -- Is the window resizable?
  • zoomable (r/o) boolean -- Is the window zoomable?
  • zoomed boolean -- Is the window zoomed?
  • visible (r/o) boolean -- Is the window visible (always true for open Finder windows)?
  • collapsed boolean -- Is the window collapsed
  • properties record -- every property of a window
  • item (r/o) reference -- the item from which this window was opened
  • current panel General Information panel / Sharing panel / Memory panel / Preview panel / Application panel / Languages panel / Plugins panel / Name & Extension panel / Comments panel / Content Index panel / Burning panel / More Info panel / Simple Header panel -- the current panel in the information window
