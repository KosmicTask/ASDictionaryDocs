Class preferences

expand

Description

The Finder Preferences

  • Defined in:
    • Type Definitions

Properties

  • window (r/o) preferences window -- the window that would open if Finder preferences was opened
  • icon view options (r/o) icon view options -- the default icon view options
  • list view options (r/o) list view options -- the default list view options
  • column view options (r/o) column view options -- the column view options for all windows
  • folders spring open boolean -- Spring open folders after the specified delay?
  • delay before springing real -- the delay before springing open a container in seconds (from 0.167 to 1.169)
  • desktop shows hard disks boolean -- Hard disks appear on the desktop?
  • desktop shows external hard disks boolean -- External hard disks appear on the desktop?
  • desktop shows removable media boolean -- CDs, DVDs, and iPods appear on the desktop?
  • desktop shows connected servers boolean -- Connected servers appear on the desktop?
  • new window target reference -- target location for a newly-opened Finder window
  • folders open in new windows boolean -- Folders open into new windows?
  • new windows open in column view boolean -- Open new windows in column view?
  • all name extensions showing boolean -- Show name extensions, even for items whose “extension hidden” is true?
