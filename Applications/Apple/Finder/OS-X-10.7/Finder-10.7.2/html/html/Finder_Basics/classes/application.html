Properties

  • clipboard (r/o) reference -- (NOT AVAILABLE YET) the Finder’s clipboard window
  • name (r/o) international text -- the Finder’s name
  • visible boolean -- Is the Finder’s layer visible?
  • frontmost boolean -- Is the Finder the frontmost process?
  • selection reference -- the selection in the frontmost Finder window
  • insertion location (r/o) reference -- the container in which a new folder would appear if “New Folder” was selected
  • product version (r/o) Unicode text -- the version of the System software running on this computer
  • version (r/o) Unicode text -- the version of the Finder
  • startup disk (r/o) disk -- the startup disk
  • desktop (r/o) desktop-object -- the desktop
  • trash (r/o) trash-object -- the trash
  • home (r/o) folder -- the home directory
  • computer container (r/o) computer-object -- the computer location (as in Go > Computer)
  • Finder preferences (r/o) preferences -- Various preferences that apply to the Finder as a whole
