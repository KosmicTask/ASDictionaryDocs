Class disk

expand

Description

A disk

  • Plural name:
    • disks
  • Defined in:
    • Containers and folders
  • Inherits from:
    • container (Containers and folders) → item (Finder items)

Properties

  • id (r/o) integer -- the unique id for this disk (unchanged while disk remains connected and Finder remains running)
  • capacity (r/o) double integer -- the total number of bytes (free or used) on the disk
  • free space (r/o) double integer -- the number of free bytes left on the disk
  • ejectable (r/o) boolean -- Can the media be ejected (floppies, CDs, and so on)?
  • local volume (r/o) boolean -- Is the media a local volume (as opposed to a file server)?
  • startup (r/o) boolean -- Is this disk the boot disk?
  • format (r/o) Mac OS format / Mac OS Extended format / UFS format / NFS format / audio format / ProDOS format / MSDOS format / NTFS format / ISO 9660 format / High Sierra format / QuickTake format / Apple Photo format / AppleShare format / UDF format / WebDAV format / FTP format / Packet written UDF format / Xsan format / unknown format -- the filesystem format of this disk
  • journaling enabled (r/o) boolean -- Does this disk do file system journaling?
  • ignore privileges boolean -- Ignore permissions on this disk?

Notes

  • Contained by:
    • application (Finder Basics)
    • desktop-object (Containers and folders)
