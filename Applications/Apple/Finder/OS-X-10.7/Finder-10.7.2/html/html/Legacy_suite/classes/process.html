Class process

expand

Description

A process running on this computer

  • Plural name:
    • processes
  • Defined in:
    • Legacy suite

Properties

  • name (r/o) international text -- the name of the process
  • visible boolean -- Is the process' layer visible?
  • frontmost boolean -- Is the process the frontmost process?
  • file (r/o) reference -- the file from which the process was launched
  • file type (r/o) type class -- the OSType of the file type of the process
  • creator type (r/o) type class -- the OSType of the creator of the process (the signature)
  • accepts high level events (r/o) boolean -- Is the process high-level event aware (accepts open application, open document, print document, and quit)?
  • accepts remote events (r/o) boolean -- Does the process accept remote events?
  • has scripting terminology (r/o) boolean -- Does the process have a scripting terminology, i.e., can it be scripted?
  • total partition size (r/o) integer -- the size of the partition with which the process was launched
  • partition space used (r/o) integer -- the number of bytes currently used in the process' partition
