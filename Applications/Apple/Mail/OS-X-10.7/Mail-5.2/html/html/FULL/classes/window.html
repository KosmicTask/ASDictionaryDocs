Class window

reduce

Description

A window.

  • Plural name:
    • windows
  • Defined in:
    • Standard Suite

Properties

  • name (r/o) rich text -- The title of the window.
  • id (r/o) integer -- The unique identifier of the window.
  • index integer -- The index of the window, ordered front to back.
  • bounds bounding rectangle -- The bounding rectangle of the window.
  • closeable (r/o) boolean -- Does the window have a close button?
  • miniaturizable (r/o) boolean -- Does the window have a minimize button?
  • miniaturized boolean -- Is the window minimized right now?
  • resizable (r/o) boolean -- Can the window be resized?
  • visible boolean -- Is the window visible right now?
  • zoomable (r/o) boolean -- Does the window have a zoom button?
  • zoomed boolean -- Is the window zoomed right now?
  • document (r/o) document -- The document whose contents are displayed in the window.

Notes

  • Contained by:
    • application (Standard Suite)
