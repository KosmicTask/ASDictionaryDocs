Class mail attachment

reduce

Description

A file attached to a received message.

  • Plural name:
    • mail attachments
  • Defined in:
    • Message

Properties

  • name (r/o) rich text -- Name of the attachment
  • MIME type (r/o) rich text -- MIME type of the attachment E.g. text/plain.
  • file size (r/o) integer -- Approximate size in bytes.
  • downloaded (r/o) boolean -- Indicates whether the attachment has been downloaded.
  • id (r/o) rich text -- The unique identifier of the attachment.

Notes

  • Contained by:
    • message (Message)
