Class smtp server

expand

Description

An SMTP account (for sending email)

  • Plural name:
    • smtp servers
  • Defined in:
    • Message

Properties

  • name (r/o) rich text -- The name of an account
  • password rich text -- Password for this account. Can be set, but not read via scripting
  • account type (r/o) pop / smtp / imap / iCloud -- The type of an account
  • authentication password / apop / kerberos 5 / ntlm / md5 / external / Apple token / none -- Preferred authentication scheme for account
  • enabled boolean -- Indicates whether the account is enabled or not
  • user name rich text -- The user name used to connect to an account
  • port integer -- The port used to connect to an account
  • server name rich text -- The host name used to connect to an account
  • uses ssl boolean -- Indicates whether SSL is enabled for this receiving account
