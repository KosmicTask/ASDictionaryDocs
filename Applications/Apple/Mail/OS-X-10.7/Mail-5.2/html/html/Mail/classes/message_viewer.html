Class message viewer

expand

Description

Represents the object responsible for managing a viewer window

  • Plural name:
    • message viewers
  • Defined in:
    • Mail

Properties

  • drafts mailbox (r/o) mailbox -- The top level Drafts mailbox
  • inbox (r/o) mailbox -- The top level In mailbox
  • junk mailbox (r/o) mailbox -- The top level Junk mailbox
  • outbox (r/o) mailbox -- The top level Out mailbox
  • sent mailbox (r/o) mailbox -- The top level Sent mailbox
  • trash mailbox (r/o) mailbox -- The top level Trash mailbox
  • sort column attachments column / buddy availability column / message color / date received column / date sent column / flags column / from column / mailbox column / message status column / number column / size column / subject column / to column / date last saved column -- The column that is currently sorted in the viewer
  • sorted ascending boolean -- Whether the viewer is sorted ascending or not
  • mailbox list visible boolean -- Controls whether the list of mailboxes is visible or not
  • preview pane is visible boolean -- Controls whether the preview pane of the message viewer window is visible or not
  • visible columns list of attachments column / buddy availability column / message color / date received column / date sent column / flags column / from column / mailbox column / message status column / number column / size column / subject column / to column / date last saved column -- List of columns that are visible. The subject column and the message status column will always be visible
  • id (r/o) integer -- The unique identifier of the message viewer
  • visible messages list of message -- List of messages currently being displayed in the viewer
  • selected messages list of message -- List of messages currently selected
  • selected mailboxes list of mailbox -- List of mailboxes currently selected in the list of mailboxes
  • window window -- The window for the message viewer
