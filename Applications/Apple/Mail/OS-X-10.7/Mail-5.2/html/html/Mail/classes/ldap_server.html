Class ldap server

expand

Description

LDAP servers for use in type completion in Mail

  • Plural name:
    • ldap servers
  • Defined in:
    • Mail

Properties

  • enabled boolean -- Indicates whether this LDAP server will be used for type completion in Mail
  • name rich text -- Name of LDAP server configuration to be displayed in Composing preferences
  • port integer -- Port number for the LDAP server (default is 389)
  • scope base / one level / subtree -- Scope setting for the LDAP server
  • search base rich text -- Search base for this LDAP server (not required by all LDAP servers)
  • host name rich text -- Internet address (myldapserver.company.com) for LDAP server

Notes

  • Contained by:
    • application (Standard Suite)
