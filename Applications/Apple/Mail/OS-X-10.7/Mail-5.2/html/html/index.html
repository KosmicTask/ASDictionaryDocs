Commands

  • extract name from -- Command to get the full name out of a fully specified email address. E.g. Calling this with "John Doe <jdoe@example.com>" as the direct object would return "John Doe"
  • extract address from -- Command to get just the email address of a fully specified email address. E.g. Calling this with "John Doe <jdoe@example.com>" as the direct object would return "jdoe@example.com"

  • GetURL -- Opens a mailto URL.
  • import Mail mailbox -- Imports a mailbox created by Mail.
  • mailto -- Opens a mailto URL.
  • perform mail action with messages -- Script handler invoked by rules and menus that execute AppleScripts. The direct parameter of this handler is a list of messages being acted upon.
  • redirect -- Creates a redirected message.
  • reply -- Creates a reply message.
  • send -- Sends a message.
  • synchronize -- Command to trigger synchronizing of an IMAP account with the server.
