Class application

expand

Description

DVD Player

  • Defined in:
    • DVD Suite

Properties

  • version (r/o) Unicode text -- The version of the DVD Player
  • app initializing (r/o) boolean -- Is the app starting up?
  • has multiple playback choice (r/o) boolean -- Does user have multiple choices to start playback from?
  • interaction override boolean -- Turn off interaction with user for last play sheet, etc
  • viewer full screen menu override boolean -- Turn off display of menu bar when mouse moves to menu bar area in full screen mode
  • has media (r/o) boolean -- Is there media in the drive?
  • dvd menu active (r/o) boolean -- Is playback currently on a dvd menu?
  • active dvd menu (r/o) none / main / title / subpicture / audio / angle / ptt -- The current dvd menu
  • dvd state (r/o) playing / playing still / paused / stopped / scanning / idle -- The state of playback
  • dvd scan rate x2 / x4 / x8 / x16 / x32 -- The scan rate for fast forwarding and rewinding
  • controller visibility boolean -- To hide or show the controller window
  • controller position point -- The position of the controller window in screen coordinates
  • controller bounds (r/o) bounding rectangle -- The window bounds in screen coordinates
  • controller screen bounds (r/o) bounding rectangle -- The screen bounds the window is on
  • controller orientation vertical / horizontal -- The controller orientation - (Not supported in 5.0 and greater)
  • controller drawer open / closed -- The state of the controller drawer
  • info visibility boolean -- To hide or show the info window - (Not supported in 5.0 and greater)
  • info position point -- The position of the info window in screen coordinates - (Not supported in 5.0 and greater)
  • info bounds bounding rectangle -- The window bounds in screen coordinates - (Not supported in 5.0 and greater)
  • info screen bounds (r/o) bounding rectangle -- The screen bounds the window is on - (Not supported in 5.0 and greater)
  • info type standard / wide -- Sets the type of info window being displayed - (Not supported in 5.0 and greater)
  • info text color RGB color -- The info text color - (Not supported in 5.0 and greater)
  • viewer visibility boolean -- To hide or show the viewer window
  • viewer position point -- To set the position of the viewer window in screen coordinates
  • viewer bounds (r/o) bounding rectangle -- The window bounds in screen coordinates
  • viewer screen bounds (r/o) bounding rectangle -- The screen bounds the window is on
  • viewer size half / normal / double / max -- To set the the viewer size
  • viewer full screen boolean -- Set to true to enter presentation mode
  • audio muted boolean -- To turn the audio on or off
  • audio volume small integer -- The DVD audio volume, 0 is silent, 255 is full on
  • elapsed time integer -- The elapsed time, in seconds, of the current title
  • elapsed extended time point -- The elapsed time, in seconds and frames, of the current title
  • remaining time integer -- The remaining time, in seconds, of the current title
  • remaining extended time point -- The remaining time, in seconds and frames, of the current title
  • title length (r/o) integer -- The length of the title in seconds
  • title extended length (r/o) point -- The length of the title in seconds and frames
  • available angles (r/o) small integer -- The number of angles available at the current time position
  • available audio tracks (r/o) small integer -- The number of audio tracks available at the current time position
  • available chapters (r/o) small integer -- The number of chapters available at the current time position
  • available subtitles (r/o) small integer -- The number of subtitles available at the current time position
  • available titles (r/o) small integer -- The number of titles available at the current time position
  • angle small integer -- The current camera angle at the current time position
  • audio track small integer -- The current audio track at the current time position
  • chapter small integer -- The current chapter at the current time position
  • displaying subtitle boolean -- Are we currently displaying subtitles?
  • subtitle small integer -- The current subtitle (caption) at the current time position
  • title small integer -- The current title at the current time position
  • closed captioning boolean -- Turn closed captioning on or off
  • closed captioning display over video / separate window -- The display type for closed captioning
  • extended bookmarks (r/o) boolean -- Supports extended bookmark functionality
  • available bookmarks (r/o) small integer -- The number of bookmarks available for the current media
  • bookmark list (r/o) list of Unicode text -- The names of the bookmarks for the current media
  • has default bookmark (r/o) boolean -- Has a default bookmark been set?
  • has last play bookmark (r/o) <class book> -- Does the media have a last play bookmark from being previously viewed?
  • extended video clips (r/o) boolean -- Supports extended video clip functionality
  • available video clips (r/o) small integer -- The number of video clips available for the current media
  • video clip list (r/o) list of Unicode text -- The names of the video clips for the current media
  • loop video clip boolean -- To turn the loop state for video clips on or off
  • clip mode (r/o) boolean -- Is video clip being played?
  • disable status window boolean -- Disable displaying the status message?
