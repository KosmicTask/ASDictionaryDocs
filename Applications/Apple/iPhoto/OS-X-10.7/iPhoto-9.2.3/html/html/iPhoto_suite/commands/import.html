Command import

Description

Import the given path.

  • Defined in:
    • iPhoto suite

Parameters

  • reference -- the object for the command
  • from Unicode text -- The path(s) to import.
  • [force copy integer] -- Pass true to ignore advanced preference and always create copy in iPhoto library. Useful if original may be in a temporary directory. Default is true.
  • [to album] -- The album to which to import the object. If none specified, import into main Library.
