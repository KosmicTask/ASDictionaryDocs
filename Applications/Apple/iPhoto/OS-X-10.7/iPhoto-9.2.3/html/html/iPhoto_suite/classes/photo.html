Description

A photo.

  • Defined in:

  • Inherits from:

Properties

  • altitude integer -- The GPS altitude in meters; MAXFLOAT means no altitude information.
  • comment Unicode text -- A comment about the photo.
  • date date -- The date of the photo.
  • dimensions (r/o) list -- The width and height of the photo in pixels.
  • height (r/o) integer -- The height of the photo in pixels.
  • id (r/o) integer -- The unique ID of the photo.
  • image filename (r/o) Unicode text -- The name of the image file.
  • image path (r/o) Unicode text -- The path to the image file.
  • latitude integer -- The GPS latitude in range -90.0 to 90.0; MAXFLOAT means no GPS information. Use reverse geocode command after setting lat/lon.
  • longitude integer -- The GPS longitude in range -180.0 to 180.0; MAXFLOAT means no GPS information. Measurement is taken from the prime meridian, so 'west' longitudes are negative numbers. Use reverse geocode command after setting lat/lon.
  • name Unicode text -- The name (title) of the photo.
  • original path (r/o) Unicode text -- The path to the original image as it was imported to iPhoto.
  • rating integer -- The star rating (0 through 5).
  • thumbnail filename (r/o) Unicode text -- The name of the thumbnail file.
  • thumbnail path (r/o) Unicode text -- The path to the thumbnail file.
  • title Unicode text -- The title (name) of the photo.
  • width (r/o) integer -- The width of the photo in pixels.
