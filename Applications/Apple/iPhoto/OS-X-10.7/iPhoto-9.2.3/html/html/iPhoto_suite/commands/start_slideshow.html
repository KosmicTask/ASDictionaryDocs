Command start slideshow

Description

Display a slideshow with the currently-selected photos or album.

  • Defined in:
    • iPhoto suite

Parameters

  • [asynchronous integer] -- Pass true to return immediately so that caller can then issue other commands.
  • [display index integer] -- Index of display to use.
  • [iChat integer] -- Special parameter for iChat
  • [using album Unicode text] -- The name of the album to play as a slideshow. If no album name is provided, the currently-selected album is used.
