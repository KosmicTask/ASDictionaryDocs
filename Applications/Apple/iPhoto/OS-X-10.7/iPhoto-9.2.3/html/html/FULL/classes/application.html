  • Defined in:

    • iPhoto suite
  • Inherits from:

Properties

  • class (r/o) type class -- The class of the object.
  • properties record -- All of the object's properties.
  • frontmost (r/o) boolean -- Is this the frontmost (active) application?
  • name (r/o) Unicode text -- The name of the application.
  • version (r/o) Unicode text -- The version of the application.
  • current album album -- The selected album.
  • events album (r/o) album -- Events album.
  • faces album (r/o) album -- Faces album.
  • flagged album (r/o) album -- Flagged photos album.
  • importing (r/o) integer -- Returns true if photo importing is in progress.
  • last import album (r/o) album -- Last import album.
  • last months album (r/o) album -- The set of N most recently imported month(s) of photos. N is based on the preference set in iPhoto.
  • last rolls album (r/o) album -- Now obsolete; please use last import album. Old: The set of N most recently imported roll(s) of photos. N is based on the preference set in iPhoto.
  • local root albums (r/o) list -- Returns local albums that are at the root level in the source view.
  • mail address Unicode text -- The address for the e-mail.
  • mail recipient Unicode text -- The recipient for the e-mail.
  • mail subject Unicode text -- The subject for the e-mail.
  • photo library album (r/o) album -- The photo library.
  • places album (r/o) album -- Places album.
  • selection list -- The current selection.
  • shared library albums (r/o) list -- Returns shared libraries from other computers on the network.
  • slideshow running (r/o) integer -- Returns true if a slideshow is currently running.
  • trash album (r/o) album -- The set of deleted photos.
  • view edit / events / organize -- The current view in the application.
