Class bookmark item

reduce

Description

An item consists of an URL and the title of a bookmark

  • Plural name:
    • bookmark items
  • Defined in:
    • Chromium Suite

Properties

  • id (r/o) integer -- Unique ID of the bookmark item.
  • title text -- The title of the bookmark item.
  • URL text -- The URL of the bookmark.
  • index (r/o) number -- Returns the index with respect to its parent bookmark folder

Notes

  • Contained by:
    • bookmark folder (Chromium Suite)
